# 建模依据资料

日期：2026-07-03

本目录保留低温脉冲加热建模所需的论文、专利和技术材料。它不是临时附件目录，也不是优先清理对象；后续 AMESim/Simulink 建模、参数假设、析锂风险判断和阶段结论都应回到这里查证。

## 阅读顺序建议

1. 先读综述和机理类材料，明确低温性能、内阻/阻抗发热、交流或双向脉冲加热、寿命影响和安全边界。
2. 再读电驱实现类论文和专利，确认电池-逆变器-电机之间的电流路径、控制限制和硬件约束。
3. 最后读析锂限制和实验测试材料，把模型结论拆成“温升有效”“电流可实现”“低温回充安全”“硬件可靠”几个独立问题。

## 建模引用纪律

1. 文献只能支撑其实际研究范围，不得把电芯/小模组结论直接外推为整车三包并联可行。
2. 采用频域发热模型时，优先查找 `Re(Z_bat(T, SOC, f))` 或等效阻抗依据；没有 EIS 数据时要标记为待确认假设。
3. 涉及低温回充半周、析锂和寿命影响时，必须单独列出安全证据，不把温升速度当作安全证明。
4. 涉及电驱方案时，必须把电池支路电流、母线电流、电机相电流、逆变器电流和热等效电流分开表达。
5. 对外报告引用本目录资料时，应同时记录文件名、年份、适用对象、可引用结论和不可外推边界。
