## 4. 刚读完、尚未整理成深读报告的文献

### 4.1 `2022-为下一代电动汽车充电系统配备全天候加热与制冷功能.pdf`（Shell TechXplorer Digest 2022，"Equipping Next-Gen EV Charging with All-Weather Heating and Cooling"）

已读完全文（7 页），要点摘录（供接手模型直接引用，尚未写入独立深读报告或 README）：

- **性质**：Shell 内部技术科普刊物文章（TechXplorer Digest），非同行评审学术论文，**不含推导公式**，只有实验结果和工程描述。与"专利与控制方案"分组下的两篇专利性质相近（工程叙述为主），但本文没有具体控制算法细节，价值主要是**工程基准数据**和**技术路线背景**。
- **技术方案**：双向脉冲加热（bidirectional pulse heating）+ 外部冷却（external cooling），合称"外部热管理"（external thermal management）。Shell 与清华大学 2018 年起合作项目，配合 350kW 高功率充电系统。
- **双向脉冲加热原理**：周期性充放电电流在电池内部产生欧姆热（Ohmic/Joule heat）和反应热（reaction heat）；波形与常见双向脉冲文献一致（正脉冲幅值Ip/周期Tp，负脉冲幅值In/周期Tn，总周期T），见原文 Figure 3。
- **关键实验数据**（可作为 M5 取舍清单中的性能基准对照点）：
  - 模组级实验室验证（2018-2020）：170h 连续脉冲加热（11°C/min）后电池容量仅衰减 1%。
  - 整车级验证（2022 北京冬奥会）：环境温度<0°C，电池初始最低温 0°C，SOC 12%；脉冲加热阶段持续 3.7min，最低温从0升至25°C，最高温从8升至33°C；瞬时最高升温率 7.2°C/min（2.45min时），**平均升温率 6.7°C/min**。
  - 仿真数据（Figure 5）：电池包 -10°C 时，350kW 充电+脉冲加热到10°C+外部冷却，总充电时间可从 110min 降至 21.8min（SOC 10-80%）。
  - 外部冷却效果：5C 快充下电芯最高温控制在 50°C 以下；整车充电全程电池温度 <45°C。
  - 充电后段结果：SOC 12%→51% 用 463s（7.7min），配合前段脉冲加热，全程（预热+快充）8min 内补能至 150km 续航。
- **对 v2 建模的潜在价值**：
  - 提供了一个**独立于 EDS 集成路线**的双向脉冲加热工程基准（充电桩侧双向DC-DC模块控制，而非车载电驱复用），可作为 M5 取舍清单中"暂不纳入 v2"分组的候选（因其加热电流施加路径是充电桩外部双向DC-DC，不经过车载电机/逆变器，与"复用电驱"的 v2 目标架构不同），但其**RTR 数值（6.7°C/min 整车级，11°C/min 模组级）**可作为文献横向对比表（原 M5 任务规划 2.4 节提到的"技术对比表"）的补充参考点。
  - 无解析公式、无控制算法细节、无电池阻抗模型，**不提供可直接复用的建模方法**，仅提供工程基准数据和背景。

**待办**：接手模型可将此文归入 README 的"专利与控制方案"或新增"充电桩侧脉冲加热/工程基准"分组，重要性定级建议为**中低**（背景/基准参考，非核心建模依据）。
