## 2. 核心方法

### 2.1 电池交流电流形成机理

典型电机驱动电路模型中，电池交流电流的形成受 DC-link 电容大小影响显著：

- **Cp < 1 mF**：电池电流波形在不同电容值下几乎重叠，无显著衰减
- **Cp > 100 mF**：电池交流电流几乎完全消失
- **谐振频率**：约 1 kHz 附近存在谐振点

该分析表明，DC-link 电容是决定电池侧能否产生显著交流电流的关键边界条件。当电容足够小（< 1 mF）时，逆变器开关动作产生的电流脉动能够穿透到电池侧，形成可用于加热的交流分量。

### 2.2 换流模式 C（非零矢量 + 零矢量 + 死区）

提出一种三阶段换流模式，每个 PWM 周期内依次执行：

| 阶段 | 矢量 | 作用 |
|---|---|---|
| 1 | 非零矢量（有效电压矢量） | 电池放电 → 电机电感储能，电池电流正向上升 |
| 2 | 零矢量（上/下桥臂直通） | 电机电流通过续流回路衰减，电池电流指数衰减 |
| 3 | 死区（所有开关关断） | 电机电感能量经体二极管回流，反向向电池充电，形成负向电流 |

**核心机制**：利用死区阶段的反向充电电流，实现电池侧的正负双向交流电流，从而产生欧姆热。

### 2.3 控制参数

- **零矢量占空比 d₀**：间接调节电池电流幅值，d₀ 越大则电池电流幅值越小
- **非零矢量与死区时间比 a₀ ≈ 1.6**：保持充放电能量平衡的关键比值
- **控制频率**：固定 10 kHz（PWM 周期 0.1 ms）

### 2.4 双 PI 电压反馈控制

提出滞环切换的双 PI 控制架构：

- **上 PI 控制器**：控制峰值电压（Vp），调节非零矢量占空比
- **下 PI 控制器**：控制谷值电压（Vv），调节零矢量占空比
- **滞环切换**：根据电池电压波动状态在两个 PI 控制器之间切换，维持电压在目标范围内

**PI 参数**：正文给出 `Kp=0.01`、`Ki=0.9`、`Tc=1×10⁻⁴ s`；但 Table II 的 `Kp=0.1` 与正文不一致。该比例增益记为 `LITERATURE_CONFLICT`，在 v2 中不得直接采用。

## 5. 实验设计

### 5.1 验证层级

**小尺度台架验证**（未在整车级验证）：

- 电池：12 串 1 并，额定电压 50.4 V，限压 30 V
- 电机：1 kW 级，额定 48 V / 27 A
- 控制器：DSP 实现双 PI 电压反馈控制

### 5.2 对比策略

- **对照组**：固定占空比策略（无电压反馈调节）
- **实验组**：电压反馈策略（双 PI 滞环控制）

### 5.3 测试条件

- 极端低温环境（-30°C → -10°C 升温过程）
- 电池老化测试（评估加速老化影响）

## 6. 关键结果

| 指标 | 数值 | 备注 |
|---|---|---|
| 加热速率提升 | **45%** | 电压反馈 vs. 固定占空比，-30 → -10°C |
| 电池老化 | **无明显加速老化** | 老化测试通过 |

## 7. 结论边界（不可外推项）

1. **验证层级有限**：仅在小尺度台架（12S1P, 1 kW 电机, 48 V 平台）验证，未在整车级（高电压平台、大容量电池包、真实驱动工况）验证，结果不可直接外推到整车系统。
2. **未考虑电池以外器件的温度敏感性**：电机绕组电阻、IGBT 导通压降、DC-link 电容 ESR 等均随温度变化，但文中未建模这些温度耦合效应，实际加热过程中的系统效率可能与文中结论偏离。
3. **加热效率仅为量级估算**：效率分析基于理论量级估算，非直接实测，不能作为精确效率数据使用。
4. **未讨论析锂判据或长期循环寿命**：虽然老化测试表明无显著加速老化，但未建立析锂阈值模型或长期循环（> 1000 次）数据，低温大电流脉冲的析锂风险未量化评估。
5. **控制策略未考虑实际车辆控温回路的实用性和可靠性**：未讨论电池管理系统（BMS）与驱动控制器之间的通信延迟、传感器精度限制、故障容错等工程实现问题。
6. **DC-link 电容边界条件限制**：结论适用于 Cp < 1 mF 的场景，而当前主流电动汽车 DC-link 电容通常在 mF 级以上（数百 µF 至数 mF），需逐一确认是否满足该边界条件。
7. **未覆盖多电池模组并联拓扑**：12S1P 单串结构无法代表多并、多模组并联的真实电池包，电流分配不均、环流等工程问题未涉及。

## 8. 对 v2 建模的参考价值

### 8.1 可迁移的方法

1. **换流模式 C 的时序逻辑**：非零矢量 → 零矢量 → 死区的三阶段 PWM 调制策略可直接作为 v2 平台控制方法的一种候选实现路径。
2. **双 PI 电压反馈控制架构**：峰值/谷值电压双闭环 + 滞环切换的控制框架可迁移到 v2 的控制子系统，替代 M4 中简单的开环 Id 参考脉冲注入方式。
3. **DC-link 电容边界分析**：Cp < 1 mF 为电池侧交流电流形成的必要条件，这一结论可直接用于 v2 平台的 DC-link 参数设计指导。
4. **a₀ ≈ 1.6 能量平衡比**：作为保持充放电能量平衡的工程经验值，可用于 v2 控制策略的初始参数设定。

### 8.2 可迁移的安全代理

1. **DC-link 电容阈值（< 1 mF）**：可作为 v2 平台 DC-link 参数设计的硬约束，在系统兼容性筛查中作为检查项。
2. **电池电流指数衰减规律**：零矢量阶段电池电流的衰减特性可作为电池侧电流幅值估算的代理模型。

### 8.3 需谨慎处

1. **验证尺度差异**：该文献基于 48 V / 1 kW 小尺度台架，v2 平台若面向更高电压（如 400 V / 800 V）和更大功率，控制参数（PI 增益、死区时间、占空比范围）需要重新整定，不可直接移植。
2. **电机电感量级差异**：文献台架电机 Ld=71.2 µH，远小于 v2 参考平台 PMSM 的电感值（~mH 级），换流时间常数和电流幅值特性将显著不同。
3. **未建模热耦合**：器件温度敏感性未纳入模型，v2 如需精确评估加热效率，应建立包含 IGBT、电机绕组、DC-link 电容的温度相关参数模型。
4. **析锂判据缺失**：v2 应建立析锂阈值模型（如负极电位监测、阳极过电位估算），弥补该文献的安全评估空白。

### 8.4 与 M4 / 其他文献的关系

- **与 M4 CMI 架构的关系**：该文献的电压反馈控制策略闭环程度高于 M4 的开环 Id 脉冲注入，可作为 v2 从开环走向闭环控制的重要参考。但需注意该文献的换流模式 C 与 M4 的 CMI 架构在换流路径上存在本质差异，不可简单替换。
- **与 2024 IEEE TTE 级联型加热（文献 1）的关系**：该文献（电压反馈式）与同为 2024 IEEE TTE 的级联型加热策略在 DC-link 电容约束上立场一致（均要求 Cp < 1 mF），但控制方法不同：前者用电压反馈闭环，后者用级联 H 桥。两者可互为 v2 平台控制方法的不同候选路径。
- **与 M4 冻结基线的关系**：M4 的 RMS 窗口审计、账本校验和参数纪律（OFFICIAL / LITERATURE / ENGINEERING_DEFAULT）应继承到 v2；该文献的参数应标记为 `LITERATURE` 来源，不自动升级为 v2 默认值。
