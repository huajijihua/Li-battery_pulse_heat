## 2. 核心方法

### 2.1 四种电池内部加热方式对比

| 方式 | 原理 | 特点 |
|------|------|------|
| 直流加热 | 电池直接放电经电阻发热 | 简单但加热不均匀，可能引发析锂 |
| 交流加热 | 外部交流信号注入，电池内阻产热 | 需额外交流电源，集成度低 |
| 脉冲电流加热 | 利用电机驱动电路，储能+续流模式产生脉冲电流 | 无需额外硬件，集成度高，本报告重点 |
| 内部镍箔加热 | 电池内部嵌入镍箔，通电产热 | 改变电池结构，工艺复杂 |

### 2.2 脉冲电流加热原理（储能+续流模式）

利用电机驱动电路实现电池脉冲电流加热，无需额外硬件：

- **储能模式（电池放电）**：利用驱动电机电感特性存储电池放电能量，通过 SVPWM 调制控制开关管导通，使电池电流流经电机绕组，电感储能。
- **续流模式（电池充电）**：切断全部 6 个开关管，电机线圈中存储的能量通过开关管的反并联续流二极管反向向电池充电，形成完整的脉冲电流周期。
- **加热脉冲信号频率**：为 SVPWM 采样频率的一半。

### 2.3 建模方法

- **电机模型**：`ACMachinePMSynch`（基于等效电路的永磁同步电机模型），可选择 Level 1–4 精度等级。
- **电池模型**：电化学模型（Autolion），可输出析锂电位（Li-plating potential），用于定量评估析锂风险。
- **开关管模型**：MOSFET 数据 map，包含导通损耗、开关损耗和反向恢复损耗。
- **控制策略**：SVPWM + Clarke/Park 逆变换，Iq=0 控制（仅利用 d 轴电流加热）。
- **仿真平台**：GT-SUITE v2025。

### 2.4 SVPWM 实现方式

基于载波的脉宽调制实现 SVPWM：采用谐波注入（零序分量注入）+ 载波信号比较的方式生成 PWM 驱动信号，等效于传统空间矢量调制。

### 2.5 具体参数

- 电池：18650 型 NCM811
- 电机：6 极永磁同步电机
- 开关管：SiC MOSFET

## 3. 关键公式/步骤

### 3.1 储能模式（电池放电）

电池 → 逆变器（开关管导通）→ 电机绕组 → 电感储能

电流路径：电池正极 → 逆变器上桥臂 → 电机绕组 → 逆变器下桥臂 → 电池负极

### 3.2 续流模式（电池充电）

电机绕组（释放储能）→ 续流二极管 → 电池充电

电流路径：电机绕组 → 逆变器下桥臂续流二极管 → 电池正极 ← 上桥臂续流二极管 ← 电机绕组

### 3.3 频率关系

`f_heating = f_SVPWM_sampling / 2`

即加热脉冲频率为 SVPWM 采样频率的一半。

### 3.4 析锂电位判据

通过 Autolion 电化学模型输出负极析锂电位（Li-plating potential），当该电位低于析锂临界阈值时，指示存在析锂风险。

## 5. 实验设计（建模方法）

### 5.1 仿真架构

```
电池（Autolion 电化学模型）
  → 逆变器（SiC MOSFET 损耗 map）
    → 电机（ACMachinePMSynch，Level 可选）
      → 控制（SVPWM + Clarke/Park 逆变换，Iq=0）
        → 析锂电位输出（Autolion 后处理）
```

### 5.2 关键建模要素

1. **电化学模型**：Autolion 提供负极析锂电位输出，这是评估析锂风险的核心手段。
2. **开关管损耗模型**：使用 MOSFET 数据 map 覆盖导通损耗、开关损耗和反向恢复损耗，比固定损耗参数更精确。
3. **电机等效电路模型**：ACMachinePMSynch 支持多级精度，可根据研究需要选择 Level 1（基本电气等效）到 Level 4（含铁耗、磁饱和等）。
4. **SVPWM 控制**：基于载波的谐波注入实现，与 M4 模型中的官方 PWM 生成方式原理一致。

### 5.3 与 M4 模型的异同

| 维度 | ATIC 方法 | M4 模型 |
|------|-----------|---------|
| 加热原理 | 储能+续流 | 储能+续流（一致） |
| 控制策略 | SVPWM，Iq=0 | SVPWM/FOC，Iq=0 参考 |
| 电池模型 | Autolion 电化学模型 | 简化电气模型（无电化学） |
| 开关管 | SiC MOSFET 损耗 map | 简化 IGBT Ron/Vf |
| 评估维度 | 析锂电位 | 能量账本+热账本 |
| 仿真平台 | GT-SUITE v2025 | MATLAB/Simulink |
| 结论类型 | 方法论介绍，无具体实验结果 | 有量化仿真结果 |

## 6. 关键结果

**本报告为方法论介绍，无具体实验或仿真结果数据。** 主要产出为：

1. 建立了脉冲电流加热的储能+续流仿真框架，可在 GT-SUITE 中完整实现。
2. 提出了通过 Autolion 电化学模型输出析锂电位的评估路径。
3. 对比了四种电池内部加热方法，说明了脉冲电流加热在集成度和硬件成本上的优势。
4. 明确了 SVPWM 采样频率与加热脉冲频率的关系（2:1）。

## 7. 结论边界（不可外推项）

1. **无实验结果**：本报告是纯方法论 PPT，不包含任何具体实验或仿真数值结果，不能据此得出任何定量结论。
2. **非学术同行评审**：ATIC 为第三方咨询/技术服务机构，报告未经同行评审，方法可信度需独立验证。
3. **电化学模型依赖标定数据**：Autolion 电化学模型的析锂电位输出精度高度依赖电极材料参数和老化状态标定，标定数据未公开，无法直接复现。
4. **析锂电位判据阈值未知**：报告中未给出析锂电位的具体临界阈值，无法判断何种工况会触发析锂。
5. **18650 单电芯 vs 电池包外推限制**：报告以 18650 NCM811 单电芯为案例，结论不能直接外推到方形电池、软包电池或大容量电池包。
6. **GT-SUITE 与 Simulink 模型差异**：GT-SUITE 在电化学建模（Autolion）和开关管损耗 map 方面与 Simulink 能力不同，方法不能直接移植，需适配接口。
7. **SiC MOSFET 与 IGBT 差异**：报告使用 SiC MOSFET，M4 模型使用 IGBT，两者开关特性、损耗分布和续流二极管行为不同。
8. **无保护/安全边界的讨论**：报告未涉及加热过程中的电压限幅、电流限幅、温度保护或控制饱和等工程安全机制。

## 8. 对 v2 建模的参考价值

### 高价值参考

- **储能+续流模式框架**：与 M4 模型的控制原理一致，验证了 M4 方法在行业内的通用性。
- **SVPWM 采样频率与加热频率关系**：`f_heating = f_sampling / 2` 可作为 v2 平台频率参数设计的参考依据。
- **析锂电位评估路径**：v2 平台应引入析锂判据（至少在电气侧建立代理模型或查表判据），使 `DeviceProtection_Evaluable` 从 `0` 变为可评估。ATIC 使用 Autolion 电化学模型的方法可作为目标参考，但不要求在 v2 初期直接集成电化学模型。
- **Iq=0 控制策略**：与 M4 模型一致，v2 可继续沿用此策略作为基线，仅利用 d 轴电流进行加热。

### 中等价值参考

- **开关管损耗 map 方法**：比 M4 当前使用的固定 `Ron/Vf` 参数更精确，但 v2 初期可继续使用简化参数，在热审计精度要求提高时再升级。
- **ACMachinePMSynch 多级精度选择**：与 M4 中 PMSM 模型的参数化设计思路一致，v2 可保留类似的分级精度选项。

### 低价值/不可直接使用

- **GT-SUITE 具体实现**：与 Simulink 平台不兼容，需在 Simulink 中重新实现等效功能。
- **Autolion 电化学模型**：专有商业工具，无法直接集成到 Simulink 模型。v2 可考虑建立基于文献数据的析锂电位代理模型或简化判据。
- **18650 具体参数**：与 v2 平台尚无具体电池规格对应关系，可作为参数参考但不作为默认值。
