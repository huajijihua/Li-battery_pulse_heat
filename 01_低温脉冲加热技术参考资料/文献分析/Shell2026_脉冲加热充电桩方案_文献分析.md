# Shell 2026 脉冲加热充电桩方案 - 文献分析

## 1. 来源与核验状态

- **来源**：`壳牌脉冲加热充电桩方案PPT截图/1.png`~`6.png`
- **材料性质**：Shell International B.V. 内部方案演示截图，标注 Confidential；不是同行评审论文，也不是公开的完整技术规范。
- **视觉核验状态（2026-07-22）**：6 张 PNG 已完成逐张视觉核验。当前记录仅覆盖截图中可辨认的文字、框图、参数表和统计图，不替代完整原始 PPT、试验原始数据或通信协议文件。

## 2. 方案边界与系统组成

截图 1 和 5 显示，该方案的脉冲功率路径位于充电基础设施侧：电网/开关柜 -> PCS -> 双向 DC/DC 转换器 -> 车辆充电终端。图中还标出 500 kWh 独立 BESS、两台双向 DC/DC 转换器和两台双枪充电终端。

因此，该方案不是 M4 所研究的“电池包 - DC-link - 逆变器 - PMSM”车载电驱复用路径。它属于外部充电桩侧双向功率变换方案，车辆需要通过 BMS 软件、通信协议和高压协调参与脉冲加热。

## 3. 通信、状态机与工程集成

### 3.1 开发分工（PNG 2）

材料将工作拆为 OEM、Shell/THU 和供应商协作内容：

- 获取选定车型的 BMS/电池模组数据。
- 通过仿真和实验联合开发脉冲加热地图与策略算法。
- 基于 GB/T 27930-2015 定义通信协议和脉冲加热功能时序。
- 定义车端和桩端的脉冲加热功能及错误处理要求。
- 车端 BMS 软件更新、供应商软件需求发布、车端调试和联合现场验证。

### 3.2 正常流程（PNG 3）

流程为：匹配 -> 握手，确认 EVSE 是否支持脉冲加热 -> 参数配置，确定加热/正常充电状态和参数 -> 车辆基于 SOC 与电池温度决定是否进入脉冲加热 -> 脉冲加热 -> 根据电池温度判断是否已充分加热 -> 正常充电或继续脉冲加热。

截图明确提到：通过修改基于 GB/T 27930-2015 的专有通信协议实现功能，修改范围主要是时序、消息和信号定义；CHM 消息增加 EVSE 是否支持脉冲加热的版本指示，并增加进入、保持、控制、监测和错误处理消息。

## 4. 参数地图与保护边界（PNG 4）

截图给出一个“pack-level pulse-heating current map”示例，按 Real-SOC、block temperature 和脉冲加热电流映射工作区间。可直接辨认的示例参数包括：

| 参数 | 截图示例值 | 解释边界 |
|---|---:|---|
| 最大脉冲加热电流 | 350 A | 方案示例值，不是本项目平台默认值 |
| 最大单体电压 | 3.85 V | 保护/进入条件示例，不是析锂判据 |
| 进入脉冲加热温度区间 | -25°C 至 5°C | 方案示例范围，依赖特定车型和电池 |
| 最大脉冲加热持续时间 | 25 min | 方案示例限制，不能外推为通用持续时间 |

图中还显示：SOC 低于 10% 或高于 90% 时部分工况不进入脉冲加热；不同温度区间按 block temperature 设置电流或退出状态；若干电流/参数被标注为因保密原因模糊处理。

这说明工程方案至少需要把 SOC、单体/模组电压、block temperature、持续时间和通信状态共同纳入进入/退出地图。该地图是控制与保护需求的工程接口，不等于电化学析锂模型。

## 5. 车队试验结果（PNG 5-6）

### 5.1 试验部署与运行结果（PNG 5）

- 2023 年第三季度在中国长春建设并部署一座脉冲加热示范站，与一汽红旗联合开展车队试运行。
- 试验地点冬季平均温度约 -10°C。
- 脉冲加热可覆盖每年超过 3 个月的长春驾驶需求。
- 相比普通快充，冬季充电时长减少约 30%~40%，约 30 min。
- 车队试运行期间平均脉冲加热速率为 2.6°C/min。
- 试验中未观察到电池 SOH 下降与脉冲加热持续时间之间的相关性。

截图右侧的技术性能图包含 PH 性能、SOC 对比、电池温度对比以及 SOH 下降与脉冲加热持续时间散点图，但截图没有提供样本数量、误差条、车辆配置和完整原始数据。

### 5.2 SOH 统计边界（PNG 6）

- 左上图：SOH 下降与脉冲加热会话次数未显示明显相关性。
- 左下图：SOH 下降与脉冲加热总持续时间未显示明显相关性。
- 右图：平均 SOH 下降与平均日行驶里程呈更强的正相关。
- 材料同时指出，充电行为、驾驶习惯等难以控制和量化的因素可能影响试验中的 SOH 变化。

这些是车队统计关联观察，不是寿命因果证明，也不是对低温脉冲加热安全性的独立验证。

## 6. 对本项目文献整理的价值

### 可作为工程接口参考

1. BMS/EVSE 握手、能力指示、参数配置、进入/保持/退出和错误处理状态机。
2. 以 SOC、温度、电压和最大持续时间组成的脉冲加热参数地图。
3. 将车端软件、充电桩控制、高压协调和现场验证作为系统集成边界。
4. 2.6°C/min 的长春车队试验数据可作为充电桩侧工程基准点。

### 不可直接迁移到 M4/v2 车载电驱模型

1. 双向 DC/DC、BESS 和充电终端不等价于车载 PMSM 逆变器换流路径。
2. 截图没有公开电池脉冲电流波形、DC/DC 控制器参数、EIS/ECM 参数或析锂判据。
3. 350 A、3.85 V、-25~5°C、25 min 等值属于特定 OEM 方案示例，不能作为通用 Reference / Platform 默认值。
4. SOH 统计没有给出完整试验设计、对照组、样本量和因果分析，不能替代析锂、寿命或产品安全评估。
