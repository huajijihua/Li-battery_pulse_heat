## 2. 核心方法

### 2.1 CMI 模式（Combined Module Invert）

**拓扑**：电池单包连接，常规驱动电路，电机线圈作为储能电感。

**工作原理**：
- 逆变器开关状态切换时，电机线圈（Ld=0.38mH, Lq=1.065mH）储能/释放，在电池回路中产生脉冲电流
- 有效电压矢量（U1–U6）时，电池电流为相电流的一半
- 零矢量（U0/U7）时，电池电流为 0，线圈能量在电机内部续流消耗
- 通过调节 PWM 周期和电角，控制电池 RMS 电流幅值

**最优工况**：周期 0.008 s（125 Hz），电角 90°，最大电池有效电流 70.4 A（1.41 C）。

### 2.2 DMSI 模式（Dual-Module Separated Invert）

**拓扑**：电池包拆分为两个独立模块（Module A / Module B），分别连接逆变器不同桥臂。开关 S 闭合时并联=CMI 模式；S 断开时两个模块独立工作。

**工作原理**：
- U7 零矢量（三个上桥臂全开）时，两个模块通过逆变器形成回路，实现模块间能量转移
- U0 零矢量（三个下桥臂全开）时，电池电流为 0
- DMSI 模式下，电池回路电阻远小于 CMI（回路缩短），因此加热电流可大幅提升
- 有效电流可达 199 A（3.98 C，50 Ah 电芯基准）

**最优工况**：周期 0.06 s，U7 占空比 0.5，电池有效电流 199 A（3.98 C）。

### 2.3 加热效率分析

能量流分解：
- **Qbat**（电池欧姆热）：占 97.7%
- **Qdis**（IGBT/线圈/线路损耗）：占 8%（效率 = Qbat / (Qbat + Qdis) = 89.7%）
- 电池自发热占绝对主导，电机/IGBT 发热仅占约 2.3%

### 2.4 循环寿命验证

50 Ah 电芯在 4 C 脉冲电流、-7 °C 环境下进行 5000 次加热循环，循环后：
- 容量衰减仅 0.5%
- SEM 图像未发现析锂迹象
- 说明 4 C 脉冲加热在电芯耐久性方面安全

## 3. 关键公式汇总

| 公式 | 物理意义 | 备注 |
|---|---|---|
| $I_{eff,B} = \sqrt{\frac{1}{T}\int_0^T i_B^2(t)dt}$ | 电池 RMS 电流 | 加热能力的核心评价指标 |
| $RTR = \frac{T_{final} - T_{initial}}{t_{heating}}$ | 温升速率（Rate of Temperature Rise） | °C/min |
| $\eta = \frac{Q_{bat}}{Q_{bat} + Q_{dis}} \times 100\%$ | 加热效率 | 仅电池欧姆热作为有效热 |
| $i_B = \frac{1}{2}i_{ph}$（CMI 有效矢量） | CMI 模式电池电流与相电流关系 | 仅 U1–U6 |
| $i_B = i_{ph}$（DMSI U7） | DMSI 模块间能量转移电流 | 仅 U7 |
| $\gamma = \frac{t_{U7}}{T}$ | U7 占空比 | DMSI 模式控制参数 |

## 5. 实验设计

### 5.1 仿真验证

- **工具**：MATLAB/Simulink 仿真
- **目的**：不同周期、电角、占空比下的电池有效电流和加热效率计算
- **范围**：CMI 和 DMSI 两种模式的多工况参数扫描

### 5.2 台架实验

**电池包加热实验**：
- 环境温度：-7 °C 和 -16 °C
- 加热策略：DMSI 模式，1.6 s 周期（含 0.06 s 加热脉冲 + 间歇）
- 温度监测：电池包中心及表面多个热电偶
- 结果记录：温升曲线、温升速率、能量效率

**循环寿命实验**：
- 电芯：50 Ah 石墨/NMC 电芯
- 加热条件：4 C 脉冲电流，-7 °C 环境
- 循环次数：5000 次
- 评估手段：容量测试 + SEM 截面图像析锂检查

## 6. 关键结果

### 6.1 电流能力

| 指标 | CMI | DMSI |
|---|---|---|
| 最大 Ieff_B | 70.4 A（1.41 C） | 199 A（3.98 C） |
| 峰值电流 | — | 400 A（8 C） |
| 最优频率 | 125 Hz | 16.7 Hz（0.06 s 脉冲） |
| 限制因素 | 电池回路电阻（含电机线圈） | 回路电阻大幅降低 |

### 6.2 温升性能

- **-7 °C，DMSI**：前 2 分钟平均 RTR 8.6 °C/min，全程 1.6 s 周期平均 7.2 °C/min
- **-16 °C，DMSI**：首分钟 RTR 9.9 °C/min，全程 8.2 °C/min
- 电池从 -7 °C 加热到 0 °C 以上约需 2–3 min

### 6.3 加热效率与能量分配

- 加热效率 89.7%
- 电池欧姆热 Qbat 占 97.7% → 电机/IGBT 热仅占 2.3%
- 加热效率几乎不随环境温度变化

### 6.4 循环寿命

- 5000 次 4 C 脉冲加热循环后容量衰减 0.5%
- SEM 未检出析锂
- 加热脉冲本身不影响电芯寿命（在 4 C 限幅内）

## 7. 结论边界（不可外推项）

1. **DMSI 需要硬件变动**：电池包需拆分为两个独立模块 + 继电器 S，无法在常规单包拓扑上直接实现。CMI 虽然无需额外硬件，但电流仅 1.41 C，加热速度有限。
2. **DMSI 不支持途中加热**：仅适用于驻车预热场景。车辆行驶中无法使用 DMSI 模式（需断开 S 恢复常规驱动）。
3. **加热效率高度依赖热边界条件**：原文效率基于 h=9.6 W/(m²·K) 的热腔环境。实际电池包热阻、冷却系统、外壳散热系数不同时，效率可能偏移。
4. **未讨论 NVH/振动噪声**：4 C 脉冲（400 A 峰值）在电机线圈中产生显著电磁力脉冲，可能引发噪声和振动，原文未涉及。
5. **析锂安全仅基于 SEM 图像确认**：未建立可解析的析锂判据（如负极电位监测、析锂边界模型）。5000 次循环无析锂不能直接外推至更高倍率、更低温度或不同电芯化学体系。
6. **电芯体系限制**：仅验证了 50 Ah 石墨/NMC 电芯。对 LFP、LTO 或其他化学体系，倍率能力和析锂边界不同，结论不可直接移植。
7. **未考虑 SOC 影响**：原文未讨论不同 SOC 下加热电流的析锂风险和加热效果差异，结论基于特定 SOC 条件。
8. **未考虑电池老化影响**：内阻随老化增加，加热电流能力和效率可能变化，原文未涉及。

## 8. 对 v2 建模的参考价值

### 8.1 可迁移的方法

| 方法/结论 | 对 v2 的参考价值 | 优先级 |
|---|---|---|
| CMI 模式 d 轴脉冲加热（Ieff_B=1.41 C） | v2 平台可继承 CMI 作为参考基线，M4 已实现类似拓扑 | **高** |
| 电压矢量-电池电流对应关系（Table 1） | 电流关系式可直接用于 v2 审计账本中的电池电流计算 | **高** |
| 加热效率分解（Qbat 97.7% + Qdis 2.3%） | v2 能量账本可参考此分配比例作为平台默认基准 | **高** |
| 最优周期/电角扫描方法 | 参数敏感性分析的方法论可直接复用 | **中** |
| 电池有效电流 Ieff_B 作为评价指标 | v2 应继续使用 RMS 电流作为加热能力核心指标 | **高** |

### 8.2 需谨慎之处

- **DMSI 模式不可直接移植**：v2 平台目前是单包拓扑（类似 CMI）。DMSI 需要模块拆分和继电器，属于硬件架构变更，不应作为 v2 默认方案。
- **4 C 电流结论不适用于 CMI**：DMSI 的 4 C 电流建立在回路电阻大幅降低的基础上，CMI 拓扑受电机线圈阻抗限制，无法达到 4 C。
- **效率 89.7% 和 Qbat 97.7% 是有条件的热腔结果**：v2 账本中应使用自己的热边界条件重新计算，不可直接引用原文数值。
- **5000 次无析锂 = 4 C 安全**：仅对原文特定电芯成立。v2 应建立自己的析锂判据（负极电位估算），而非直接引用此结论。

### 8.3 与 M4 的关系

- **M4 已实现类似 CMI 的拓扑**：M4 基线模型（V4）已实现 d 轴脉冲加热、闭环 FOC、门极互补控制，与本文 CMI 模式方案一致。
- **M4 的电流水平（约 40 A RMS）低于本文 CMI 的 1.41 C（70.4 A）**：差异源自默认参数不同（Rs、Ld/Lq、DCLink 等），v2 平台参数匹配时可参考本文参数水平。
- **M4 账本已验证 Energy_Ledger_Valid=0**：尚未具备完整的析锂判据和热边界模型，与本文的析锂安全结论（SEM 确认）处于不同阶段。
- **本文为 v2 提供文献基准**：本文的 CMI 结果可作为 v2 平台参数匹配和工况设计的参考上限（1.41 C / 125 Hz / 90° 电角），v2 超越此基准需有文献或建模依据。
