## 2. 核心方法

### 2.1 三电极模型电池搭建与 EIS 测试

使用叠片式三电极模型锂离子电池（NCM111/石墨，2.7 Ah），以直径 100 μm 镀铜丝镀锂作为参比电极。使用电化学工作站（CORRTEST CS1350）在全频率范围 10⁴~10⁻¹ Hz、5 mV 扰动下测量三电极 EIS，覆盖 -20°C 至 5°C 六个温度点和 25%/50%/75%/100% 四个 SOC 点。

### 2.2 EEC 参数拟合

采用 ZSimpWin 软件拟合 EIS 数据，EEC 模型包含三个串接部分：欧姆阻抗 Z₁（R₀）、SEI 阻抗 Z₂（RSEI∥CPE_SEI）、电荷转移阻抗 Z₃（Rct∥CPE_dl）。分别对全电池、正极、负极独立拟合，得到各温度/SOC 下的 R₀、RSEI、Rct、QSEI、nSEI、Qdl、ndl。

### 2.3 三种析锂抑制控制策略

**TVCS（端电压控制策略）**：基于饱和浓度判据，限制端电压在充放电截止电压内（UC=4.2 V, UD=2.5 V）。用全电池阻抗计算最大允许电流幅值 ITerminal。方法简单，只需两电极数据，但低频时负极电位低于 0 V vs Li/Li⁺→析锂。

**NEPCS（负极电位控制策略）**：基于负极电位判据，直接限制 |U₃^Neg| = Iac·|Z₃^Neg| ≤ Ue（负极平衡电位，>0 V vs Li/Li⁺）。精度最高，但需要三电极数据获取负极阻抗。

**FCICS（全电池阻抗控制策略）**：用全电池 Z₃^Full 替代负极 Z₃^Neg，IFull = Ue / |Z₃^Full|。由于 |Z₃^Full| > |Z₃^Neg|，IFull < INeg，保守但损失加热速率（6.49%~61.4%），且高频时仍导致端电压越限。

### 2.4 边界频率与组合策略

定义边界频率 fBoundary 为 TVCS 与 NEPCS 适用性曲线的交点频率。低频段（f ≤ fBoundary）用 NEPCS，高频段（f > fBoundary）用 TVCS，形成组合策略，同时避免析锂和端电压越限。

## 5. 实验设计

### 5.1 验证层级

- **单电池级**：叠片式三电极模型电池（2.7 Ah, 56 g），非电芯组或电池包级
- **EIS 测试**：电化学工作站 + 气候箱
- **模型验证实验（TVCS 验证）**：-20°C, 50% SOC, 以 2 Hz/2.1 A 交流运行 10 s，验证端电压不超过 4.2 V

### 5.2 对比策略

| 策略 | 判据基础 | 输入数据需求 | 是否可防析锂 | 是否防越限 |
|---|---|---|---|---|
| TVCS | 饱和浓度 | 全电池 EEC | 低频时否 | 是 |
| NEPCS | 负极电位 | 三电极 EEC | 是 | 高频时否 |
| FCICS | 全电池阻抗代理 | 全电池 EEC | 是（保守） | 高频时否 |
| 组合策略 | 混合 | 三电极 EEC | 是 | 是 |

### 5.3 测试条件

- 温度：-20°C, -15°C, -10°C, -5°C, 0°C, 5°C
- SOC：100%, 75%, 50%, 25%
- 频率范围：0.1 Hz ~ 1000 Hz
- 激励：正弦交流电流（SAC, 0 相位角）

## 7. 结论边界（不可外推项）

1. **单电池级验证，未扩展到电池包**：实验基于 2.7 Ah 单层叠片电池，未验证多并/多串模组或电池包中的电流分配不均、环流、温度梯度等效应对析锂边界的影响。

2. **EEC 参数仅在特定电池化学体系下有效**：NCM111 / 石墨体系，电解液为 LiPF₆ 有机溶剂。不同正极（NCM811, LFP, NCA）、负极（SiOx, LTO）或电解液配方将改变 EEC 参数和 D1/D2/D3 比值，边界频率和限流幅值不可直接迁移。

3. **正弦交流激励简化，不代表真实逆变器电流波形**：文中假设 SAC 为 0 相位角正弦波，实际逆变器输出含 PWM 谐波、死区效应、非正弦波形，可能产生额外的极化或析锂触发条件。

4. **未考虑温度动态变化对 EEC 参数的影响**：EIS 在固定温度点测量，加热过程中电池温度连续变化导致 EEC 参数时变，文中温度自适应模型（附录 A2）仅做集总参数估算，未验证参数追踪的实时性和精度。

5. **未量化析锂的累积效应**：论文仅给出单次正弦周波的析锂门槛，未讨论循环累积析锂量、SEI 增厚、容量衰减等长期寿命影响。

6. **对地/绝缘监测未考虑**：交流预热过程中共模电压、漏电流、绝缘监测等整车安全工程问题未涉及。

7. **0 Hz 边界频率的物理含义有限**：100% SOC 时 fBoundary=0 意味着全频段推荐 TVCS，但 TVCS 在低频本身存在析锂风险，该结论受限于 100% SOC 时 Ue 较小的特殊条件。

## 8. 对 v2 建模的参考价值

### 8.1 可直接迁移的判据公式

1. **析锂判据（公式 9）**：|U₃^Neg| = Iac·|Z₃^Neg| ≤ Ue — 可作为 v2 平台电池析锂安全评估的核心判据，植入 Model 或后处理审计脚本中。
2. **组合策略限流公式（公式 20）**：边界频率下方的 NEPCS → 上方的 TVCS 切换策略，可直接作为 v2 控制策略的数学基础。
3. **产热率公式（公式 A2）**：q = ½·Iac²·Re(ZFull) — 可在 v2 热模型中用全电池实部阻抗计算焦耳热，与 M4 现有热账本格式兼容。

### 8.2 可迁移的参数边界

1. **D1/D2/D3 比值范围**：D1=41~51%, D2=18~41%, D3=60~93% 可作为 v2 平台电池模型阻抗分配的参考检查范围，用于验证电池模型参数是否在合理物理区间内。
2. **边界频率量级（300~400 Hz at -20°C, 50~25% SOC）**：可作为 v2 控制方法候选频率的参考分界，低于此频率需考虑析锂保护。
3. **FCICS 产热损失率（6.49~61.4%）**：量化了使用全电池阻抗代理负极阻抗的保守程度，为 v2 选择控制策略提供取舍依据。

### 8.3 对控制策略设计的指导

1. **NEPCS 优于 FCICS 的结论**：本文明确建议 FCICS 不推荐（既损失加热速率又无法避免越限），v2 应优先考虑 NEPCS 或组合策略，而非 FCICS。
2. **TVCS 低频缺陷**：仅用端电压约束无法在低频段防析锂，v2 必须在低频时引入负极电位估计或安全代理。
3. **边界频率的动态调整**：SOC 和温度双重影响边界频率，v2 控制策略应实现频率自适应调整，而非固定值。

### 8.4 与 M4 基线的关系

1. **M4 基线 CMI 架构的析锂盲区**：M4 的 RMS 审计和账本校验未包含析锂判据，DeviceProtection_Evaluable=0。本文的析锂判据（公式 9）可直接填补这一空白，使 v2 的 DeviceProtection_Evaluable 变为可评估。
2. **参数来源标记**：本文的 EEC 参数比值和边界频率应标记为 `LITERATURE` 来源，对应 `LITERATURE-Jian2023`，不自动升级为 v2 默认参数。
3. **与 M4 低频 d 轴脉冲的关系**：M4 的 CMI 开环低频 d 轴脉冲频率（25/50/100 Hz）均低于本文的边界频率（300~400 Hz），意味着 M4 的工况实际处于析锂风险频段，v2 必须引入析锂判据对这一风险做量化评估。

### 8.5 需谨慎处

1. **电池化学差异**：本文使用 NCM111/石墨、2.7 Ah，v2 若使用不同化学体系或容量，EEC 参数和边界频率需重新标定，不可直接套用。
2. **单电池 vs 电池包**：本文结论基于单电池，v2 需考虑模组内单体不一致性对析锂边界的影响。
3. **正弦波 vs 逆变器 PWM**：v2 实际电流波形为逆变器 PWM 合成，非理想正弦，析锂判据的适用性需额外验证。
