## 2. 核心方法

### 2.1 问题定位与既有方案对比

作者归纳车载电内部加热三类路线：(1) 专用车载电路（buck-boost/H桥/LC谐振，硬件成本高）；(2) 集成于 BMS 均衡电路（非标准配置）；(3) 集成于电驱系统 EDS（复用现有硬件）。EDS 路线既有方案的局限：
- Du et al. [28]（=M4 参考的 CMI 架构同源文献）：常规驱动电路脉冲电流，硬件成本低、支持途中加热，但**升温速率不足**，且未深入研究途中加热实现。
- Li et al. [29,30]（DMSI/TMSI 双/三模块分离逆变器）：显著提升 RMS 加热电流，但**不支持途中加热**，且需改动电池包物理架构。
- Zhu et al. [31]（IBSH 集成电池自加热器）：升温快，但转矩抵消需将转子永磁磁通方向对齐激励磁通方向，**限制适用场景**，且不支持途中加热。

### 2.2 NPBH 拓扑与工作原理（核心创新点）

- **拓扑**（图2）：电池包拆分为串联的两个相同模块 E1、E2（共同提供直流母线电压）；DC-link 电容 C；三相全桥逆变器（Q1-Q6）驱动 PMSM；**电机中性点通过一个开关 SW 与电池包中点相连**。这是与传统驱动电路相比唯一增加的硬件——**仅一个开关**。
- 加热时 SW 闭合：E1、E2 与电机绕组之间的储能交替充放电。绕组电流的**共模分量**（zero-axis current i0）作为加热电流，**差模分量**产生驱动转矩，二者天然解耦。
- 通过零电压矢量（000 或 111）的同步控制精确调节 i0；一个开关周期内的电流演化分为 4 个阶段（图3，以转子电角 π/2 为例）：
  - 阶段1：Q1/Q2/Q3 导通，驱动电流处于续流状态，转矩渐减，i0 增大；上模块 E1 充电、下模块 E2 放电；绕组储能，DC-link 电容释能。
  - 阶段2：Q3关/Q6开，驱动转矩增大，i0 基本不变。
  - 阶段3：Q1关/Q4开，与阶段2类似，转矩继续增大，i0 基本不变。
  - 阶段4：Q2关/Q5开，驱动电流续流，转矩和 i0 均减小；E1放电、E2充电；绕组释能，DC-link电容储能。
- 闭环下 i0 跟踪正弦参考（PI 控制器），E1/E2 参数相同时 i1、i2 幅值相等相位相反。加热电流幅值 `Iheat=|i1|=|i2|=3/2|i0|`（式1，忽略线路和逆变器损耗）。

### 2.3 控制方法

- 总控制逻辑（图6）：加热策略根据电池 SOC、温度 Tb 确定 `Iheat` 和频率 `f` 的最优值；途中加热时还受驱动转矩 `Tem`、电池电压 `Udc`、车速影响。
- 加热电流占空比 `αh`（式2）：`αh = (2/Udc)·[K0,P(i0,ref-i0) + K0,I∫(i0,ref-i0)dt]`——PI 电流闭环。
- 驱动电流占空比 `αu,drv/αv,drv/αw,drv`（式3）：由 FOC 计算的相电压通过双极 SPWM 得到。
- 最终每相占空比 = 加热分量 + 驱动分量的叠加（式4）：`αu=αu,drv+αh` 等。
- 途中加热时驱动电流本身也会加热电池，此效应在加热策略中考虑（未展开具体耦合公式）。

### 2.4 电池电热模型（二阶 ECM，与 M5 规划 2.5 节"电池频域电热模型"直接对应）

- 二阶等效电路模型（图7）：`UOCV`（开路电压）、`R0`（欧姆内阻）、`R1/C1`、`R2/C2`（一/二阶极化阻抗）。
- **电池阻抗（式5）**：
  ```
  Z(Tb,SOC,f) = [R0 + R1/(1+(2πf)²R1²C1²) + R2/(1+(2πf)²R2²C2²)]
                - j[2πf·R1²C1/(1+(2πf)²R1²C1²) + 2πf·R2²C2/(1+(2πf)²R2²C2²)]
  ```
  （已通过公式图像视觉核验：虚部第二项分子确为 `R2²C2`，与第一项 `R1²C1` 结构对称，此前文本提取阶段的"疑为排版丢失"标注系提取误差，非原文问题，现已更正）
- **温度演化（能量守恒，式6）**：`m_b·C_p·dT_b/dt = q_h - q_d`
- **产热项（式7）**：`q_h = Id²·Re(Z)|f=0 + Ih²·Re(Z)|f=fh`，其中：
  - `Id = 3|Us,drv|·Idrv·cosφ / (2·Udc·ηe)`（驱动电流对应的电池侧 RMS 分量，φ 为电机功率因数，ηe 为电驱效率）
  - `Ih = Iheat/√2`（加热电流 RMS 值）
  - 忽略瞬态转矩变化对产热的影响。
- **散热项（式8）**：`q_d = h·Ab·(Tb-Ta)`
- **SOC 演化（式9）**：`dSOC/dt = Id/Cb`（加热电流均值为零，不影响 SOC；仅驱动电流分量消耗容量）

### 2.5 转矩消除的解析证明（核心卡点：本文档独有，2024templock 未涉及）

- 忽略漏感，定义自感/互感（式10）：
  ```
  Luu = Ls0 + Ls2·cos(2θ)
  Lvv = Ls0 + Ls2·cos(2(θ-2π/3))
  Lww = Ls0 + Ls2·cos(2(θ+2π/3))
  Luv=Lvu = -Ls0/2 + Ls2·cos(2(θ+2π/3))
  Luw=Lwu = -Ls0/2 + Ls2·cos(2(θ-2π/3))
  Lvw=Lwv = -Ls0/2 + Ls2·cos(2θ)
  ```
  其中 `Ls0=(2/π·Nph·kw)²·(τl/p)·λδ0`，`Ls2=(1/2)(2/π·Nph·kw)²·λδ2`
- 电磁转矩（式12）：`Tem = -p·[i_h]^T·(Ls·[i_h] - ψf·[sinθ,sin(θ-2π/3),sin(θ+2π/3)]^T)`
- 加热电流单相分量（式13）：`i0=iu,h=iv,h=iw,h=(2/3)Iheat·sin(ωt)`（三相共模、同幅同相）
- 代入后（式14）推导出 `Tem≡0`，**恒等于零，与转子电角 θ、电流相位 ωt 均无关**——这是全文最核心的解析结论：因为三相共模电流对磁链和二阶谐波电感项的贡献在三相求和后各自相消（`sinθ+sin(θ-2π/3)+sin(θ+2π/3)=0` 及类似的二阶谐波项恒等式）。
- **对比 IBSH [31]**：IBSH 需要将转子磁通方向对齐激励磁通方向（`θ=kπ`）才能消转矩，是**位置相关**的特例消除；NPBH 是**位置无关**的恒等消除，这是二者的本质区别，也是 NPBH 论文反复强调的优势点。

### 2.6 加热与驱动电流解耦的解析证明

- 总相电流（式15）：`iU=iu,h+iu,drv=(2/3)Iheat·sin(ωt)+Idrv·cos(θ+φ)` 等（驱动电流为常规 FOC 正弦分量）。
- Clarke-Park 变换（式16）：`[id,iq,i0]^T = C·[iU,iV,iW]^T = [Idrv·cosβ, Idrv·sinβ, (2/3)Iheat·sin(ωt)]^T`
  - id、iq 完全由驱动电流幅值 `Idrv` 和相位角 β 决定；i0 完全由加热电流幅值 `Iheat` 和相位 ωt 决定——**二者在 dq0 变换后天然解耦**，不存在交叉项。
- 转矩重写（式17）：`Tem = (3/2)·p·iq·[id(Ld-Lq)+ψf]`——**转矩表达式中不含 i0**，进一步验证加热电流对驱动转矩零影响。

## 5. 实验设计

### 5.1 测试平台

- EDS 台架 + 电池恒温箱（图8）；预热用方形 58Ah 电池+3个10µH电感模拟绕组；途中加热用圆柱 4Ah 电池+实际电机。

### 5.2 转矩消除验证（表3）

- 转子角固定 0°，`Iheat=0.75/1.5/3C`，`f=100Hz`：验证不同加热电流幅值下的转矩波动。
- `Iheat=1.5C`，`f=100Hz`，转子角 `0/7.5/15/22.5/30°`：验证不同转子位置下的转矩波动（角度周期 = 360°/(4对×3相) = 30°）。
- 结果：转矩波动幅值随 `Iheat` 增大而增大，随转子角呈周期性变化，但**在所有测试条件下波动幅值均小于 0.2Nm**，验证转矩消除有效性（工程可接受，非严格数学零，因实际电机结构存在漏感等未建模因素）。

### 5.3 解耦控制验证（表4，四组独立变量测试）

- 转速 100/150/200rpm（`Tem=2Nm, Iheat=1.5C, f=50Hz`固定）
- 转矩 0/3/6Nm（`n=100rpm, Iheat=1.5C, f=50Hz`固定）
- `Iheat` 0/1.125/2.25C（`n=100rpm, Tem=4Nm, f=50Hz`固定）
- `f` 10/30/50Hz（`n=100rpm, Tem=4Nm, Iheat=1.5C`固定）
- 结果：驱动侧参数变化对 i0 无影响，加热侧参数变化对驱动转矩无影响，验证解耦。
- 另有 WLTC 工况仿真验证（图15-17）：启动后前10分钟内，单相 i0 幅值从 50A 起步、每100s递增10A；频率前5分钟50Hz、后5分钟100Hz。仿真结果显示速度/转矩跟踪精确，加热电流不受驱动电流影响。

### 5.4 途中加热定常工况验证（表5）

- 转速固定 100rpm，初始温度 -20℃，加热时长 10min：
  - 转矩变化 2/4/8Nm（`Iheat=2C, f=50Hz`固定）：驱动转矩越大，电池输出电流越大，产热越多，升温越快；模型预测 RMS 误差 0.37℃(2Nm)/0.33℃(4Nm)/0.75℃(8Nm)。
  - `Iheat` 变化 1/1.5/2/3C（`Tem=2Nm, f=50Hz`固定）：终温分别 -17.39/-14.47/-11.19/0.68℃；RMS误差 0.19/0.30/0.28/0.25℃。
  - `f` 变化 50/100/200Hz（`Tem=2Nm, Iheat=3C`固定）：终温 0.68/-2.94/-4.04℃；RMS误差 0.59/0.4/0.38℃。频率越低、阻抗实部越大、产热越多——与式5的频域阻抗特性一致。

### 5.5 途中加热组合工况验证（表6，两种驾驶风格）

- 激进驾驶：4阶段，转矩从8Nm渐降至4Nm，转速从100渐升至300rpm，`Iheat` 1.5->2->3C，`f` 50->200Hz；RMS误差 0.53℃。
- 温和驾驶：5阶段，转矩恒定在2-4Nm波动，转速100-300rpm，`Iheat` 3->3.6C（更早更大加热电流因驱动转矩更小），`f` 从200Hz逐步降至50Hz（随电池升温逐步降频以最大化产热同时防止初期高SOC电压越限）；RMS误差 0.82℃。

### 5.6 预热优化策略验证（图20）

- 初始温度 -30℃，`Iheat=3.6C`固定，SOC 30%/60%/80% 三组对比：
  - 30%和60% SOC：升温过程接近，**平均 RTR 达 4.76℃/min**（-30->0℃）——这是 Table 8 对比表中 NPBH 的 RTR 数值出处。
  - 80% SOC：升温明显更慢，因高 SOC 下 EIS 特性在中等 SOC 区间变化较小，而高/低 SOC 区间变化更剧烈；高 SOC 时 OCV 接近充电截止限，需大幅提频降低阻抗以满足电压约束，代价是产热降低。

## 6. 结论边界（不可外推项）

1. **硬件改动边界**：NPBH **不是零硬件改动**方案，需要在电机中性点和电池包中点之间新增一个开关 SW，且电池包需要拆分为两个可引出中点的模块（E1/E2 串联）——这一点与"复用现有驱动电路"的表述有细微差别，实质是"最小硬件改动"（1个开关），而非完全零改动。这与 M4 当前 CMI 架构（单包、无额外开关、纯控制层脉冲）在硬件层面**不等价**，v2 若采用 NPBH 拓扑需评估电池包中点引出的工程可行性，不能仅在控制层模拟。
2. **验证层级**：转矩消除和解耦验证在 1kW 小电机台架完成；途中加热温度验证在圆柱 4Ah 小电池上完成；预热大电流验证在 58Ah 方形电池但用**分立电感替代电机**（非真实电机绕组），因预热阶段电机静止无背电动势，只需模拟绕组感抗——这意味着预热验证并未在"电池+真实电机+逆变器"完整闭环下进行，而是电池+等效感性负载。
3. **RTR 4.76℃/min 的适用条件**：仅在 30%/60% SOC、`Iheat=3.6C`（相对于 4Ah 圆柱电芯的箱体电流倍率）条件下取得；80% SOC 下显著更低。Table 8 对比表中列出的单一 RTR 数值是特定条件下的代表值，不是该方法在所有 SOC/电流下的保证值。
4. **能耗指标 0.2%/℃**：与 IBSH（0.201%/℃）几乎一致，暗示能耗指标主要由电池内阻决定（与转矩消除方式关系不大），但原文未展开能耗指标的计算口径细节（未给出类似 2024templock 式16-20 的显式能量平衡分解公式），该数值需结合 Table 8 来源文献 [33]（Gao 2025 姊妹论文）交叉核实。
5. **析锂/寿命安全**：全文未讨论负极电位、析锂判据或循环寿命影响；4.76℃/min 高倍率加热（3.6C 量级）下的析锂风险未被评估，属于本文结论边界之外，v2 若采用 NPBH 拓扑仍需依赖其他文献（如 Jian 2023）的析锂判据独立评估。
6. **转矩消除的"恒等于零"是理想化解析结果**：基于忽略漏感、忽略 PWM 调制引起的高频谐波的假设（原文式13后"Neglecting the harmonics caused by PWM modulation"）；实测转矩波动 <0.2Nm 的非零残差正来自这些被忽略的因素，作者在结论中明确指出实际应用中因逆变器开关不一致可能导致电流不平衡、逆变器退化问题，列为未来工作。
7. **未评估拓扑演进的工程成本**：本文未讨论电池包拆分为 E1/E2 两个可引出中点模块对现有电池管理系统（均衡、SOC估计等）的影响，也未讨论开关 SW 的选型、耐流耐压要求或失效模式。

## 8. 对 v2 建模的参考价值（初步，供任务 2 取舍清单细化）

- **可迁移的方法**：二阶 ECM 频域阻抗模型（式5，M5 规划已列为"路线C"核心参考）、频域产热叠加模型（式7，区分驱动电流欧姆产热与加热电流频域产热两项）、最优频率-SOC-温度查表策略（Table 7，来源需标注为二次引用姊妹论文 [33]）。
- **可迁移的解析方法论**：转矩恒零证明的推导思路（电感矩阵二阶谐波项在三相求和中相消）可用于评估 v2 若改用中性点拓扑时的转矩解耦充分性，即使不直接采用 NPBH 拓扑本身。
- **拓扑决策点**：NPBH 拓扑需要电池包物理拆分为两个可引出中点的模块 + 新增开关，这是**硬件级变更**，超出"控制层改进"范畴。若 v2 近期目标是"在 M4 CMI 硬件基础上只改控制策略"（对应 M5 规划 3.4 节步骤2的"路线A+B"），则 NPBH 拓扑应放入 M5 规划已列的"远期评估（路线E）"，不适合近期直接采用；dSVPWM（2024templock）由于纯控制层改动，与近期路线更契合。
- **析锂判据缺口**：两篇文献均未提供析锂判据，v2 的 `DeviceProtection_Evaluable` 可评估路径仍需依赖 Jian 2023 等专门析锂文献，不能从本两篇文献获得。
