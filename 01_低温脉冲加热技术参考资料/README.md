## 文件分组与重要性分级

基于 `00_项目管理/M5_文献评估报告_v01.md` 和统一证据表更新。分级表示对 M5 仿真规划的作用，不表示分析文件的写作深度，也不表示文献参数可以直接成为设备默认值。

| 分级 | 文献 | M5 作用 |
|---|---|---|
| **A1 核心主参考** | 2024templock、2024 IEEE TTE | 首版无新增硬件 dSVPWM（死区空间矢量脉宽调制）换流控制、温度监督、电压快速回撤和 DC-link 边界 |
| **A2 核心模型/安全/基线参考** | 2026bit NPBH、2021 iScience、Jian 2023、Zhang 2015 | 二阶 ECM/频域热模型、CMI（Combined Module Invert）基线、DMSI 对照、析锂判据和集总热平衡 |
| **B1 重要替代/辅助参考** | 2025 BYD 四线制、CN114789679A | N 线拓扑全场景工程基准、RMS 闭环和虚拟标定接口 |
| **C1 辅助/对照材料** | CN112977173A、ATIC | 开环反例、储能/续流信号流、损耗 map 和电化学仿真目标能力 |
| **辅助工程基准** | Shell 2022/2026、迅湃、PMSM 背景及未深读参数文献 | 充电桩侧接口、工程量级和背景，不进入车载电驱首版主线 |

首版路线为：单包三相三线标准逆变器 + PMSM 硬件边界内，采用 dSVPWM（dead-zone Space Vector PWM，死区空间矢量脉宽调制）换流型主控制；频域电热和析锂判据分别作为模型层和安全层；NPBH、BYD 四线制和 DMSI 作为后续拓扑替代。M4 对应 CMI（Combined Module Invert，文献原称）架构下的开环低频 d 轴脉冲电流自加热路线。两篇 Claude 深度分析文件 `2024templock...` 和 `2026bit...` 已单独在逐篇报告中标记其高证据权重；其余文件仍按各自原件核验状态使用。

所有文献参数进入 v2 前必须保留 `LITERATURE` 身份、原文出处、适用范围和替换影响。`DeviceProtection_Evaluable` 不因仿真运行成功自动变为 1。

## 建模引用纪律

1. 文献只能支撑其实际研究范围，不得把电芯/小模组结论直接外推为整车三包并联可行。
2. 采用频域发热模型时，优先查找 `Re(Z_bat(T, SOC, f))` 或等效阻抗依据；没有 EIS 数据时要标记为待确认假设。
3. 涉及低温回充半周、析锂和寿命影响时，必须单独列出安全证据，不把温升速度当作安全证明。
4. 涉及电驱方案时，必须把电池支路电流、母线电流、电机相电流、逆变器电流和热等效电流分开表达。
5. 对外报告引用本目录资料时，应同时记录文件名、年份、适用对象、可引用结论和不可外推边界。
6. 文献参数进入 v2 模型须标 `LITERATURE` 来源，记录原文页码/表格/公式编号。
