# 零维脉冲加热0528参数需求清单

## 1. 电池参数

| 类别 | 需要参数 | 用途 | 优先级 |
|---|---|---|---|
| 架构 | 双支路/三支路实际连接方式、每支路串并联、支路是否可独立切换 | 判断可形成的电气回路 | 高 |
| 基础参数 | 单体容量、支路额定电压、总能量、支路质量、热容估计 | 电流、SOC、温升计算 | 高 |
| 电性能 | SOC-OCV曲线、低温DCIR/HPPC、不同SOC和温度下阻抗 | 电池发热计算 | 高 |
| 限制 | 低温充放电脉冲窗口、BMS支路电流限制、允许母线电压波动 | 约束与可行区展示 | 高 |
| 脉冲能力 | 高频脉冲允许峰值/RMS电流、不同频率和占空比下的持续时间边界 | 电流幅值扫描、限流和析锂风险边界 | 高 |
| 安全 | 析锂边界、低温大电流试验结果、寿命影响数据 | 后续深化 | 中 |

## 2. 电机/MCU参数

| 类别 | 需要参数 | 用途 | 优先级 |
|---|---|---|---|
| 电机 | Ld/Lq/Rs随电流和温度的变化、堵转可用电感、绕组温度边界 | 电流波形与铜耗 | 高 |
| MCU | 开关频率范围、峰值/有效值电流、零速脉冲输出能力 | 电气约束 | 高 |
| MCU | 目标脉冲电流命令、调制方式、限流降额曲线、热保护策略 | 幅值扫描和控制器边界建模 | 高 |
| 拓扑 | 双电机控制器内部直流母线关系、是否支持单支路集中脉冲 | 判断整体同步和集中单支路方案可实施性 | 高 |
| 损耗 | 逆变器效率图、开关损耗参数、冷却能力 | 初步效率和器件压力 | 中 |

## 3. 整车/高压盒参数

| 类别 | 需要参数 | 用途 | 优先级 |
|---|---|---|---|
| 高压连接 | 双/三支路与高压盒、母线、电驱的连接图 | 架构建模边界 | 高 |
| 切换器件 | 继电器/IGBT/MOSFET可用性、动作频率、电流和电压等级 | 集中单支路方案和高压盒路径可行性 | 高 |
| 保护 | 预充、熔断器、绝缘监测、故障降级要求 | 后续安全策略 | 中 |
| 成本边界 | 是否允许新增电感、开关器件和线束 | 工程实施判断 | 中 |

## 4. 当前简化模型默认值

当前模型会先使用3806旧模型经验值作为可运行占位：约618V平台、单支路324Ah、1250Hz参考频率、Ld约0.26mH、电机支路550Arms/778Apeak边界。

这些默认值仅用于粗筛，不作为新车型最终参数。收到真实数据后，应先更新`02_参数库`，再分别运行`03_模型源码/run_dual_branch_pulse_heating.m`和`03_模型源码/run_triple_branch_pulse_heating.m`。
