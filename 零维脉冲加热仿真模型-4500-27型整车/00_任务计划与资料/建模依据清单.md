# 零维脉冲加热0528建模依据清单

## 1. 资料范围与使用原则

资料来源目录：`E:\agentwork_codex_0509\建模依据资料`

本清单用于把已下载资料转化为可用于 MATLAB 零维模型、COMSOL 电路结构模型、Simulink/Simscape 自建方程的建模依据。重点不是全文综述，而是明确每个模型模块的控制方程、源项、边界条件、适用范围和不能外推的地方。

统一原则：

- 简单开关模型只计算电压、频率、占空比、回路电阻和电感自然形成的电流。
- 逆变器模型可以给目标电流或电压矢量，但必须通过电流环、PWM/SVPWM、母线电压和硬件边界实现。
- 电池发热不能只用经验常数，应至少追溯到 `I_rms^2 R`、`I^2 Re(Z)` 或电化学/电热模型。
- 析锂边界在没有目标电芯低温 EIS、HPPC 或电化学参数前，只能作为风险接口和参考边界，不作为确定结论。

## 3. 可采用的核心方程与模型模块

### 3.1 电机绕组与FOC控制

**依据资料**

- NXP `3-phase Sensorless PMSM Motor Control Kit...`，第4-6页：PMSM三相电压方程、d/q同步旋转坐标方程、交叉耦合补偿、电流环PI增益推导。
- TI `SensorlessFieldOrientedControlof3-PhasePermanent.pdf`，第6-8页：Clarke/Park变换、FOC基本结构、测量相电流到 `i_d/i_q`，PI输出 `V_d/V_q`，逆Park和SVPWM驱动逆变器。
- Microchip `DS00004398`，第12-13页：SVPWM、两电平逆变器8种状态、6个有效矢量和2个零矢量，用相邻有效矢量和零矢量合成目标空间矢量。

**可用于建模的方程链**

1. 三相电流经 Clarke/Park 变换得到 `i_d`、`i_q`：

```text
i_d = i_alpha cos(theta) + i_beta sin(theta)
i_q = -i_alpha sin(theta) + i_beta cos(theta)
```

2. PMSM d/q轴电压方程使用完整形式作为Simulink依据：

```text
u_d = R_s i_d + L_d di_d/dt - omega_e L_q i_q
u_q = R_s i_q + L_q di_q/dt + omega_e L_d i_d + omega_e psi_PM
```

3. 堵转或低速脉冲加热时，如果 `omega_e` 近似为0，可退化为RL支路：

```text
u_d ~= R_s i_d + L_d di_d/dt
u_q ~= R_s i_q + L_q di_q/dt
```

4. 电流环PI：

```text
e_d = i_d_ref - i_d
e_q = i_q_ref - i_q
v_d_ref = Kp_d e_d + Ki_d integral(e_d) + decoupling/feedforward
v_q_ref = Kp_q e_q + Ki_q integral(e_q) + decoupling/feedforward
```

5. SVPWM合成目标电压矢量：

```text
U_out = (T_k/T_s) U_k + (T_l/T_s) U_l + (T_z/2T_s) U_0 + (T_z/2T_s) U_7
T_s = T_k + T_l + T_z
```

**适用条件**

- Simulink/Simscape中应采用“目标电流 -> PI电流环 -> 电压指令 -> SVPWM/PWM -> 逆变器 -> 电机绕组电流”的链条。
- MATLAB零维阶段可将堵转/低速近似为 `R_s-L_d` 或 `R_s-L_q` 支路，但必须标注这是简化后的电压驱动模型。
- COMSOL电路结构模型可用受控脉冲电压源或受限电压源近似控制器输出，不应直接用理想电流源证明逆变器可实现性。

**不能外推的边界**

- `omega_e ~= 0` 只适合堵转、低速或专门控制零平均转矩的脉冲加热，不能直接代表正常驱动工况。
- 恒定 `L_d` 只适合粗筛；高电流下应考虑磁饱和 `L_d(I,T)`。
- 目标电流是否能达到，必须检查母线电压、调制上限、PI限幅、采样周期、PWM频率和过流保护。

### 3.2 脉冲电流调幅与逆变器实现

**依据资料**

- `CN114789679A...`，第1-8页：将 `Id_ini/Id_fb` 输入PI得到 `Ud_req`，将 `Iq_tag/Iq_fb` 输入PI得到 `Uq_req`，再经Park逆变换得到 `U_alpha/U_beta`，输入SVPWM控制三相桥臂。
- `一种电动汽车及其动力电池脉冲加热系统和加热方法_陈富.pdf`，第3-8页：根据转子位置对 `Ud` 和预设 `Uq` 做Park逆变换，SVPWM输出六个功率开关脉宽；脉冲信号周期为 `2/f`，前后半周期控制加热过程。
- `Voltage-Feedback...`，第3-7页：用电池、直流母线电容、逆变器和电机定子建立耦合模型；改变零矢量占空比 `d0` 调节AC加热强度；用双PI控制电池峰谷电压。

**可用于建模的控制结构**

```text
I_target or Id_ref/Iq_ref
    -> current feedback Id_fb/Iq_fb
    -> PI current controller
    -> Ud_req/Uq_req
    -> inverse Park
    -> U_alpha/U_beta
    -> SVPWM/PWM
    -> inverter switching states
    -> motor winding current and battery current
```

对电压反馈型脉冲加热，可采用：

```text
battery voltage peak/trough target
    -> PI controller
    -> zero-vector duty d0 or modulation command
    -> PWM generator
    -> inverter + PMSM + DC-link circuit
```

**适用条件**

- MATLAB零维中的 `current_amplitude_scale` 可作为“目标电流系数/调制降额系数”，用于研究电流幅值对加热和安全边界的影响。
- 若要更物理，应把 `current_amplitude_scale` 拆成 `I_target_A` 与 `modulation_index` 或 `V_cmd`。
- Simulink模型应把PI限幅、抗积分饱和、电压饱和、PWM周期和采样同步作为控制器边界。

**不能外推的边界**

- 专利中的策略说明可作为结构依据，但不能替代目标车型的控制器实参。
- 文献中的 `10-15 kHz`、`10 kHz` 等频率来自特定样机和论文设定，不能直接覆盖本项目当前 `1250 Hz` 堵转载频边界。
- `d0` 零矢量调幅策略依赖具体PWM和逆变器拓扑；如果供应商控制器不开放该接口，只能作为Simulink研究方案。

### 3.3 电驱系统参与电池加热的电流路径

**依据资料**

- `Drive circuitry...`，第3-13页：提出CMI/DMSI两种驱动电路加热模式，说明电机线圈参与能量转移；第5-7页给出 `i_q=0`、相电流RMS限制、电池热功率等方程；第21-22页说明实验中使用SVPWM和逆变器损耗考虑。
- `A Novel Battery Temperature-Locking...`，第3-6页：基于PMSM FOC和SVPWM设计实现AC电池电流的PWM波形，提出行驶中温度锁定控制。
- `A Novel Battery Self-Heating Method Based on Drive Circuit Reconfiguration...`，第4-5页：通过驱动电路重构形成零轴电流 `i0`，在电机绕组和电池模块之间循环，用PWM关系调节加热电流。

**可用于建模的工程关系**

1. 零转矩或低转矩约束：

```text
i_q = 0
T_e ~= 0
```

2. 电机/逆变器RMS电流限制：

```text
max(I_A_rms, I_B_rms, I_C_rms) <= I_motor_allowed
```

3. 电池发热功率：

```text
P_heat = I_battery_rms^2 R_battery
```

4. DMSI/CMI类拓扑要明确支路电流路径，不允许只用总电流平均分配代替真实拓扑。

**适用条件**

- 本项目四类拓扑应先定义真实电流路径，再进入电流和发热计算。
- 双电机集中单支路必须重点核对单支路RMS/峰值电流、高压盒路径和电池局部发热。
- 三支路整体同步必须重点核对支路阻抗差异导致的不均流。

**不能外推的边界**

- 论文中DMSI/CMI拓扑不一定等同于本项目三包/双电机拓扑，不能直接套用其电流放大倍数。
- 行驶中加热案例含驱动转矩叠加，本项目若以停车/堵转加热为主，应只引用其逆变器和电流路径建模方法。

### 3.4 电池热源项与热平衡

**依据资料**

- `Internal heating...frequency domain`，第3-5页：频域热生成模型，交流电流幅值和阻抗实部共同决定热生成率；给出集总热方程并用EIS拟合参数。
- `Modeling and analysis of high-frequency...`，第4-6页：开关频率决定RMS加热电流；一阶ECM包含 `R_B`、`R1C1`、`L_b`、`V_OC`；热模型包含欧姆热和高频电荷传输热。
- `Drive circuitry...`，第6页：电池热功率采用 `P_heat = I_eff_B^2 R_bat`。

**可用于建模的源项**

1. 低阶零维和COMSOL初版可用欧姆热：

```text
Q_dot_bat = I_rms^2 R_bat(T,SOC)
```

2. 有EIS频域数据时，优先用阻抗实部：

```text
Q_dot_bat = I_rms^2 Re(Z(T,SOC,f))
```

如果文献使用交流电流幅值 `A`，则：

```text
I_rms = A / sqrt(2)
Q_dot = (A / sqrt(2))^2 Z_Re
```

3. 集总热平衡：

```text
m c dT/dt = Q_dot_bat - h S (T - T_amb)
```

4. 高频AC加热可参考扩展热模型：

```text
m c dT/dt + h S (T - T_amb) = I_rms^2 R_B + k f_sw I_rms
```

**适用条件**

- 当前MATLAB零维模型可继续使用 `I_rms^2 R_bat`，但应明确这是低阶欧姆热源项。
- 若后续取得目标电芯低温EIS，建议把 `R_bat(T,SOC)` 升级为 `Re(Z(T,SOC,f))`。
- COMSOL模型中电池内阻发热可作为 Lumped Thermal System 的热源；若加入ECM，热源应由电路元件损耗自动或显式传入热模型。

**不能外推的边界**

- 文献中的EIS和热参数来自特定18650/方形电芯，不能直接作为本项目324Ah电芯定量参数。
- 高频电荷传输热项中的 `k` 是实验拟合系数；无目标电芯高频实验前，不应作为最终定量热源。
- `I_rms^2 R` 不覆盖可逆熵热、极化动态、浓差极化和析锂副反应热。

### 3.5 析锂与安全边界

**依据资料**

- `Internal heating...electro-thermal modeling`，第5-9页：TC-SEM模型、温度修正参数、负极固液相电位差析锂判据、电流幅值约束面。
- `Electrochemical Modeling...Safe Alternating Current Heating...`，第29-31页：用P2D电化学模型确定AC加热安全边界；电流幅值随频率和温度变化；超过边界会导致容量快速衰减和负极析锂。

**可用于建模的安全判据**

1. 负极析锂基础判据：

```text
phi_s - phi_l > 0
```

2. 电流幅值约束可表示为：

```text
I_limit = f(T, SOC, frequency)
```

3. 若只有参考资料而无目标电芯参数，则模型中应输出：

```text
plating_reference_margin = I_limit_ref / I_actual
```

并标注为参考边界或待标定边界。

**适用条件**

- MATLAB零维模型可先保留析锂风险接口和边界线展示。
- Simulink模型可把析锂边界作为BMS/VCU允许加热电流上限输入。
- COMSOL模型可显示风险标志，但不应在无标定参数时自动降额。

**不能外推的边界**

- P2D/TC-SEM边界必须依赖目标电芯参数、温度、SOC、频率和老化状态。
- 文献中的 `0.2 Hz`、`0.5C`、`0.05C`、某温度边界不能直接等于商用车高频脉冲加热边界。
- 30s/60s规格书窗口只能作为低频/准直流参考，不等同于高频脉冲析锂边界。

## 4. 映射到本项目三类模型

### 4.1 MATLAB零维模型

推荐保留两层计算：

```text
第一层：Vdc, R_loop, L, f, duty -> I_raw_peak, I_raw_rms
第二层：I_target/current_amplitude_scale + 限制边界 -> I_actual_peak, I_actual_rms
```

依据：

- `u ~= R i + L di/dt`：来自NXP PMSM d/q方程低速简化。
- `current_amplitude_scale`：解释为目标电流系数或调制降额，依据CN114789679A的Id/Iq闭环调幅和Voltage-Feedback的零矢量占空比调幅。
- `P_battery = I_rms^2 R`：依据AC heating频域热模型和Drive circuitry热功率表达。
- 限制边界：电机/控制器RMS和峰值、电池支路RMS和峰值、频率、析锂参考、母线/高压路径。

必须新增或保留的输出字段建议：

| 字段 | 目的 |
|---|---|
| `I_raw_peak_A`、`I_raw_rms_A` | 开环电压驱动自然电流 |
| `I_target_A` 或 `current_amplitude_scale` | 控制目标或调制降额 |
| `I_actual_peak_A`、`I_actual_rms_A` | 限制后的实际电流 |
| `V_required_V` | 达到目标电流所需等效电压 |
| `V_available_V` | 母线和调制边界下可用电压 |
| `limiting_factor` | 电压、MCU、电池、频率、析锂等限制来源 |

### 4.2 COMSOL电路结构模型

推荐结构：

```text
Battery OCV + R_bat(T,SOC) + branch switch
    -> controlled pulse voltage source or limited inverter source
    -> R_s + L_d/L_q motor branch
    -> Lumped Thermal heat source Q_dot = I_rms^2 R_bat
```

依据：

- 电路瞬态：Voltage-Feedback中的电池-逆变器-母线电容-电机等效电路。
- 电机绕组：NXP PMSM d/q方程低速RL简化。
- 热源：AC heating频域模型和集总热方程。

COMSOL中不建议直接使用理想电流源作为正式逆变器模型。若为了热上限做扫描，可单独标注“理想电流源假设”，不得用于证明实车可实现。

### 4.3 Simulink/Simscape模型

推荐结构：

```text
Id_ref/Iq_ref or I_rms_ref
    -> current measurement
    -> Clarke/Park
    -> PI current loop with anti-windup
    -> voltage saturation
    -> inverse Park
    -> SVPWM/PWM
    -> inverter
    -> PMSM/RL branch
    -> battery ECM + thermal model
    -> safety supervisor
```

依据：

- 控制链：TI第6-8页、NXP第4-6页、Microchip第12-13页。
- 电池加热闭环：CN114789679A第1-8页、Voltage-Feedback第3-7页。
- 热和安全：AC heating、high-frequency AC heating、TC-SEM和Safe AC Heating。

必须包含的限制：

- 电压饱和和调制比限制。
- PI抗积分饱和。
- 采样周期和PWM周期。
- 峰值电流、RMS电流、过温、母线电压、SOC、析锂参考边界。

## 5. 当前模型可直接采用与待补充内容

### 5.1 可直接采用

- MATLAB零维：继续使用开环电压驱动 + 限流缩放结构。
- MATLAB零维：将 `current_amplitude_scale` 明确写成“目标电流或调制降额系数”。
- COMSOL：继续使用等效脉冲源 + 电机 `R-L` + 电池内阻热。
- Simulink：按FOC链条搭建控制策略，不用理想电流源代替逆变器。
- 图表和报告：同时显示原始电流、目标电流、实际电流、限制来源和热结果。

### 5.2 必须补充的工程参数

| 缺口 | 用途 | 影响 |
|---|---|---|
| 控制器电流环带宽、采样周期、PWM周期 | 判断目标电流能否跟踪 | 影响Simulink闭环可信度 |
| 控制器最大调制比、电压限幅、最小脉宽、死区时间 | 判断 `V_required <= V_available` | 影响电流调幅可实现性 |
| MCU峰值/RMS电流、过流阈值、热降额 | 安全限制 | 影响可用加热电流 |
| 电机 `L_d(I,T)`、`L_q(I,T)`、`R_s(T)` | 电流动态和损耗 | 影响频率-电流关系 |
| 电池低温EIS/HPPC，尤其 `Re(Z,T,SOC,f)` | 频域热源项 | 影响发热功率和温升 |
| 电池高频/脉冲析锂试验边界 | 安全边界 | 影响能否定量限流 |
| 高压盒、接触器、熔断器、线束电流限制 | 拓扑可实现性 | 影响集中单支路方案风险 |
| 逆变器损耗图或器件开关损耗参数 | 控制器热风险 | 影响持续加热能力 |

## 6. 模型验收检查

执行模型更新或后续建模时，至少满足：

- 电池热源有来源：`I_rms^2 R` 或 `I^2 Re(Z)`。
- 电机绕组有来源：PMSM d/q方程低速RL简化。
- 逆变器控制有来源：FOC/SVPWM链条和PI电流环。
- 电流调幅有来源：Id/Iq闭环、Ud/Uq调制或零矢量占空比调幅。
- 安全边界有来源：电机/MCU电流限制、电池电压/SOC/温度、析锂参考边界。
- 任何“可实现电流”都必须说明是否经过电压饱和、控制器限流、电池边界和高压路径边界检查。
