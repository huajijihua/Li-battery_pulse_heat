# 4500-27型整车零维脉冲加热阶段报告

更新时间：2026-06-11

## 当前模型边界

当前模型只保留 4500-27 型整车实车约束方案：

- 三包并联整体输出，单电机脉冲；
- 三包并联整体输出，双电机脉冲。

不再保留早期通用架构入口，也不再保留旧日期命名的占位参数和报告。

## 默认工况摘要

默认工况：`T_init = -20 C`，`SOC = 50%`，`f = 1250 Hz`，`duty = 0.50`，`R_heat_factor = 1.0`，`I_motor_rms_limit = 550 Arms`。

| 方案 | 初始温升速率 | 30min 平均温度 | 到 0 C 时间 | 30min 等效SOC消耗 | 初筛判断 |
|---|---:|---:|---:|---:|---|
| 三包并联单电机 | 0.366 C/min | -10.31 C | 未达到 | 1.84% | 温升偏弱 |
| 三包并联双电机 | 1.430 C/min | 9.01 C | 17.68 min | 5.32% | 建议进入参数补充和更细模型 |

## 主要结论

- 双电机方案在当前占位参数下具备明显温升能力，是后续 L1 控制/电热模型的主候选。
- 单电机方案 30 min 仍未达到 0 C，不建议作为主攻方案。
- 当前结果仍受电池高频等效阻抗、热边界、控制器电流闭环能力和电机/逆变器热限制影响，不能作为最终安全结论。
