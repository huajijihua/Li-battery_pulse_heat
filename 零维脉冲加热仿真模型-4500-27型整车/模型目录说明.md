# 零维脉冲加热仿真模型目录说明

本目录当前用于4500-27型整车低温脉冲加热零维集总粗筛。

当前主线已从早期“通用双支路/三支路架构粗筛”收敛为4500-27实车约束下的方案评估。4500-27参数表明确：3个电池包、非独立输出、并联输出、不可独立接入电驱。因此当前工程方案只按“三包并联整体输出”处理。

## 目录结构

| 目录 | 内容 |
|---|---|
| `00_任务计划与资料` | 建模计划、建模依据、参数需求、逆变器建模口径 |
| `01_原始数据` | 原始数据说明 |
| `02_参数库` | 参数表、参数构造函数、参数来源说明 |
| `03_模型源码` | MATLAB主程序和公共计算函数 |
| `04_仿真工况` | 温度、SOC、频率、占空比等工况定义 |
| `05_仿真结果` | 当前有效仿真结果输出目录 |
| `06_阶段报告` | 历史阶段报告 |

## 当前实车约束方案

1. 三包并联整体输出，单电机脉冲。
2. 三包并联整体输出，双电机脉冲。

早期“单电机给单支路”“双电机集中给单支路”等方案不符合4500-27当前参数表中的高压连接约束，不作为当前实车工程可行方案。

## 当前模型口径

- 零维模型用于原理筛选和方案排序，不替代控制策略开发、安全定型或可靠性验证。
- 电池发热按 `I_rms^2 * R_bat(T,SOC)` 粗筛；若后续取得EIS，应升级为 `I_rms^2 * Re(Z(T,SOC,f))`。
- 30s/60s窗口和析锂模型当前只作为参考边界展示，不能直接证明kHz脉冲安全。
- 电机/MCU电流、频率、母线电压和逆变器可实现性必须在后续更细模型中验证。
