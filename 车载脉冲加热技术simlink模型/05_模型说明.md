# 模型说明 v03-B2

日期：2026-07-07
当前主模型：`02_模型/pulse_heating_single_pack_v03.slx`
当前阶段：v03-B2 供电修复完成，进入控制整定与 v03-C 限制接口前置阶段。

## 1. 当前结论

`pulse_heating_single_pack_v03.slx` 是当前唯一保留的 Simulink/Simscape 主模型。模型使用官方默认 `Battery Equivalent Circuit` 数据缩放的 `216S1P` 通用等效包，DC-link 约 801 V，闭环和直接调制均可形成 PMSM 相电流。

v03-B2 已确认此前电流卡死不是控制器或 VSC/PMSM 本体问题，而是旧 Battery Pack Builder 生成资产参数化/初始化导致 DC-link 近零电压。旧 `battery_pack_252S1P` 生成资产、`pulse_heating_single_pack_v01.slx`、`pulse_heating_v02.slx` 和旧 KPI/参数脚本已从当前工作树删除；如需追溯，使用 Git 历史提交 `e23e0dc`。

当前不能声称工程可行或电池安全已验证。剩余主要问题是 PI/调制限幅未整定、KPI v02 未建立、电池低温 EIS/HPPC 和 BMS 边界缺失。

## 3. 当前模型口径

| 项目 | 当前值 | 说明 |
|---|---|---|
| 电池块 | `batt_lib/Cells/Battery Equivalent Circuit` | Simscape Battery 官方库 |
| 默认结构 | `216S1P` | 通用等效包，不是 4500-27 实车硬约束 |
| 初始 SOC | `0.50` | 结构验证工况 |
| 初始温度 | `278.15 K` | 官方默认温度表最低点；-20 C 需后续补数据 |
| DC-link 电容 | `1e-3 F` | 占位值 |
| DC-link 初始电压 | `801.9432 V` | 与 SOC 50% 官方 OCV 缩放一致 |
| 逆变器/执行器 | `Average-Value Voltage Source Converter (Three-Phase)` | 当前为平均值测试执行器，不代表开关级逆变器 |
| 控制链 | `Iabc -> Id/Iq(theta=0) -> PI -> Saturation -> DQ_to_ModWave -> VSC` | 已闭合但未整定 |
