### 0.4 v03-B2 调制/VSC 口径诊断（2026-07-07）

诊断目标：区分电流不响应是电池/DC-link、控制器、VSC 还是 PMSM 问题；同时区分参数数值问题和模块建模问题。

执行脚本：

- `03_脚本/diagnose_pulse_heating_v03b2.m`
- `03_脚本/diagnose_vsc_pmsm_isolation_v03b2.m`

诊断结果摘要：

| 测试 | 结果 | 判断 |
|---|---|---|
| v03-B 闭环 0 A | `IdRefRMS=0`、`IabcRMS=0`，但电池端传感器显示约 486 A 异常基线 | 控制器零命令本身正常；电池包/DC 区域存在异常状态 |
| v03-B 闭环 20/200 A | 控制器有非零调制输出，但 `Id_fb` 和 `Iabc` 仍近零 | 不是单纯 PI 增益太小；主电路没有得到有效 DC 电压 |
| v03-A 直接调制 0.2/0.8 | 仍几乎无 PMSM 相电流 | 问题不在闭环控制器，而在 DC 供电/电池包区域 |
| Battery_Pack 输出检查 | `batteryVoltage≈5e-8 V`，`socCell=1`，`batteryCurrent≈-487 A` | Battery Pack Builder 资产未按目标 252S/278Ah/约800V 初始化 |
| ModuleType1 参数读取 | `S=252`，但 `BatteryCapacityCell=27`、`batteryVoltage=0`、`vParallelAssembly=repmat(0,252,1)`、`socCell=1` | 结构是 252S1P，但参数/初始状态是默认未目标化口径 |
| VSC+PMSM 隔离测试 | 用理想 800 V DC 源替代 Battery_Pack 时，`IabcRMS≈174 A`，峰值约 423 A | VSC 和 PMSM 模块本体可产生电流；受阻点不在 VSC/PMSM 本体 |

结论：当前主模型的电流受阻首先发生在 Battery_Pack/DC-link 供电区域。根因是 `battery_pack_252S1P_lib/Modules/ModuleType1` 虽然结构为 252S1P，但仍使用默认参数和默认初始状态，尤其 `batteryVoltage=0`、`vParallelAssembly=0`、`BatteryCapacityCell=27Ah`，导致 DC 母线没有目标高压。简单在主模型实例上 `set_param` 状态目标没有改变仿真输出，说明生成库/Mask 初始化机制会覆盖实例设置。

问题分类：

- 不是控制器数值问题：闭环 20/200 A 和直接调制 0.8 都无有效相电流。
- 不是 VSC/PMSM 模块本体问题：理想 800 V DC 隔离测试可产生 174 Arms 量级相电流。
- 是电池包资产参数化/初始化问题：Battery Pack Builder 生成资产需要重建或在库级正确参数化，不能只在主模型实例上临时写状态目标。

执行修复：

1. 已将 `build_pulse_heating_single_pack_v03a.m` 中的 Battery_Pack 来源从 `battery_pack_252S1P/Pack1` 改为 `batt_lib/Cells/Battery Equivalent Circuit`。

## 1. 模型概述

单电池包-逆变器-PMSM 堵转脉冲加热 Simscape 第一版物理网络模型。走通"电压指令 → 受控电压源 → PMSM 堵转 → 电池 ECM+热"完整物质流，输出电流/功率/电压裕度 KPI。

第一版为**开环电压驱动验证**，未实现 FOC 电流闭环。电压指令为 300V@1250Hz 三相正弦波，直接驱动 Controlled Voltage Source。这是物质流验证的第一步，后续版本将替换为完整 FOC 电流闭环。

## 2. 模型架构

### 2.1 物理网络（Simscape 域）

```text
[电池 BEC (ECM + LumpedThermalMass)]
   │ +, - (electrical)
   ├── DC-link 电容 (1mF)
   └── 电气参考 (Elec_Ref)

[Controlled Voltage Source (Three-Phase)]
   │ 三相输出 (a, b, c)
   ├── Current Sensor (Three-Phase) → PS-Simulink Converter → Iabc_log
   └── PMSM (a, b, c)
        ├── 中性点 → 电气参考
        ├── 机械 R → Ideal Torque Source (T=0, 堵转)
        └── 机械 C → Mechanical Rotational Reference

[Solver Configuration] → 电气网络
```

### 2.2 信号流（Simulink 域，第一版简化）

```text
Va_gen (Sine 300V@1250Hz, 0°)  ─┐
Vb_gen (Sine 300V@1250Hz, -120°)─┤→ Mux → S-PS_Converter → Controlled VS
Vc_gen (Sine 300V@1250Hz, -240°)─┘

T_load_0 (Constant=0) → S-PS_Torque → Torque_Source (堵转 T=0)

PS-S_Converter_Iabc → Iabc_log (To Workspace)
```
