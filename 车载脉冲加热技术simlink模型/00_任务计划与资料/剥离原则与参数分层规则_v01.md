# 剥离原则与参数分层规则 v01

日期：2026-07-08
范围：本模型工作区全部参数、脚本、模型和文档

## 1. 核心目标

本项目定位为**通用车载脉冲加热系统仿真平台**，不是具体产品仿真或特定车型仿真。模型服务于结构规范、能量路径、控制链、限制接口、KPI 语义和验证闭环，不服务于某个具体车型、电池包、电机或历史频率点。

## 2. 剥离对象

以下内容不得进入默认模型参数、默认模型结构和默认结论：

1. 252S3P、278Ah、1250Hz 等具体产品/历史工况参数。
2. 4500-27 L0.5 中为了粗筛产生的车型绑定假设。
3. 旧 battery_pack_252S1P* 生成资产及其初始化问题。
4. 任何来源不可靠、不可追溯的公司内部参数。
5. 任何把通用平台结论写成具体产品可行性结论的表述。

允许这些内容出现在"历史说明/可选案例/背景风险"章节，但不允许出现在"默认参数/默认模型/当前结论"章节。

## 3. 参数分层规则

所有参数必须带来源标签。没有标签的参数不能进入默认模型。

| 层级 | 标签 | 含义 | 优先级 |
|------|------|------|--------|
| 平台默认 | `PLATFORM_DEFAULT` | 通用平台自洽匹配的默认参数，来自官方案例/文献/工程经验组合 | 默认模型直接使用 |
| 官方 | `OFFICIAL` | MathWorks 官方案例/官方库默认值 | 可直接使用，需注明版本 |
| 文献 | `LITERATURE` | 公开文献、专利中的量级参考 | 可直接使用，需注明来源领域 |
| 工程经验 | `ENGINEERING_DEFAULT` | 工程经验自洽匹配，非具体产品数据 | 可直接使用，需注明匹配逻辑 |
| 占位 | `PLACEHOLDER` | 明确缺口，不代表通过，不代表已知 | 值为 NaN 或默认占位值 |
| 历史案例 | `HISTORICAL_CASE` | 历史案例/公司资料 | 禁止进入默认模型；仅允许在背景说明或可选案例中出现 |
| 粗筛 | `L0.5_SCREENING` | 4500-27 L0.5 粗筛阶段参数 | 禁止进入默认模型；仅作为背景参考 |

### 3.1 标签判定规则

- 参数来源为 MathWorks 官方文档、官方示例、官方库默认值 → `OFFICIAL`
- 参数来源为公开期刊、会议论文、学位论文、专利 → `LITERATURE`
- 参数通过工程经验匹配使系统自洽（如电压等级、RC 常数量级） → `ENGINEERING_DEFAULT`
- 参数为平台默认参数集的组成部分，由官方/文献/工程经验组合而来 → `PLATFORM_DEFAULT`
- 参数无任何可信来源，当前无法获取 → `PLACEHOLDER`
- 参数来自公司内部资料、4500-27 L0.5、旧产品参数表 → `HISTORICAL_CASE` 或 `L0.5_SCREENING`

## 4. 1250 Hz 语义规范

1250 Hz 在当前通用平台模型中不作为"产品默认频率"，其角色限定为：

- `diagnostic_frequency_Hz`：诊断频率，用于验证控制链频率响应和 KPI 语义。
- `stress_case_frequency_Hz`：应力工况频率，用于测试限制接口行为。

任何文档、脚本、KPI 中不得将 1250 Hz 表述为"目标频率"、"默认频率"或"产品工作频率"。若后续需要高频诊断，应使用变量 `diagnostic_frequency_Hz` 而非硬编码常量。

## 5. 216S1P 语义规范

当前 216S1P 电池包配置解释为"平台默认高压电池等效包"，作用是为 DC-link 电压、控制器和 PMSM 工作区间提供粗匹配。不是任何具体产品的替代或等效。

## 6. PMSM 参数语义规范

当前 PMSM 参数（Ld, Lq, Rs, pm_flux_linkage, 极对数等）解释为"平台默认 PMSM 参数集"，来源为 `OFFICIAL`/`LITERATURE`/`ENGINEERING_DEFAULT`。不再标注为"继承 L0.5"。

接口契约中 PMSM 参数来源标签：
- Ld/Lq/Rs：当前为 `ENGINEERING_DEFAULT`，需在参数注册表中注明量级来源
- pm_flux_linkage：`PLACEHOLDER`
- nPolePairs：`ENGINEERING_DEFAULT`
- J/热质量：`PLACEHOLDER`

## 7. 污染检查门槛

每次进入下一阶段前，对默认路径（脚本、模型说明、接口契约、KPI 脚本）执行文本审计，检查是否残留以下污染项：

| 污染关键词 | 允许出现位置 | 禁止出现位置 |
|-----------|-------------|-------------|
| 252S | 历史说明/背景风险 | 默认参数/默认模型/当前结论 |
| 252S3P | 历史说明/背景风险 | 默认参数/默认模型/当前结论 |
| 278Ah | 历史说明/背景风险 | 默认参数/默认模型/当前结论 |
| 4500-27 | 历史说明/背景风险 | 默认参数/默认模型/当前结论 |
| 1250Hz 作为产品事实 | 不允许 | 不允许 |
| L0.5 参数继承为默认值 | 不允许 | 不允许 |
| battery_pack_252S1P | 历史说明 | 默认参数/默认路径 |
| "产品"、"车型"、"实车" 绑定到默认模型 | 不允许 | 不允许 |
| `[L0.5]` 标签 | 历史说明 | 接口契约/默认参数表 |

## 8. 保留方式

历史资料 / L0.5 / 公司参数不粗暴删除，而是隔离处理：

- 历史资料仅作为背景、风险提示、可选外部案例。
- 通用平台默认参数仅来自官方案例 + 文献 + 工程经验匹配。
- 主模型 v03/v04 仅依赖通用平台默认参数。
- 真正要删的是 active 默认路径里的依赖和表述，不是历史证据链。

## 9. 验证标准

剥离完成后，至少满足：

1. 主模型 `model_check(all)` healthy。
2. 最小仿真仍能跑通。
3. 默认参数有来源标签。
4. 模型说明明确写成通用平台，不绑定公司产品。
5. KPI 字段区分电池端、DC-link、电机相和热等效电流。
6. 文档不再把历史产品参数当作规范建模依据。
