脉冲加热Simulink系统级模型
参数需求清单 v2.0

编制: 动力总成前瞻部 | 日期: 2026-05-08 | 版本: v2.0（Simulink系统级模型） | 用途: 与电池/电驱/整车供应商技术交流

版本说明：本清单在v1.0（0D模型参数）基础上，针对Simulink系统级建模需求进行全面升级。
主要变化：
• 电池：从纯电阻R(T) → 2阶RC等效电路（新增R1/C1/R2/C2极化参数、OCV全曲线、熵热系数）
• 逆变器：从理想开关 → IGBT器件级（新增导通压降、开关损耗、热阻抗）
• 热模型：从单节点集总 → 多节点热网络（新增电芯内部热阻、模组间热阻）
• 新增整车/系统级参数（冷却系统、线束阻抗、BMS通信接口）

标记说明：NEW = v2.0新增参数   升级 = 相比v1.0精度要求提升

最关键参数：M01（L_d）+ M19（控制开放性）
• L_d 直接决定脉冲电流幅值和加热功率。务必提供堵转条件下的实测值（非额定工况标称值）。
• M19 是方案一（软件策略）的可行性前提。若逆变器不开放底层PWM控制，则必须转向方案B/C。

L_d对加热效果的影响（0D模型计算结果）：
• L_d = 0.5 mH, f=500Hz → 温升速率 ~1.2°C/min ✓
• L_d = 1.5 mH, f=100Hz → 温升速率 ~0.8°C/min（边界可行）
• L_d = 3.0 mH → 方案一基本不可行
