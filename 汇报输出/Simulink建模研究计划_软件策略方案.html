三、各子系统建模方案

3.1 电池组子系统（Simscape Electrical）

模型选择：2阶RC等效电路（Thevenin ECM）

         R0          R1          R2
  OCV ──┤├──────┤├──┬──────┤├──┬──── V_terminal
                    │          │
                   C1         C2
                    │          │
                   GND        GND
参数物理含义来源温度/SOC依赖
OCV(SOC, T)开路电压供应商OCV-SOC曲线2D Lookup Table
R0(T)欧姆内阻（含SEI、集流体、接触电阻）HPPC测试Arrhenius拟合
R1(T), C1(T)电荷转移极化（τ1 = 1~5 s）EIS / HPPC拟合Lookup Table
R2(T), C2(T)扩散极化（τ2 = 30~200 s）EIS / HPPC拟合Lookup Table
关键方程：
V_terminal = OCV(SOC, T) - I·R0 - V_RC1 - V_RC2
dV_RC1/dt = I/C1 - V_RC1/(R1·C1)
dV_RC2/dt = I/C2 - V_RC2/(R2·C2)
dSOC/dt = -I / Q_nom

产热方程（用于热耦合）：
Q_gen = I²·R0 + V_RC1·I + V_RC2·I + I·T·(dOCV/dT) [最后一项为可逆熵热]

Simulink实现方式

  • 方案A（推荐）：Simscape Battery模块（R2022b+），直接配置ECM参数表
  • 方案B：手动搭建Simscape Electrical元件（Controlled Voltage Source + R + RC网络），灵活度更高
  • 串并联配置：先建单体模型 → 用 Simscape Component 封装 → 按实际Pack拓扑（Ns串Np并）组合

3.2 逆变器子系统（Simscape Electrical / Power Systems）

模型层级选择

层级模型精度仿真速度适用阶段
L0 - 理想开关Ideal Switch + 互补PWM低最快控制策略开发初期
L1 - 平均值Average-Value Inverter中快系统级联调
L2 - 开关级IGBT + Diode（含Vce_sat、Err、Eoff）高慢损耗精确计算、EMC分析
建议策略：Phase 1 用L0理想开关快速验证控制逻辑 → Phase 2 切换L2开关级模型精确计算IGBT损耗和热分布。两套模型通过Variant Subsystem切换，共享同一控制器接口。

脉冲加热专用PWM逻辑

  • 锁定d轴电流注入：三相桥臂按特定矢量（V1/V2交替或V0/V7零矢量）切换
  • 双极性PWM：上下桥臂互补导通，死区时间 t_dead = 2~4 μs
  • PWM频率可调：1~20 kHz（脉冲加热典型 1~5 kHz）
  • 占空比D：控制有效电压 V_eff = (2D-1)·V_dc

3.3 电机绕组子系统

堵转工况简化模型

脉冲加热时电机不转（机械锁止或零转矩指令），等效为纯RL负载：

  V_phase ──── R_s(T) ──── L_d(I) ──── 中性点
参数说明建模方式
R_s(T)绕组铜电阻，随温度线性变化R_s = R_s0·[1 + α_Cu·(T - T_ref)]
α_Cu = 0.00393 /°C
L_d(I)d轴电感，含磁饱和效应1D Lookup Table: L_d = f(I_d)
供应商提供或FEA仿真
L_qq轴电感（脉冲加热不涉及）仅d轴注入时可忽略
关键说明：脉冲加热仅注入d轴电流（Id≠0, Iq=0），电机不产生转矩。等效电路退化为单相RL串联回路。这是与正常驱动工况的本质区别——无需建立完整的dq坐标系电机模型。

热阻/热容物理含义典型值范围
C_core电芯内部热容~900 J/(kg·K) × m_cell
R_cc电芯内部→表面导热热阻0.5~2 K/W（取决于极片层数）
R_conv电芯表面→冷却液对流热阻脉冲加热时冷却泵关闭 → ∞
R_ambPack壳体→环境散热热阻0.01~0.05 K/W（自然对流+辐射）
C_pack_total整Pack热容（~400kWh系统）~3.8 MJ/K
简化策略：脉冲加热过程中冷却泵关闭（否则会带走热量），因此冷却回路热阻可设为无穷大。主要散热路径为Pack壳体→环境自然对流。对于400kWh商用车Pack，环境散热功率约200~500W（远小于加热功率63kW），可在初期模型中忽略。

3.5 控制策略子系统（Stateflow + Simulink Function）

状态机设计

  ┌──────────────────────────────────────────────────────────────┐
  │                    Stateflow 状态机                           │
  │                                                              │
  │  [IDLE] ──触发条件──→ [INIT] ──参数就绪──→ [HEATING]         │
  │    ▲                                        │    │           │
  │    │                                   正常  │    │ 异常      │
  │    │                                        ▼    ▼           │
  │    └──────────────── [DONE] ←──────── [FAULT]               │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘
状态进入条件执行动作退出条件
IDLE上电默认监测T_bat、SOCT_bat < T_threshold AND SOC > SOC_min
INIT加热请求读取参数表、选择f/D组合、预检IGBT状态参数校验通过
HEATING初始化完成输出PWM使能 + 频率/占空比指令
PID闭环调节电流幅值		T_bat ≥ T_target OR SOC < SOC_min
FAULT任何异常立即关断PWM、记录故障码故障清除 + 人工确认
DONE正常结束关闭PWM、记录加热时长/电耗返回IDLE

PID电流闭环

  • 控制目标：维持I_rms在设定值（不超过IGBT额定电流的80%）
  • 调节手段：调整占空比D（0.3~0.7范围）
  • 采样：每个PWM周期采样一次电流，滑动平均滤波
  • 抗饱和：积分限幅 + Back-calculation anti-windup

安全保护逻辑（独立于状态机，硬件级优先）

保护项阈值动作恢复条件
过流I_peak > I_IGBT_max立即关断（<10μs）不可自恢复
过压V_dc > V_bus_max × 1.1关断 + 泄放电阻投入V_dc回落
电池过温T_bat > 60°C停止加热T_bat < 55°C
电机过温T_motor > 120°C降功率/停止T_motor < 100°C
SOC下限SOC < 15%停止加热不可自恢复

七、风险识别与应对

#风险概率影响应对措施
1电池ECM参数不足（供应商未提供R1C1/R2C2）高中退化为1阶RC或纯R模型；用HPPC数据自行辨识；Phase 1可先用文献典型值

八、总结与下一步行动

整体时间线：4个Phase共计 9~13周（约2.5~3个月），与路线A实车验证并行推进。

立即启动事项：
1. 确认MATLAB/Simulink Toolbox许可证（Simscape Electrical + Stateflow为必需）
2. 向电池供应商索取HPPC数据（用于ECM参数辨识）
3. 向电驱供应商确认L_d值和PWM接口协议
4. 启动Phase 1——搭建基础电气回路，与0D脚本交叉验证

核心原则：模型从简到繁，每一步都与已有基准（0D解析解）对标验证。不追求一步到位，而是逐步增加复杂度，确保每一层新增的物理效应都是可控、可验证的。
