# 脉冲加热技术协同研发会议
## 建模信息提取与三支路电池-双电机控制器架构建议

会议时间：2026-05-26 10:10:31  
适用目的：为后续脉冲加热架构设计与仿真建模提供结构化输入  
整理口径：仅提取对建模、参数准备、方案筛选有直接价值的信息，不复述一般性讨论

## 1. 当前对建模最有价值的已知信息

### 1.1 项目阶段定位

- 当前优先级不是立刻定最终控制策略，而是先输出**几个可供选择的脉冲振荡电路架构**。
- 当前优先级高于安全细节深挖。会议已明确：先把构型做出来，再对析锂、退磁、寿命、保护细节做深入分析。
- 所有方案应先做仿真打标，再提交整车端评审。

### 1.2 目标对象的关键平台特征

- 会议共识口径：目标方向是**三支路并联电池 + 双电机控制器**架构。
- 已经拿来做初步验证的参考车型是：`3806`，`6x4`，`G6G`，`四电池包两串两并`，约 `618V / 400kW`。
- 这里存在一个必须显式区分的点：
  - `3806` 当前已建零维模型对应的是 `4包 2串2并` 的实际平台口径；
  - 会议里提出的“`三支路并联电池 + 双电机控制器`”是下一步要重点研究的**候选商用车脉冲加热架构方向**；
  - 两者不能混成同一个既成事实，后续建模时要把“参考平台”和“目标新架构”分开。

### 1.3 对现有乘用车方案的可复用认识

- 长安深蓝软件方案：
  - 特点：无需硬件改动，靠电机正反抖动实现。
  - 建模启示：可作为**纯软件脉冲加热下限方案**。
  - 风险：加热能力有限，噪声明显，更适合驻车。
- 比亚迪海狮07硬件方案：
  - 特点：双电池 + 单电机 + 中性点引出 + 附加回路器件。
  - 建模启示：可作为**硬件增强型脉冲振荡拓扑原型**。
  - 风险：成本上升，且乘用车拓扑未必直接适配商用车多支路系统。

### 1.4 会议里已经形成的关键技术判断

- 商用车不能简单照搬乘用车整体充放思路。
- 商用车的差异化机会在于：
  - 多支路电池；
  - 多电机/双电机控制器；
  - 中性点引出和支路间能量转移能力。
- 仅用单电机做传统脉冲可能浪费系统震荡源能力。
- 双电机同时脉冲的原始想法已被认为需要重新审视，原因是可能存在反向扭矩抵消或控制耦合问题。

## 2. 现有仿真和可直接借用的结果

### 2.1 我方已有3806初步仿真结果

- 工况对象：基于 `3806` 车型的双电机同时脉冲方案。
- 得到结果：
  - 温升速率约 `1.93 ℃/min`
  - 从低温升到 `0 ℃` 约 `18 min`
  - 系统效率约 `80.9%`
- 使用提醒：
  - 这组结果不能直接作为推荐方案结论；
  - 因为会议已认为“双电机同时脉冲”本身的可行性需要重新评估。

### 2.2 杨贺捷团队GT/JT模型给出的趋势信息

- 模型基础：基于比亚迪拓扑，`双电池 + 单电机 + 中性点引出 + 额外电感`。
- 已得到的趋势规律：
  - 频率越低，加热越快，但析锂/沉积风险越高；
  - 电流幅值越大，加热越快，但析锂风险越高；
  - SOC越高，加热越快；
  - 环境温度越低，加热越快，但高风险区更容易出现。
- 这部分价值：
  - 可作为后续商用车方案仿真的**物理趋势参考**；
  - 但不能直接作为商用车平台的定量结论，因为拓扑、电芯体系、支路数都不同。

## 3. 对建模最重要的输入与待确认项

### 3.1 可以先固化为建模假设的内容

- 建模目标优先顺序：
  - 先比较不同振荡回路架构是否能形成有效电池内耗加热；
  - 再比较温升效率、支路电流、器件压力；
  - 最后再压入析锂与寿命边界。
- 输出指标优先顺序：
  - `dT/dt`
  - 支路峰值电流 / RMS电流
  - 各电池支路净能量变化
  - 母线电压波动
  - 电机侧铜耗 / 逆变器损耗
  - 初步效率

### 3.2 当前最缺、必须尽快补齐的真实参数

- 电池侧：
  - 电芯类型、额定容量、串并联方式
  - OCV-SOC
  - 低温阻抗或HPPC参数
  - Pack允许脉冲加热能力曲线
  - 低温回充或析锂约束
- 电驱侧：
  - 电机 `Ld/Lq/Rs`
  - 中性点引出是否标准具备
  - 双电机控制器拓扑结构
  - 开关频率上限
  - 器件额定电流与热限制
- 系统侧：
  - 三支路电池与高压盒连接关系
  - 是否允许通过继电器形成支路切换
  - 是否需要IGBT实现快速切换
  - 允许新增的器件类型和成本空间

### 3.3 当前必须避免混淆的点

- “已有3806双电机模型”不等于“已完成三支路商用车脉冲架构论证”。
- “比亚迪/乘用车双电池单电机拓扑”不等于“商用车最优方案”。
- “频率越低、幅值越大加热越快”只是规律，不是设计结论，因为器件、电池和控制边界会更早卡住。

## 4. 当前最急迫任务：给出可选择的脉冲振荡电路架构

以下方案按“先能画、先能仿、先能比较”的原则整理，不代表最终推荐顺序。

### 方案A：单电机对整包统一脉冲

- 含义：沿用较接近传统单源脉冲加热思路，利用一台电机绕组或等效电感，对整车电池系统统一形成振荡回路。
- 优点：
  - 架构最简单；
  - 最快能与当前零维模型对接；
  - 可作为基线方案。
- 缺点：
  - 无法体现多支路电池优势；
  - 可能浪费双电机控制器潜力；
  - 对大系统等效内阻较敏感。
- 建模优先级：高
- 作用：作为所有新方案的对照基线。

### 方案B：单电机对三支路电池轮询脉冲

- 含义：单电机作为唯一振荡源，通过支路切换，分别对 A、B、C 三个电池支路施加脉冲加热。
- 优点：
  - 结构相对清晰；
  - 比整包统一脉冲更能体现支路化控制；
  - 易于评估不同支路的不一致性影响。
- 缺点：
  - 需要高压盒/继电器配合切换；
  - 切换逻辑和支路均衡会更复杂；
  - 加热过程可能不是连续全系统同时进行。
- 建模优先级：高
- 作用：验证“多支路选择性加热”是否比整包统一加热更优。

### 方案C：两支路成对振荡，第三支路旁路待机

- 含义：任意两支路之间形成局部能量振荡回路，例如 `A↔B`，然后换为 `B↔C`，再换为 `C↔A`。
- 优点：
  - 直接利用多支路电池的结构优势；
  - 可以研究支路间能量转移而不是整包整体充放；
  - 便于构建“环形震荡”的前置简化模型。
- 缺点：
  - 回路定义和切换条件比方案B更复杂；
  - 高压切换逻辑和安全隔离要求更高；
  - 需要明确第三支路在待机时的电气状态。
- 建模优先级：很高
- 作用：这是最值得验证的商用车差异化方向之一。

### 方案D：三支路环形震荡 A→B→C→A

- 含义：按丁浩提出的设想，构建三支路之间的循环充放电路径，使能量在三支路中轮转。
- 优点：
  - 最能体现商用车多支路创新价值；
  - 有机会降低单支路瞬时负荷；
  - 可能改善持续加热能力。
- 缺点：
  - 拓扑和控制最复杂；
  - 需要先定义清楚每一时段的开关状态与回路方向；
  - 继电器方案可能不够快，IGBT方案成本可能偏高。
- 建模优先级：中高
- 作用：作为创新型方案保留，但不建议一开始就上最高保真模型。

### 方案E：双电机分工，一台主振荡，一台辅助均流/补能

- 含义：不是双电机完全同相同时脉冲，而是两台电机承担不同角色，例如一台形成主振荡回路，另一台负责辅助支路切换、均流或延长振荡支撑。
- 优点：
  - 比“双电机完全同步脉冲”更有工程可解释性；
  - 有机会利用两台电机资源，而不是简单叠加。
- 缺点：
  - 控制耦合复杂；
  - 实现方式高度依赖双电机控制器内部拓扑；
  - 需要更明确的控制器开放能力。
- 建模优先级：中
- 作用：作为双电机资源利用方案进行探索，但不作为首个落地对象。

## 5. 建议的建模顺序

### 第一层：先画出统一的拓扑抽象图

- 统一画法必须明确：
  - 三个电池支路如何命名：`Bat_A`、`Bat_B`、`Bat_C`
  - 双电机控制器如何命名：`Inv_M1`、`Inv_M2`
  - 中性点、母线、电感、继电器/IGBT的电气位置
  - 每个方案中哪些开关闭合、哪些断开

### 第二层：先做低保真电路可行性筛选

- 先用等效电池 + 等效电感/绕组 + 开关逻辑，判断：
  - 是否能形成有效振荡；
  - 主电流流经哪些元件；
  - 哪个器件先过流或过压；
  - 哪个方案最值得进入高保真阶段。

### 第三层：对入围方案再上更高保真模型

- 对筛选后前 `2~3` 个方案，再增加：
  - 电池RC动态
  - 低温阻抗
  - 电机 `Ld/Lq/Rs`
  - 逆变器损耗
  - Pack热模型

### 6.1 必须形成的图纸/文档

- `5` 个候选架构原理图：
  - 方案A：单电机对整包统一脉冲

  - 方案D：三支路环形震荡
  - 方案E：双电机分工振荡
- 一张方案对比表，字段至少包括：
  - 新增硬件
  - 控制复杂度
  - 对高压盒要求
  - 对双电机控制器要求
  - 预期加热潜力
  - 预期风险
  - 建模优先级

### 6.2 建模上建议先做的3个对象

1. 方案A：做基线  
2. 方案C：做商用车差异化核心验证  
3. 方案B：做工程可实施性折中方案  

方案D 和 方案E 先保留为第二批，等前三个方案的电流路径和器件压力看清后再深化。

## 7. 结论

- 这次会议对建模最重要的价值，不在于给出了最终技术路线，而在于明确了当前阶段的目标已经从“证明脉冲加热原理”转为“输出几种适合商用车多支路系统的振荡架构备选”。
- 当前最急迫的工作不是继续围绕现有双电机同相零维模型做小修小补，而是尽快把三支路电池-双电机控制器的几种可行拓扑画出来，并建立统一比较口径。
- 建模上应采用“两步走”：
  - 先用简化电路快速筛架构；
  - 再对入围方案做高保真电池、电机、热、控制联合仿真。
