# 商用车动力电池低温脉冲加热仿真研究

本仓库用于整理商用车动力电池低温脉冲加热技术的资料、参数、模型、仿真结果和阶段输出。

当前项目定位：从商用车现有高压架构、电池包连接关系、电驱硬件能力和整车控制约束出发，建立分层仿真评估框架，判断车载低温脉冲加热技术是否存在工程可行窗口。

## 当前核心判断

低温脉冲加热的第一性原理是利用电驱系统形成脉冲电流，通过电池内阻或阻抗实部产生热量：

```text
Q_bat ~= I_rms^2 * R_bat(T, SOC, f)
```

但商用车项目的核心矛盾不是热源公式本身，而是快速升温所需的大RMS电流，与电池安全、电驱可靠性、整车高压架构、控制权限和经济价值之间的冲突。

4500-27当前参数表明确约束：

- 电池包数量：3；
- 非独立输出；
- 并联输出；
- 不可独立接入电驱；
- 高压平台约811.44 V；
- 双电机电驱桥。

因此当前实车方案边界应按“三包并联整体输出”处理，不应把单包独立接入作为工程可行方案。

## 建模分层

本项目按以下层级推进：

1. L0 零维电热筛选：判断温升量级、能耗、电流和方案排序。
2. L1 电流可实现性：验证逆变器、电流环、PWM和母线电压是否能实现目标脉冲电流。
3. L2 电池安全边界：评估低温回充、析锂、BMS限流和端电压风险。
4. L3 电机与逆变器风险：评估绕组温升、退磁、逆变器结温、NVH和寿命。
5. L4 整车控制与应用价值：评估使能、退出、降级、场景和经济价值。

## 结论表达约束

项目结论应区分：

- 已验证事实；
- 基于证据的推断；
- 待确认假设。

零维模型只能用于原理筛选和方向判断，不能直接证明整车安全可行、控制可实现或硬件寿命满足。
