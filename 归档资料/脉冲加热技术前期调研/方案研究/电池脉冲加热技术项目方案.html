一、项目背景与技术原理

1.1 立项背景

新能源商用车在低温环境下动力电池性能严重受限，导致用户抱怨集中：低温充电慢/充不进电、输出功率受限车辆动力不足、续航里程大幅缩水。传统PTC加热速率低（-10°C下仅约0.65°C/min，-20°C下约0.78°C/min），能量效率仅35%~40%，且热量从外部传递至电芯内部路径长、热损失大。

脉冲电流加热技术利用电池低温下自身内阻变大的特性，通过控制电机开关管的导通方式，使电机与电池之间产生高频充放电，实现从电芯内部快速均匀自加热，是一种新型高效的动力电池加热方式。该技术较传统PTC加热在温升速率（1.3~1.5倍提升）和能量效率（63%以上）方面均有显著优势。

1.2 脉冲电流加热技术原理

核心物理机制：

低温环境下，电池内阻急剧增大。在交流激励下，锂离子只在电极材料颗粒局部做短程振荡迁移，不深度嵌入负极晶格，因此不易发生析锂。此时可以利用远大于常规充电倍率的电流（如3C~8C），通过以下三种产热机制实现快速加热：

  1. 欧姆热：电流流过电池欧姆内阻（集流体、电极、电解液）产生的焦耳热
  2. 极化热：电化学反应过电位产生的热
  3. 可逆熵热：锂离子嵌入/脱出过程中的熵变热

整车实现方式：

利用电机控制器IGBT开关管有序通断，以电机绕组电感作为储能元件，在电池与电机之间产生高频脉冲充放电。车辆挂P档使电机处于"堵转"状态（无转矩输出），通过调节开关管占空比控制脉冲电流有效值。

简化工作原理：

状态1（电池放电）：Q1和Q4闭合，Q2和Q3断开 → 电流从电池正极经电机绕组流向负极，电池放电

状态2（电池充电）：Q2和Q3闭合，Q1和Q4断开 → 电感中储存的能量回馈至电池，电池充电

通过高频交替切换上述两种状态，实现电池的高频充放电脉冲加热。

3.2 仿真任务一：电芯电化学-热耦合建模（COMSOL）——核心任务

3.2.1 模型架构

采用伪二维（P2D）电化学模型 + 三维热模型耦合方案：

P2D电化学模型（1D+1D）：

  • x方向：负极集流体(Cu) → 负极多孔电极 → 隔膜 → 正极多孔电极 → 正极集流体(Al)
  • r方向（粒子径向）：活性材料颗粒内锂离子扩散（Fick定律）
  • 电解液：浓溶液理论，锂离子迁移与扩散
  • 电极动力学：Butler-Volmer方程描述反应速率
  • 输出：局部电流密度、电势分布、锂浓度分布 → 产热分布 q(x,t)

3D热模型：

  • 几何：实际电芯尺寸（方形/软包/圆柱）
  • 输入：P2D模型计算的产热分布 q(x,t) 映射至3D几何
  • 输出：温度场 T(x,y,z,t)、最大温差 ΔT_max、温升速率

3.2.2 关键产热方程

总产热： q_total = q_ohm + q_pol + q_rev

欧姆热： q_ohm = σ_eff·|∇φ_s|² + κ_eff·|∇φ_e|²（固相+液相欧姆损耗）

极化热： q_pol = a_s · i_loc · η（Butler-Volmer过电位产热）

可逆熵热： q_rev = a_s · i_loc · T · (∂U/∂T)（熵变产热/吸热）

3.2.3 析锂判据（安全边界核心）

判据：负极固相电势 φ_s_neg < 0 V（vs Li/Li⁺）→ 有析锂风险

高频脉冲下析锂抑制机制：

  1. 脉冲周期远小于锂离子扩散时间常数 τ_diff = L²/D_s
  2. 负半周期（充电方向回馈）使锂离子脱出，避免表面浓度累积
  3. 频率越高，锂浓度振荡幅度越小，析锂风险越低

3.3 仿真任务二：模组/电池包集总参数建模（MATLAB/Simulink）

3.3.1 基于COMSOL结果的等效电路模型（ECM）建立

从COMSOL高保真模型降阶至可用于系统控制的等效电路模型：

2阶RC等效电路结构：

V_cell = V_oc(SOC,T) - I·R_ohm - V₁ - V₂

dVᵢ/dt = -Vᵢ/(Rᵢ·Cᵢ) + I/Cᵢ   (i = 1, 2)

mC_p·dT/dt = I²R_ohm + Σ(Vᵢ²/Rᵢ) - hA(T - T_amb)

参数识别流程：

  1. 从COMSOL仿真得到不同温度/频率/电流下的电压响应
  2. 使用System Identification Toolbox拟合ECM参数
  3. 建立参数查找表：R_ohm(T,SOC), R₁(T,SOC,I,f), C₁(T,SOC,I,f)…

3.3.2 Simulink系统模型架构

整车脉冲加热系统Simulink模型包含以下核心模块：

  1. 电池包模型：ECM × N_cells + 热网络（Simscape Battery或自定义）
  2. 电机堵转模型：PMSM在ω=0时的R-L等效（L_d, L_q, R_s）
  3. 逆变器+IGBT模型：三相桥式电路（Simscape Electrical）
  4. SVPWM/SPWM调制器：产生IGBT门极驱动信号
  5. 加热控制策略（Stateflow）：
    • State 1：初始化检测 → 判断T_batt < T_threshold
    • State 2：参数选择 → 查表确定最优 f, I_amp, D
    • State 3：脉冲加热执行 → PID闭环调节占空比维持目标电流
    • State 4：终止判断 → T_batt ≥ T_target 或异常保护触发
  6. 热管理回路模型：冷却液/余热回收回路（电机冷却液与电池液冷回路串联）

3.4 仿真任务三：整车系统级联合仿真

方案A（推荐—模型降阶）：

  1. COMSOL中运行参数化扫描，生成数据集
  2. 在MATLAB中训练降阶模型（ROM）：使用DMD（动态模态分解）或神经网络
  3. 建立从 (I, f, T_init, SOC) → (T(t), φ_s_min) 的映射
  4. 将ROM嵌入Simulink作为S-Function或MATLAB Function模块

方案B（高精度但慢）：

使用COMSOL LiveLink for MATLAB直接联合仿真，在Simulink每个时间步调用COMSOL求解器。

循环规程脉冲加热至10°C → 静置 → 放回-20°C → 再加热

每50次中间检查标准容量测试（25°C, 1C放电）；EIS测试；拆解检查（SEM观察负极表面是否析锂）

实验四：模组/电池包级加热验证

  • 模组级（≥4串）：每电芯中心+表面布置热电偶，验证电芯间温差（要求<5°C）
  • 电池包级：验证液冷/加热回路集成；评估电机余热回收效果；全包温度场测量

六、仿真阶段交付物与预期结论

6.1 阶段一交付物（策略与算法开发 · M1-M6）

交付物形式核心内容
电芯电化学-热耦合模型 COMSOL .mph文件 + 模型说明书 经验证的P2D + 3D热模型；含全参数定义和边界条件
脉冲参数敏感性分析报告 PPT / Word f - I_amp - T_init 对温升速率、析锂风险、温度均匀性的影响规律
最优脉冲参数MAP 二维等高线图 + 查找表 不同温度和SOC下的推荐 f 和 I_amp 组合
析锂安全边界报告 图表集 + 分析文档 各温度/频率/电流组合下的负极电位分布；安全加热窗口

预期结论示例：

"在-20°C条件下，频率≥500Hz、电流≤5C的方波脉冲可保证负极电位始终>0V（不析锂），同时实现≥1.2°C/min的温升速率。欧姆热占总产热65-75%，是主导加热机制。频率过低（<200Hz）时负极电位在充电半周期接近0V，存在析锂风险。"

6.2 阶段二交付物（拓扑设计与策略实现 · M7-M12）

交付物形式核心内容
整车脉冲加热系统Simulink模型.slx文件 + 模型说明书电池包 + 电机 + 逆变器 + 控制器 + 热回路
自适应加热控制策略Stateflow + MATLAB代码含状态机逻辑、PID参数整定、故障保护策略
整车加热性能预测报告PPT / Word不同温度下的加热时间、能量消耗、SOC消耗预估
热管理架构改进方案原理图 + 分析报告电机余热回收回路设计（电机冷却液与电池液冷回路串联）

交付物形式核心内容
