一、模型意义与应用场景

核心目标：在不增加硬件的前提下（复用电机逆变器），评估利用双极性PWM方波对电池组进行内部脉冲加热的可行性，为低温冷启动策略提供快速定量决策依据。

解决的工程问题

问题模型回答
给定电机电感 L 和开关频率 f，能否在30 min内将电池从 −20°C 加热到 0°C？动态温升仿真（第5部分）
加热过程中是否存在析锂风险？析锂安全约束模块
SOC消耗多少？系统效率如何？库仑计数 + 效率分解
哪些 (L, f) 组合落在可行域内？可行性边界图（第6部分）
Pack内温度不均匀性是否可接受？简化1D热阻估算（第5.5部分）

适用范围与假设

  1. 电池组集总参数（0D），单一温度节点，忽略内部温差
  2. 电机堵转（ω=0），无反电动势，绕组等效为 RL 串联
  3. 逆变器理想化：无死区、无开关损耗（仅在动态仿真中计入）
  4. 单相等效（d轴或两相导通等效）
  5. 准稳态假设：每个热步长内电流达到周期稳态
  6. 1-RC Thevenin 等效电路：含极化阻抗 R1//C1，基波频率近似
  7. OCV 恒定（LFP 平台区内误差 <3%）

二、建模机理

2.1 电路拓扑

电池组(V_oc, R_bat) → 逆变器(IGBT全桥, 双极性PWM) → 电机绕组(L_d, R_s, 堵转)

2.2 RL 稳态电流解析解

双极性 PWM 方波激励下 RL 串联电路的周期稳态解：

状态1 (0 ≤ t < DT): i₁(t) = V_s + (I_min − V_s)·exp(−t/τ)
状态2 (DT ≤ t < T): i₂(t') = −V_s + (I_max + V_s)·exp(−t'/τ)

其中: V_s = V_dc/R_total, τ = L/R_total, α = T_period/τ, β = D·α

联立周期性边界条件:
I_max = V_s·(e^α − 2·e^(α−β) + 1) / (e^α − 1)
I_min = V_s·(2·e^β − e^α − 1) / (e^α − 1)

D=0.5 特殊情况: I_max = V_s·tanh(α/4), I_min = −I_max

2.3 1-RC Thevenin 等效电路（频率相关阻抗）

R_bat_eff(f, T) = R_ohm(T) + R1(T) / [1 + (2πf·R1(T)·C1)²]

物理含义:
f >> f_p: C1 短路 R1 → R_bat_eff ≈ R_ohm（高频加热更高效）
f << f_p: C1 开路 → R_bat_eff ≈ R_ohm + R1 = DCIR
特征频率: f_p(T) = 1/(2π·R1(T)·C1)

2.4 热模型（集总参数法）

C_th · dT/dt = P_bat − (T − T_amb) / R_th

P_bat = I_rms² · R_bat_eff(f, T) [电池内部产热]
P_motor = I_rms² · R_s [电机铜耗，不加热电池]

2.5 内阻温度依赖（Arrhenius 模型）

R_cell(T) = R_cell_25 · exp[Ea·(1/T − 1/T_ref)]
R_pack(T) = (N_s/N_p) · R_cell(T) · (1 + k_parasitic)

2.6 析锂安全判据

安全条件: 负极表面过电位 η_neg + U_e > 0

方波基波分析:
V_neg = (4/π) × I_cell_peak × |Z_neg(f, T)|
析锂限制: I_pack_peak < N_p × (π/4) × U_e / |Z_neg(f, T)| × k_safety

负极阻抗模型:
Z_neg(f, T) = R_SEI(T) + R_ct(T) / √[1 + (f/f_ct)²]
f_ct(T) = 1 / (2π·R_ct(T)·C_dl)

高频时 f >> f_ct，双电层电容短路 R_ct，法拉第电流极小，析锂风险大幅降低。

六、文件结构与使用方法

文件类型功能
pulse_heating_0D_model_v2.m主脚本完整系统模型，直接运行即可生成全部结果
lithium_plating_criterion.m函数析锂安全电流上限计算，被主脚本调用
verify_plating_criterion.m验证脚本独立验证析锂判据与文献对标
使用步骤：
1. 将三个 .m 文件放在同一目录
2. 在 MATLAB 中打开 pulse_heating_0D_model_v2.m
3. 根据实际电芯/电机参数修改"第1部分: 参数定义"中的数值
4. 直接运行（F5），命令行输出数值结果，同时生成6张图
5. 如需单独验证析锂模块，运行 verify_plating_criterion.m
待校核项（收到实测数据后替换）：
• R_cell_25 / Ea_R — 需 HPPC 或 EIS 实测
• R_ohm_frac / R1_frac / C1_cell — 需多频率 EIS 辨识
• R_ct_ref / Ea_ct / C_dl — 需三电极 EIS（负极半电池）
• L_d / R_s — 需电机堵转实测确认
• M_bat / Cp_bat / R_th_pack — 需整包热测试
