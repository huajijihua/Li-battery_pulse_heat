# M5 文献调研与新平台建设任务规划 v01

日期：2026-07-22

## 0. 背景与动因

M4 阶段收尾（见 `M4_阶段收尾报告_v01.md`）识别到电池侧有效电流过低的架构性卡点：电机侧 Id=40A 而电池侧 I_batt RMS 仅约 5.5A。M4 的正式学术路线命名为 **CMI（Combined Module Invert，文献原称）架构下的开环低频 d 轴脉冲电流自加热路线**。经 PDF 文献调研确认，该卡点是当前 CMI 架构和低频加热控制方式共同形成的物理局限，在文献横向对比中属性能最低档（RTR 约 2.88 °C/min，电池侧最大约 1.41C）。

据此决定转换建模路线：在 M4 冻结基线之上，新建经文献验证的通用型车载（电池-控制器-电机）低温脉冲加热系统仿真平台 `车载脉冲加热仿真平台_v2`。M5 首版正式命名为 **dSVPWM（dead-zone Space Vector PWM，死区空间矢量脉宽调制）换流型脉冲自加热路线**，保持单包、三相三线、标准两电平逆变器和 PMSM 硬件边界。控制层采用 dSVPWM 主线，频域电热和析锂判据分别作为模型层和安全层，NPBH/四线制/DMSI 作为后续拓扑替代。`M4-M5_过渡收尾与M5准入审计_v01.md` 已关闭过渡态，M5-00 完成；M5-01 已准入但未启动。

**关键约束**：任务 2 的核心准入已完成：7 篇重点原件完成逐页视觉核验，10 份分析文件完成 A1/A2/B1/C1 分级，并已形成评估报告、过渡审计和 v2 接口契约。CN679、CN173、ATIC 的原件页码/图表核验仍需定向补强，但不阻断 M5-01；路线角色不构成具体设备参数、安全或产品级授权。

### 2.4 当前文本模型已完成的预调研（供多模态模型接手索引）

以下结论已由当前文本模型通过 PyMuPDF 文本提取获得，可作为多模态深读的索引与验证基线（图表/公式细节需多模态复核）：

- **2024 IEEE TTE**：DC-link 电容 Cp<1mF 电流无显著衰减，>100mF 交流消失，谐振约 1kHz；非零矢量/死区比值 a0≈1.6；开关频率 10kHz；零矢量占空比 d0 间接调节电池电流 + 双 PI 电压反馈；-30->-10°C 提速 45%（637s vs 1160s）；电池电流 ±8->14A。
- **2021 iScience**：CMI 模式（与 M4 的正式路线同源）最大 1.41C；DMSI 电池分组达 4C/8.6°C/min；效率 89.7%（电池发热占 97.7%）。
- **2026 NPBH**：中性点经开关接电池中点，零轴电流 i0 直接穿电池，转矩 inherent=0，加热/驱动解耦；RTR 4.76°C/min；二阶 ECM 频域阻抗；最优策略频率表（-30°C/37Hz ~ 0°C/2Hz，高 SOC 高频防过压）。
- **ATIC PPT**：储能模式（电池放电->电机线圈储能，SVPWM）+ 续流模式（切 6 管->续流二极管反向向电池充电）；升温约 5°C/min。
- **专利 CN114789679A**：母线电流有效值闭环 + 转子角度前馈 Id_ini 查表 + PI，Iq_tag=0。
- **Jian 2023 交流预热策略**：析锂判据（负极电位 >0V vs Li/Li⁺）；边界频率（低频 TVCS 析锂、高频 NEPCS 过压）；组合策略。
- **技术对比表（2026 NPBH Table 8）**：常规驱动电路脉冲电流 RTR 2.88（最低）< 车载 AC 加热器 3.39 < IBSH 3.45 < NPBH 4.76 < TMSI/DMSI 8.6。

### 2.5 分层取舍原则（宁缺毋滥）

**首版主线：**

- 硬件边界保持单包三相三线标准逆变器 + PMSM，不新增中点、N 线或开关。
- 控制层采用 2024templock + 2024 IEEE TTE 的 dSVPWM（死区空间矢量脉宽调制）控制族，电压快速回撤和温度监督分层实现。
- 电池模型层采用 Zhang 2015 的频域热生成/集总热平衡结构，参考 NPBH 的二阶 ECM 和双频产热分解。
- 安全层采用 Jian 2023 的负极电位与 TVCS/NEPCS 组合判据候选，数据资格满足前 `DeviceProtection_Evaluable` 保持 0。

**独立对照：**

- CN114789679A 的母线 RMS 闭环、转子角度前馈和 BMS 温度-电流表用于控制对照，不与 dSVPWM 同时驱动同一调制层。
- CN112977173A 的开环 Ud 脉冲用于开环反例和储能/续流信号流对照。

**远期拓扑评估：**

- NPBH 中性点开关、BYD 四线制 N 线和 iScience DMSI 电池分组均属于硬件/接口变化，首版不写入默认拓扑。

**辅助材料：**

- ATIC、Shell、PMSM 背景和未深读参数文献只提供方法提示、工程量级或背景，不承担首版定量参数、安全阈值或验收判据。

详细分级见 `M5_文献评估报告_v01.md` 和 `文献分析/M5_文献核验状态与技术维度统一表_v01.md`。

### 3.3 与 M4 的关系

| 维度 | 继承自 M4 | 改进（基于文献） |
|---|---|---|
| 账本框架 | 电气+热账本、固定窗口复算、残差判据 | 沿用口径，扩展电池频域热生成项 |
| 参数纪律 | 来源身份标签、Reference/Platform 边界 | 增加 `LITERATURE` 来源，接入文献参数 |
| 研究规范 | AGENTS 第 11 节全部 | 继续遵循 |
| 并行/交接 | parsim、长计算交接 | 继续遵循 |
| **控制方式** | CMI 架构下的开环低频 d 轴脉冲（50Hz） | **首版改为 dSVPWM（死区空间矢量脉宽调制）主调制 + 电压快速回撤 + 温度监督（2024templock/IEEE TTE）** |
| **控制对照** | 开环给定 Id | **CN679 母线 RMS 闭环作为独立对照，不与 dSVPWM 叠加** |
| **电池模型** | 固定内阻 | **Zhang 频域热生成结构 + NPBH 二阶 ECM 候选，先并列审计再替换** |
| **安全约束** | DeviceProtection_Evaluable=0 | **Jian 负极电位/TVCS-NEPCS 候选路径；资格满足前保持 0** |
| **拓扑** | M4 CMI 单包驱动边界 | **首版保持单包三相三线；远期评估 NPBH/BYD/DMSI 硬件拓扑** |

### 3.4 建设步骤（依据 M5 文献评估报告更新）

1. **基线迁移**：从 M4 冻结模型复制结构与账本框架到 v2，建立 v2 唯一活动模型。
2. **控制层重构**：实现 dSVPWM 非零矢量+零矢量+死区矢量+补偿矢量，按文献映射选择 `bn`、`d0` 或 `a0` 的主执行量，电压快速回撤和温度慢速监督；`bn=0` 退化检查优先。
3. **电热模型升级**：接入二阶 ECM 频域阻抗和集总热平衡，区分电池支路/驱动/加热电流，完成 PWM 波形与频域公式的等效校核。
4. **安全判据建立**：接入负极电位/边界频率/端电压组合候选，建立 `UNKNOWN/CANDIDATE/EVALUABLE` 状态，资格满足前保持 `DeviceProtection_Evaluable=0`。
5. **分层验证**：先验证门极、换流、电池 RMS 和账本，再验证热响应和安全候选；不以文献温升速率直接作为 v2 达标值。
6. **独立控制对照**：使用 CN679 RMS 闭环和必要的 CN173 开环 Ud 作为独立 `SimulationInput` 工况，不与首版主控制器叠加。
7. **远期拓扑评估**：在首版接口稳定后评估 DMSI、NPBH 中性点开关和 BYD 四线制。

### 3.5 验证标准
- 账本复算残差封闭（沿用 M4 口径，≤1e-12 量级）。
- dSVPWM 实际门极和电池支路 RMS 可独立读回，且相对 M4 的变化可归因于换流控制，不把文献 C 倍率作为硬性达标值。
- 加热速度、能量效率、安全候选三类指标均建立可审计路径，频域近似与波形假设显式记录。
- 析锂安全判据可计算；只有满足参数、化学体系、波形和验证前提时，`DeviceProtection_Evaluable` 才允许为 1。
- 所有结论区分 Reference/Platform 与 LITERATURE 边界，不越界宣称设备/产品级。
