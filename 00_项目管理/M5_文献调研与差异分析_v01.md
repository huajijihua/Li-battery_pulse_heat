## 1. 核心发现：M4 卡点与文献定位

### 1.1 M4 卡点回顾

M4 基线（CMI 架构下的开环低频 d 轴脉冲电流自加热路线）的电池侧有效电流过低（Id=40A 电机侧 → I_batt RMS ≈ 5.5A）的机理已有充分证据：

- **主因**：零速锁止下电机近似纯电感负载，Id=40A 主要是无功电流，在电机电感与 DC-link 电容间交换，不流回电池。DC-link 电容滤波，电池只承担有功功率（约 92 W 损耗）+ 少量穿透纹波。
- **电容非主因**：2024 IEEE TTE 明确验证 Cp<1 mF 时电池电流波形几乎重叠，无显著衰减。当前 470 μF=0.47 mF 在安全范围。
- **文献定位**：M4 正式路线的 CMI 架构 = 2021 iScience 的 CMI 模式 = 2026 NPBH Table 8 中的"常规驱动电路脉冲电流[Du 2022]"，在文献对比中 RTR 最低（2.88°C/min），电池侧最大等效电流仅 1.41C。

### 1.2 文献中各技术路线能力对比

| 技术路线 | 文献 | 硬件改动 | 电池侧最大电流 | 加热速率 | 支持途中加热 | 转矩消除方式 |
|---|---|---|---|---|---|---|
| **CMI（M4 正式路线同源）** | Du 2022 / iScience 2021 | 无 | 1.41C (70.4A/50Ah) | 2.88°C/min | 是 | 设 Iq=0 |
| **dSVPWM 电压反馈** | IEEE TTE 2024 | **无** | 母线 AC 峰峰值 200A | 较固定占空比 +45% | 是（行驶中） | 本文未专门讨论（速度环稳定） |
| **dSVPWM 温度锁定** | WEVJ 2024（templock） | **无** | 母线 AC 峰峰值 200A | 锁定 -10°C（防止掉温） | 是（CLTC 行驶中） | 死区纹波 <±1Nm |
| **DMSI 电池分组** | iScience 2021（清华） | +1 继电器 + 电池包拆分 | **4C (200A)** | **8.6°C/min** | **否** | 设 Iq=0 |
| **NPBH 中性点开关** | Sustainability 2026（北理工） | +1 开关 + 电池包中点引出 | 3.6C 预热 / 途中 2-3C | **4.76°C/min** | 是 | **恒等消除（位置无关）** |

| **车载 AC 加热器** | Shang 等 | 每模块 ≥1 开关 + 电感 | — | 3.39°C/min | 否 | 无需 |

**关键观察**：
1. **纯控制层改进（零硬件改动）** 只有两条路线：dSVPWM（IEEE TTE 2024 / WEVJ 2024）和 CMI 架构下的开环低频 `Id` 脉冲（M4 现有）。dSVPWM 通过死区换流产生母线 AC，改变了 M4 原有开环低频参考的电流形成机制。
2. **DMSI 性能最高但不支持途中加热**，且需要修改电池包物理架构。
3. **NPBH/比亚迪四线制** 支持途中加热且性能优异，但需硬件改动（+1 开关/N 线）。

## 2. 关键文献技术方法深读总结

### 2.1 2024 IEEE TTE（Li Wei 等，浙大）— 电压反馈 dSVPWM

**核心方法**：
- 非零矢量（电池放电→电机电感储能）→ 零矢量（电机电流衰减，电池电流指数衰减）→ 死区（电机电感经体二极管反向向电池充电）的换流循环（称为"换流模式 C"）
- 零矢量占空比 d0 间接调节电池电流幅值（d0 越大，充放电电流幅值均减小）
- 非零矢量与死区时间比 a0 固定为 1.6（保持充放电能量平衡，最大化加热效率）
- **双 PI 电压反馈**：上 PI 控制峰值电压（≤48V），下 PI 控制谷值电压（≥32V），滞环切换
- 控制频率固定 10kHz（避免低频析锂风险 + 避免高频逆变器效率下降）

**关键参数**：
- PI 参数：Kp=0.01, Ki=0.9（控制周期 Tc=1×10⁻⁴s）
- PWM 周期：0.1ms（10kHz）
- 仿真步长：2×10⁻⁷s
- DC-link 电容安全范围：<1mF（避免 LC 谐振）

**实验验证**：
- 小尺度台架（12 串 1 并，50.4V/30V 限压，1kW 电机）
- 对比固定占空比策略，电压反馈策略加热速率提升 45%（-30→-10°C）
- 电池老化测试：无明显加速老化迹象

**与 M4 差异**：
- **相同点**：复用同一逆变器硬件，均为 EDS 集成路线
- **核心差异**：M4 正式路线用开环 `Id` 脉冲（低频 50Hz 直接给定 d 轴电流参考），IEEE TTE 用**非零矢量+零矢量+死区 SVPWM 换流**（10kHz PWM 周期内产生母线 AC，电池电流周期 = PWM 周期 0.1ms，而非加热脉冲周期）
- **控制目标**：M4 是 `Id` 幅值/频率/占空比开环给定，IEEE TTE 是**电池电压峰值/谷值闭环**（PI 调节 d0）

### 2.2 2021 iScience（Li Yalun 等，清华）— CMI/DMSI 双模式

**核心方法**：
- CMI 模式（与 M4 正式路线同源）：常规驱动电路，电池单包连接，电机线圈储能释放产生脉冲电流。最大加热电流 1.41C（70.4A/50Ah），最优周期 0.008s（125Hz），电角 90°
- **DMSI 模式**：电池包拆分为两个模块，分别连接逆变器不同桥臂，通过 U7 零矢量实现模块间能量转移
- DMSI 模式下加热电流可达 4C（200A），RTR 8.6°C/min（-7°C 环境），效率 89.7%（电池发热占 97.7%）
- 5000 次循环后容量衰减仅 0.5%，无析锂迹象

**关键参数**（Table 2）：
- 电池包：110S2P，50Ah/电芯，3.7V 标称
- PMSM：Ld=0.38mH, Lq=1.065mH, Rs=15.4mΩ, p=4, ψf=0.19Wb, Im=200A
- IGBT：RI=1.6mΩ，额定 650V

**与 M4 差异**：
- CMI 模式与 M4 架构同源，但该文献工作在更高电流等级（110S2P 大电池包，电机 200A 额定）
- M4 当前 5.5A 电池侧 RMS 远低于该文献 CMI 模式的 1.41C 上限，说明 M4 正式路线的控制方式（开环低频 `Id` 脉冲）未充分利用 CMI 架构的潜力，控制方式本身可优化

### 2.3 2025 比亚迪（Ling 等，eTransportation）— 四线制 FOC 解耦

**核心方法**：
- 电池包拆分为上下两个半包，中性点引出 N 线连接电机
- 扩展 Clarke-Park 变换（3×3 矩阵），增加 IN 维度
- **加热电流 IN 与驱动电流 Id/Iq 在 dq0 坐标系下完全解耦**（与 NPBH 原理类似）
- 双 PI 独立控制：转矩控制（Id/Iq）+ 加热控制（IN）
- 全场景应用：充电（-20°C 省 30.7% 充电时间）、停车启动（-38°C，0.73°C/min，耗 4.2% SOC）、行驶（-30°C）

**关键实验数据**：
- 50Hz 频率下加热速率最高（与内阻频域特性一致）
- IRMS=1.5C（225A）在 50Hz 下对电池寿命影响小（300 次循环后 SOH 仍 97%）
- 整车充电测试：-20°C，自加热 24.3min 升温至 15°C，充电电流升至 123A
- 与热泵对比：充电时间 71.7min vs 103.5min，省 30.7%

**与 M4 差异**：
- **硬件改动**：需要电池包中性点引出 + N 线——这是硬件级变更，超出控制层改进范畴
- **控制方法**：扩展 FOC 解耦（IN 独立控制），而非 M4 的 Id 脉冲注入
- 比亚迪专利/方案已量产验证，工程可行性经过验证

### 2.4 Jian 2023（Applied Energy）— 析锂判据与组合策略

**核心方法**——v2 安全判据（路线 D）的直接依据：
- 三电极模型电池实测 EIS，拟合 EEC 参数
- **三种析锂抑制策略对比**：
  - **TVCS（端电压控制）**：限制端电压在充放电截止电压内，低频时负极电位低于 0V → 析锂
  - **NEPCS（负极电位控制）**：直接限制负极电位 >0V vs Li/Li⁺，精度最高，但需要三电极数据
  - **FCICS（全电池阻抗控制）**：用全电池阻抗替代负极阻抗，保守但损失加热速率（6.49%~61.4%）
- **边界频率**：TVCS 与 NEPCS 适用频率的交叉点，低于边界频率用 NEPCS，高于用 TVCS

**边界频率数据**（-20°C, SOC=50%）：
| SOC | 100% | 75% | 50% | 25% |
|-----|------|-----|-----|-----|
| 边界频率 | 0 Hz（全频 TVCS） | 300 Hz | 323 Hz | 389 Hz |

**关键公式**（v2 可复用）：
- 析锂判据：|U₃^Neg| = I_ac · |Z₃^Neg| ≤ U_e（负极平衡电位）
- 产热：q = ½·I_ac² · Re(Z_Full)（频域产热）

### 2.5 Zhang 2015（JPS）— 频域热生成模型

**核心方法**——v2 电池模型（路线 C）的基础：
- 频域热生成率：Q̇ = (A/√2)² · Re(Z)（A = AC 幅值，Re(Z) = 阻抗实部）
- 集总能量守恒：mc·dT/dt = Q̇ - hS(T - T∞)
- 18650 实验验证（-20→5°C，15min，7A/1Hz），最大偏差 <1.5°C
- 预加热时间等值线图：振幅↑频率↓保温↑ → 加热时间↓

**与 M4 关系**：M4 的 P_batt_heat = I²×R_batt 是该频域模型在固定频率下的简化形式。v2 应改用 Re(Z(T,SOC,f)) 替代固定 R_batt。

## 3. 技术路线差异分析（与 M4 对比）

### 3.1 控制方式差异

| 维度 | M4 当前 | 文献推荐（v2 候选） |
|---|---|---|
| **换流机制** | CMI 架构下的开环低频 d 轴脉冲（50Hz `Id` 参考） | 非零矢量+零矢量+死区 SVPWM 换流（10kHz PWM 周期，d0 调节） |
| **控制目标** | `Id` 幅值/频率/占空比开环 | 电池电压谷值闭环（PI 调节 d0）或母线电流有效值闭环 |
| **控制频率** | 加热脉冲 50Hz，PWM 2kHz | 统一 10kHz（PWM=控制周期） |
| **转矩消除** | 设 Iq=0 | 设 Iq=0 + 转子角度前馈查表 Id_ini |
| **安全约束** | DeviceProtection_Evaluable=0 | 电压限幅（急回撤）+ 析锂判据（负极电位边界频率） |

### 3.2 电池模型差异

| 维度 | M4 当前 | 文献推荐（v2 候选） |
|---|---|---|
| **阻抗模型** | 固定内阻 R_batt=0.4198Ω | 二阶 ECM 频域阻抗 Z(T,SOC,f) |
| **产热计算** | P=I²×R_batt（固定值） | Q̇=½I²×Re(Z(T,SOC,f))（频率相关） |
| **温度预测** | 固定窗口热账本（40ms 短时） | 集总能量守恒 + 频域产热（长时预测） |

### 3.3 安全判据差异

| 维度 | M4 当前 | 文献推荐（v2 候选） |
|---|---|---|
| **析锂判据** | 未建立 | 负极电位 >0V vs Li/Li⁺ + 边界频率组合策略 |
| **电压约束** | 被动监控（LimitConfig） | 主动约束（PI 电压闭环 + bn 快速回撤） |
| **DeviceProtection** | 始终为 0 | 可评估（满足前提时可为 1） |

## 4. 文献间的矛盾与待确认项

1. **最优频率的矛盾**：2024 IEEE TTE 固定 10kHz（认为低频析锂风险高），比亚迪实车验证 50Hz 效果最佳且寿命无影响，iScience CMI 最优周期 0.008s（125Hz）。矛盾源于：实验室小电池 vs 真实电池包，以及析锂判据的差异。v2 应采用频域阻抗模型统一评估。

2. **转矩消除的充分性**：NPBH 和比亚迪声称"恒等消除"（数学证明），但实际台架测试仍有 <0.2Nm 残差（漏感等未建模因素）。M4 的 Iq=0 命令在零速锁止下转矩可忽略，但行驶中加热时 Iq 需跟踪驱动转矩，不能简单设零。

3. **加热效率定义不一致**：iScience 2021 用 QT/Qtot（温升热/总热）=89.7%，2024 templock 用 Q_bat_ine/E_ele（电池内能增量/电能消耗）=95.8-97.6%，口径不同不能直接比较。v2 应统一口径。

4. **DC-link 电容影响**：2024 IEEE TTE 认为 Cp<1mF 安全，M4 的 470μF 在安全范围内。但 M4-02B 扫描显示电容减半可使 I_batt 从 5.5A 升至约 11.2A（代价是纹波增大）。这说明在 CMI 架构下电容影响与控制方式有耦合效应，需在 v2 中重新评估。

## 5. 对 v2 平台建设的指导

本节依据 `M5_文献评估报告_v01.md` 更新。控制、热模型、安全判据和拓扑是分层关系，不是四条并行主路线。

### 5.1 首版硬件边界

保持 M4 的单包三相三线、标准三相两电平逆变器、DC-link、PMSM 和控制接口，不新增电池中点、N 线或额外开关。这样可以将首版性能变化归因于控制机制和电池模型，并复用 M4 的实际门极、RMS 审计和账本基础设施。

NPBH、BYD 四线制和 DMSI 属于硬件拓扑替代，分别需要中点开关、电池包 N 线或电池分组继电器，不进入首版默认硬件。

### 5.2 首版控制层

**主控制：2024templock + 2024 IEEE TTE 的 dSVPWM（死区空间矢量脉宽调制）控制族。**

- 采用非零矢量、零矢量、死区矢量和补偿矢量形成电池侧交流电流。
- 使用电压峰值/谷值快速回撤作为快速约束。
- 使用电池温度和温升趋势调节 `bn`/加热强度作为慢速监督。
- `bn=0` 应退化为标准 SVPWM；`bn>0` 的实际母线、电池支路和相电流必须独立读回。
- 不直接移植文献小尺度电机、10 kHz、PI 增益或加热速率数值；这些均是 `LITERATURE` 候选和扫描边界。

**控制对照：CN114789679A。**

母线 RMS 闭环、转子角度前馈和 BMS 温度-电流表作为独立对照控制器。它不能与 dSVPWM 同时独立驱动同一调制层，首版不把两者叠加成“双主环”。

### 5.3 电池电热模型层

以 Zhang 2015 的频域热生成和集总热平衡为结构基础，参考 NPBH 的二阶 ECM 和驱动/加热双频产热分解，逐步替换 M4 固定 `R_batt` 代理。

- 建立 `Z(T,SOC,f)`、`Re(Z)`、温度状态、环境/冷却边界和 SOC 接口。
- 区分电池支路电流、驱动有功电流、加热交流电流及各类损耗热源。
- 对 PWM/脉冲波形执行谐波等效或时域交叉校核；未经校核时只标为平台估算。
- Zhang 和 NPBH 的单电芯/台架参数只保留 `LITERATURE` 身份，不作为具体设备默认值。

### 5.4 安全判据层

以 Jian 2023 的负极电位判据和 TVCS/NEPCS 组合策略建立候选安全层：

- 低频侧优先检查负极电位约束；高频侧同时检查端电压约束。
- 输出边界频率、负极电位代理、端电压峰谷、允许电流和触发原因。
- 原件核验、化学体系映射、D1/D2/D3 假设、波形适用性和模型验证未满足前，`DeviceProtection_Evaluable` 保持 0。
- 建立 `UNKNOWN/CANDIDATE/EVALUABLE` 状态，而不是由仿真成功自动置 1。

### 5.5 拓扑替代层

首版控制、电热和安全接口稳定后，使用同一个 v2 模型评估：

- NPBH：电池中点-电机中性点开关、`i0` 加热电流和转矩解耦。
- BYD 四线制：N 线、扩展 Clarke-Park、充电/停车/行驶全场景。
- DMSI：电池包分组、继电器状态和不支持途中加热的边界。

这些变化属于硬件和接口路线，不是首版 dSVPWM 的控制参数分支。

## 8. M4/M5 路线命名与 dSVPWM 技术说明

本节固化 M4 到 M5 的学术命名、技术差异和 dSVPWM 建模要点。后续报告可以引用本节，不再重复建立并列的路线解释。

### 8.1 正式路线命名

为区分硬件架构、控制算法、电热模型和安全判据，项目统一采用以下命名：

| 阶段 | 学术标识 | 项目中文命名 | 技术层级 |
|---|---|---|---|
| M4 | CMI（Combined Module Invert，文献原称）+ low-frequency d-axis pulsed-current heating | **CMI（Combined Module Invert，文献原称）架构下的开环低频 d 轴脉冲电流自加热路线** | 既定单包驱动架构 + 低频加热控制 |
| M5 首版 | dead-zone Space Vector PWM（dSVPWM）commutation-based self-heating | **dSVPWM（死区空间矢量脉宽调制）换流型脉冲自加热路线** | 同一标准逆变器硬件边界内的调制/换流重构 |

这里的“开环”只限定 M4 的加热参考层：M4 通过低频 `Id_ref` 波形指定加热幅值、频率和占空比，底层电流环仍可能对 `Id_ref` 进行跟踪。因此，M4 不是所有控制环都开环，而是**没有以电池端电流、电压或温度作为主控制目标的加热闭环**。

### 8.2 M4 到 M5 的技术连续性

M5 不是脱离 M4 重新选择一个无关课题，而是沿同一条“电池 - DC-link - 三相逆变器 - PMSM - 电池热响应”的研究链条推进：

1. **M4 建立基础**：完成标准驱动电路、FOC、实际门极、物理测量和电气/热账本的 Reference / Platform 基线。
2. **M4 识别矛盾**：证明电机侧 `Id` 电流不能直接等同于电池支路电流，零速锁止下电池侧有效交流电流过低。
3. **M5 改变机制**：保持首版硬件边界，改用 dSVPWM 有意安排死区换流，使电机电感储能能够经反并联二极管回到 DC-link/电池，形成更强的正负交变电池电流。
4. **M5 补齐审计**：在换流控制之外增加频率相关电热模型和安全候选判据，但不把模型输出自动升级为设备性能或安全结论。

因此，阶段性路线可以简写为：

```text
M4: CMI（Combined Module Invert，文献原称）架构下的开环低频 d 轴脉冲电流自加热路线
    -> 识别电池侧有效电流过低的换流机理卡点
M5: 同一标准逆变器边界 + dSVPWM 死区换流型脉冲自加热
    -> 验证电池侧电流路径能否被控制层重构
```

### 8.3 dSVPWM 的通俗物理解释

`dSVPWM` 中的 `d` 表示 `dead-zone`，即“死区”，不是 M4 的 d 轴。`SVPWM` 是空间矢量脉宽调制，用逆变器的不同开关状态合成电机所需的平均电压。

普通 SVPWM 的一个 PWM 周期通常包含有效矢量和零矢量：

```text
有效矢量：电池 -> 逆变器 -> 电机绕组，绕组储能
零矢量：电机电流在逆变器内部续流，电池侧电流接近零
```

dSVPWM 用受控的死区矢量和补偿矢量替换部分零矢量时间：

```text
非零矢量：电池 -> 电机绕组，电感储能
零矢量：电机电流衰减
死区矢量：电感电流经反并联二极管回到 DC-link/电池
```

由此，电池在一个 PWM 周期内同时出现放电和回充电流。低温电池的内阻会把这部分正负交变电流转化为内部热量，近似关系为 `P_batt_heat ~= I_batt_RMS^2 * R_batt`。补偿矢量用于尽量维持原有平均电压矢量，实际转矩纹波、开关损耗、母线电压和电池电流仍必须由 v2 的六路实际门极和物理测量读回确认。

### 8.4 M4 与 dSVPWM 的控制差异

| 对比项 | M4 CMI 路线 | M5 dSVPWM 路线 |
|---|---|---|
| 加热指令 | 低频 `Id_ref` 脉冲 | 死区/补偿换流的 dwell 时序 |
| 主控制目标 | 电机 d 轴电流参考 | 电池端电压/电流约束下的换流强度 |
| 电池电流形成 | 有效矢量传输，零矢量期间基本不传输 | 非零矢量储能 + 死区二极管回充 |
| 加热电流周期 | 由低频 `Id_ref` 脉冲决定 | 主要由 PWM 换流周期产生，外层可有温度包络 |
| 硬件边界 | 单包、三相三线、标准逆变器和 PMSM | 首版保持相同边界，不增加中点、N 线或开关 |
| 关键验证 | `Id` 跟踪、电池 RMS 和账本 | 实际门极、矢量时序、正负 `I_batt`、电压峰谷、转矩纹波和账本 |

### 8.5 M5 建模中需要保留的接口边界

1. `bn`、`d0`、`a0` 是文献中的不同调制参数化方式，不能未经映射就作为三个独立主控制量同时闭环。首版应选择一个主执行量，其余作为约束、派生量或独立对照。
2. `bn=0` 必须退化为标准 SVPWM；`bn>0` 必须从实际 `PWM_Gates[1:6]` 读回死区/补偿时序。
3. `I_batt`、`I_dc`、`i_abc`、`i_d/i_q` 必须分别记录，不能用电机相电流或命令侧 duty 代替电池支路电流。
4. 电压快速回撤和温度慢速监督属于控制约束层，不等于析锂安全证明。
5. 文献中的 10 kHz、`a0=1.6`、PI 增益、温度阈值和升温提升比例均是 `LITERATURE` 候选，不能直接写入 v2 默认配置。

### 8.6 文献依据与相邻路线边界

- `2024templock` 是文献库中严格命名为 dSVPWM 的直接依据，提供死区矢量、补偿矢量、`bn=0` 退化、温度监督和电压快速回撤结构。
- `2024 IEEE TTE` 未直接使用 dSVPWM 名称，但其“非零矢量 -> 零矢量 -> 死区矢量”换流模式 C 与 dSVPWM 属于同一核心机制，提供 `d0/a0` 和峰谷电压反馈依据。
- `2021 iScience CMI` 是 M4 的架构和电流路径基线，不是 dSVPWM；其 DMSI 是电池分组和继电器改变后的远期拓扑。
- `CN114789679A` 是标准 SVPWM 下的母线 RMS 闭环对照，不是 dSVPWM；`CN112977173A` 和 ATIC 是储能/续流信号流参考，不承担首版 dSVPWM 依据。
- NPBH、BYD 四线制和 DMSI 改变电池中点、N 线或电池分组，属于远期硬件拓扑，不应静默并入 M5 首版。

上述两篇 dSVPWM 同族文献均有小尺度台架或数值证据，但不能视为 v2 已验证结果。IEEE 文献存在 PI 比例增益冲突，templock 存在电压口径差异；所有参数和性能数值仍需在 v2 中重新整定和独立审计。
