# 4500-27低温脉冲加热L0.5仿真计划与阶段结论

版本：v0.1  
日期：2026-06-09  
适用对象：4500-27型整车，驻车堵转低温脉冲加热原理性仿真筛选。

## 1. 当前阶段目标

本阶段不证明量产可行，也不证明电池高频脉冲安全。当前目标是：

```text
在4500-27现有三包并联整体输出、双电机电驱桥和811V高压平台约束下，
判断是否存在值得进入L1电流可实现性模型和台架验证的低温脉冲加热方案。
```

阶段性可行的定义为：

- -20 C、SOC 50%工况下具有有效升温量级；
- 到0 C时间尽量不超过20 min；
- 到0 C等效SOC消耗优先控制在5%以内；
- 不明显突破当前已知电机、逆变器和频率边界；
- 电池30s/60s窗口仅作为参考，不作为kHz脉冲硬否决；
- 结论只能推荐进入下一阶段验证，不能推荐直接开发或装车。

## 2. 已确认建模边界

参数表限定：

- 电池包数量：3；
- 电池包非独立输出；
- 三包并联整体输出；
- 不可独立接入电驱；
- 高压平台：811.44 V，最低630 V，最高919.8 V；
- 电驱桥：R11E300，双CAM255PT56电机；
- 当前只考虑驻车状态，因为脉冲加热需要电机堵转。

因此当前只保留两个方案：

| 方案 | 定义 | 工程定位 |
| --- | --- | --- |
| A | 三包并联整体输出 + 单电机堵转脉冲 | 低复杂度基准方案 |
| B | 三包并联整体输出 + 双电机同步堵转脉冲 | 当前主攻方案 |

以下方案暂不作为当前4500-27工程方案：

- 单包或单支路独立接入电驱；
- 双电机集中作用于单包或单支路；
- 双电机错相；
- 双电机不同频率；
- 双电机轮换作为独立加热方案。

其中双电机轮换在当前模型里等效为单电机平均加热，只能作为电驱热压力缓解策略讨论，不能提高电池平均加热能力。

## 3. 当前模型级别

当前MATLAB模型定义为：

```text
L0.5：等效PWM-RL电流计算 + 零维电池集总热模型
```

电气部分用母线电压、电机绕组电阻、电机d轴电感、频率和占空比计算堵转R-L回路电流：

```text
L * di/dt + R * i = u_pwm(t)
```

模型输出单台电机RMS/峰值电流，再按三包并联电导分流计算支路电流和电池发热：

```text
Q_bat = I_branch_rms^2 * R_branch
```

热模型将每个电池包等效为集总热容，并按弱对流边界估算30 min内温升、能耗和SOC消耗。

当前模型不是完整波形级模型。它不显式计算两台电机的瞬时相位关系、母线电容纹波、采样、死区、最小脉宽、电流环PI和保护触发。

## 4. 本轮模型改造

本轮模型已增加三项抗质疑扫描：

1. 高频阻抗敏感性：

```text
R_heat_factor = 0.3, 0.5, 0.7, 1.0
```

该系数用于表达真实kHz频段电池发热阻抗可能低于1s DCR。当前模型中该系数同时作用于支路等效电阻和电池发热电阻。

2. 电机RMS限值分级：

```text
I_motor_rms_limit = 320, 450, 550 Arms
```

320 Arms代表持续能力参考，550 Arms代表短时能力参考，不能按30 min连续能力理解。

3. 三包均流敏感性：

```text
名义均流：1.0 : 1.0 : 1.0
内阻偏差：0.9 : 1.0 : 1.1
```

## 5. 关键结果

默认工况：

```text
T_init = -20 C
SOC = 50%
f = 1250 Hz
D = 0.5
R_heat_factor = 1.0
I_motor_rms_limit = 550 Arms
```

| 指标 | 方案A 单电机 | 方案B 双电机同步 |
| --- | ---: | ---: |
| 初始温升速率 | 0.366 C/min | 1.430 C/min |
| 30 min后温度 | -10.31 C | 9.01 C |
| 到0 C时间 | 未达到 | 17.68 min |
| 到0 C等效电耗 | 未达到 | 24.77 kWh |
| 到0 C等效SOC消耗 | 未达到 | 3.58% |

默认工况下：

- 方案A温升能力偏弱，不建议作为主攻方案；
- 方案B是当前唯一具有有效温升量级的方案；
- 方案B可推荐进入L1电流可实现性模型和台架验证讨论；
- 该推荐不等于工程定型。

## 6. 方案B敏感性结论

方案B在默认频率、占空比、名义均流下的关键敏感性结果：

| R_heat_factor | 电机限值 | 20 min温度 | 30 min温度 | 到0 C时间 | 结论 |
| ---: | ---: | ---: | ---: | ---: | --- |
| 0.3 | 320 Arms | -15.77 C | -13.84 C | 未达到 | 温升不足 |
| 0.3 | 450/550 Arms | -11.99 C | -8.55 C | 未达到 | 温升不足 |
| 0.5 | 320 Arms | -13.12 C | -10.11 C | 未达到 | 温升不足 |
| 0.5 | 450/550 Arms | -7.39 C | -2.41 C | 未达到 | 接近但不足 |
| 0.7 | 320 Arms | -10.59 C | -6.66 C | 未达到 | 温升不足 |
| 0.7 | 450/550 Arms | -3.31 C | 2.76 C | 25.19 min | 有温升但偏慢 |
| 1.0 | 320 Arms | -7.01 C | -1.96 C | 未达到 | 受持续电流边界限制 |
| 1.0 | 450/550 Arms | 1.95 C | 9.01 C | 17.68 min | 建议进入L1验证 |

说明：

- 在默认1250 Hz、D=0.5下，方案B原始单电机RMS电流约361.5 A，因此450 Arms和550 Arms限值结果相同；
- 若只能按320 Arms持续边界执行，方案B在30 min内仍难达到0 C；
- 若真实高频发热阻抗低于0.5倍1s DCR，方案B默认工况温升不足；
- 热边界敏感性采用等效换热系数h = 0、2、5、10、20 W/m2/K。当前默认h=5 W/m2/K仅是液冷/泵阀状态未知下的占位；在本模型默认产热量级下，h增大会降低20 min终温，但在该范围内影响弱于电流边界和R_heat假设；
- 当前结论明显依赖高频阻抗假设，因此必须补充EIS或高频脉冲试验。

频率和幅值扫描显示，在R_heat_factor=1.0、550 Arms边界下，方案B提高频率/幅值可将到0 C时间压缩到约6-8 min，但电机电流接近短时边界，不能作为长时可用能力直接声明。

## 7. 风险与不能声明的结论

当前可以声明：

- 4500-27实车约束下，当前工程方案应收敛到三包并联整体输出；
- 单电机方案温升偏弱；
- 双电机同步方案在当前L0.5模型下具有有效温升量级；
- 阻抗、限流、均流敏感性会显著影响结论。

当前不能声明：

- 电池高频脉冲充放电安全；
- 不会析锂；
- BMS允许低温回充半周；
- MCU一定支持驻车堵转高频电流控制；
- 550 Arms可以持续用于整个预热过程；
- 电机不会退磁、不会产生不可接受NVH、不会造成绝缘或热老化风险；
- 逆变器结温和器件寿命满足要求。

## 8. 下一步最小验证问题

### 电池/BMS

1. CBBV2/278Ah电芯是否有-30 C至0 C、SOC 10%-90%、1.25-4 kHz频段EIS数据？
2. 是否允许低温下高频回充半周？允许峰值、RMS、持续时间和SOC/温度窗口是什么？
3. 30s/60s窗口能否作为低频参考边界？是否有针对kHz脉冲的修正口径？
4. BMS是否会因端电压波动、回充半周或支路电流估算触发保护？
5. 三包并联时支路均流监测和保护策略如何定义？

### MCU/逆变器

1. 驻车堵转状态下是否允许高压使能和PWM输出？
2. 是否支持零速高频电流闭环或等效d轴电流控制？
3. 1.25-4 kHz范围内最小脉宽、死区、采样频率和调制限制是什么？
4. 550 Arms边界对应持续时间、冷却条件和保护阈值是什么？
5. 双电机同步电流控制是否由同一VCU/MCU协调？同步误差如何限制？

### 电机/可靠性

1. 堵转高频电流是否触发磁钢退磁风险？
2. 绕组、端部、母排和连接器局部热点如何评估？
3. 该载荷是否需要加入可靠性耐久试验？
4. 驻车堵转高频激励是否存在可感知NVH或结构振动问题？

## 9. 后续建模计划

下一步不优先上COMSOL或完整Simulink模型，先进入L1电流可实现性模型：

```text
目标电流/目标PWM
  -> 电机R-L回路
  -> 电流环和电压饱和
  -> PWM频率、死区、最小脉宽
  -> 实际电流RMS/峰值
  -> 电池发热和风险指标
```

进入L1的前提是：MCU/逆变器人员确认驻车堵转高频电流控制不是直接禁止。如果控制权限被否定，则后续应停止复杂模型扩展，转为技术不可行原因归档。
