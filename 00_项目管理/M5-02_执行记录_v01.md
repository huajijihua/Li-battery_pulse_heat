## 0. 阶段目标

在 v2 唯一模型中实现 dSVPWM（dead-zone Space Vector PWM，死区空间矢量脉宽调制）调制层最小闭环，验证 `bn=0` 退化为标准 SVPWM、`bn>0` 死区/补偿矢量换流可独立审计。

## 1. 技术方案

### 1.1 设计决策（用户确认）

| 决策项 | 选择 | 理由 |
|---|---|---|
| dSVPWM 实现方式 | 自定义门极生成器 | blk_856 SVM 模式内部时序不透明，无法显式注入死区矢量 |
| PWM 频率 | 保持 2kHz（M4 继承） | Reference 基线，文献 10kHz 仅作后续扫描点 |
| 主控制量 | bn（加热强度） | templock 文献定义，bn=0 退化，bn>0 调节死区占比 |
| 仿真窗口 | 0.04s / [0.03,0.04]s | 与 M5-01 一致的短时诊断窗口 |

### 1.2 dSVPWM 矢量时序

基于 templock 文献式(1)-(12)和 IEEE TTE 换流模式 C：

- **bn 定义**：`bn = (T0 - T0') / T0`，T0' = T0*(1-bn)
- **死区时间**：`Td = a0 * T0 * bn / (1 + a0)`，a0=1.6（`LITERATURE:2024_IEEE_TTE_a0_1.6`）
- **补偿矢量时间**：`Tc = T0 * bn / (1 + a0)`
- **bn=0**：Td=0, Tc=0, T0'=T0，退化为标准 SVPWM
- **七段对称时序**：V1/2 -> V2/2 -> U0/2 -> Dead/2 -> Comp -> Dead/2 -> U0/2 -> V2/2 -> V1/2

### 1.3 门极编码

| 矢量 | 开关状态 | 门极 [A_H;A_L;B_H;B_L;C_H;C_L] |
|---|---|---|
| U0(000) | 全下管 | [0;1;0;1;0;1] |
| U4(100) | A上,B下,C下 | [1;0;0;1;0;1] |
| U6(110) | A上,B上,C下 | [1;0;1;0;0;1] |
| Dead | 全关断 | [0;0;0;0;0;0] |
| Comp | V1反相 | 1 - V1_state |

- `Gate_Select`（Switch, Criteria=u2>Threshold, Threshold=0）：
  - u1 = dSVPWM_Gate_Generator（经 SigSpec_gates，bn>0 路径）
  - u2 = From_bn_ref（控制端）

## 6. 已知事项

1. **blk_7 悬空线 warning**：dSVPWM_Gate_Generator 输出端有 2 个 model_check 报告的 missing_destination warning。经确认为 Simulink 对 MATLAB Function (Stateflow EMChart) 块的已知误报--输出端口实际连接有效（SigSpec_gates 输入端口验证），不阻塞编译和仿真。
2. **Goto TagVisibility 修复**（2026-07-24 修复）：Goto_bn_ref 的 TagVisibility 原为 local（只在 blk_2 内可见），导致 From_bn_ref（在 blk_7 内）收不到 bn_ref 信号，dSVPWM 实际从未生效。已改为 global。
3. **过调制处理修复**（2026-07-24 修复）：dSVPWM 在 T0<=0（过调制）时原输出 [1;1;1;1;1;1] 导致逆变器短路。已改为退化为标准 SVPWM（bn=0），避免短路。
4. **死区段直接捕获**（2026-07-24 修复）：修复 Goto TagVisibility 和过调制处理后，bn=0.3 工况死区段（sum=0）可直接读回：1153/20001 个采样点。死区段持续时间约 17us（Td），与理论计算一致。
5. **Clock 信号**：dSVPWM MATLAB Function 用 Clock 块提供仿真时间 t_sim，替代 persistent 计数器，确保 t_mod = mod(t_sim, Tsw) 正确。
6. **IGBT_Vf_ref=1.2V**（base）与 AGENTS.md 更正的 0.8V 冲突仍为 M4 冻结基线遗留，不在 M5-02 阶段修改。

## 7. 结论边界

1. v2 模型已实现 dSVPWM 调制层最小闭环，bn=0 退化为标准 SVPWM，bn>0 产生死区/补偿矢量换流。死区段可直接从 PWM_Gates 读回（sum=0 的采样点）。
2. bn=0.3 工况的 I_batt 负向回充电流增大 8.8 倍、RMS 增大 4.5 倍，死区段 1153 个采样点直接可读回，证实 dSVPWM 死区换流机制有效工作。这是 Reference/Platform 诊断证据，不是设备加热性能结论。
3. 未实现电压峰谷 PI 闭环、温度监督、电压快速回撤保护--这些属于 M5-03 或后续阶段。
4. 未实现频域电热模型、析锂安全判据--这些属于 M5-03/04。
5. 过调制处理：dSVPWM 在 T0<=0 时退化为标准 SVPWM（bn=0），避免逆变器短路。过调制可能发生在 Id_ref 较大或 Vdc 较低时。
