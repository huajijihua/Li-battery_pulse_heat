## 使用建议

1. 先读 `AGENTS.md` 和 `README.md`，明确项目边界和推荐读取顺序。
2. 推进 Simulink 主线时，读 `05_模型说明.md` -> `V4_建模主计划_v01.md`。
3. 若要复现 4500-27 L0.5 结果，运行 `run_4500_27_pulse_heating_screening.m`。
4. 需要历史汇报或可再生成大结果时，按 `轻量化交接清单_20260703.md` 从 Git 快照恢复。
