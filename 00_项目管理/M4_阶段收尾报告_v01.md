# M4 阶段收尾报告 v01

日期：2026-07-22

## 0. 摘要

M4 阶段以 `Reference / Platform` 通用基线完成了低温脉冲加热技术原理研究与可审计能量账本建设，收口判据全部满足。技术原理研究过程中识别到**电池侧有效电流过低**的架构性卡点：电机侧脉冲电流 Id=40A，而电池侧 I_batt RMS 仅约 5.5A。M4 的学术路线命名为 **CMI（Combined Module Invert，文献原称）架构下的开环低频 d 轴脉冲电流自加热路线**，其中“开环”限定加热参考层，底层电流环仍可对 `Id_ref` 进行跟踪。经 PDF 文献调研确认，该卡点是当前 CMI 架构和低频加热控制方式共同形成的物理局限，在文献对比中属性能最低档（RTR 约 2.88 °C/min）。

据此决定转换建模路线：将 M4 模型文件夹冻结为参考基线，新建文献验证型平台 `车载脉冲加热仿真平台_v2`，在 M4 账本框架与参数纪律基础上，采用文献验证的换流控制方法重构电池侧电流路径。本报告对 M4 做正式收尾，记录其状态、能力、结论与卡点机理，作为转向 v2 的依据。

## 4. 已验证结论（Reference / Platform 级）

1. **账本封闭性**：电气/热账本在完成窗口残差 ≤ `2.27e-13`，复算与模型输出一致。
2. **参数敏感性（M4-02B）**：`DCLink_C` 与 `Rs` 影响活动物理路径；`R_pack` 与 IGBT 参数目前只影响审计代理。
3. **控制策略（M4-03B）**：在 CMI 架构下，25/50/100 Hz 低频 d 轴脉冲存在 Id 跟踪与电池 RMS 电流的权衡；25%/75% 占空比形成方向偏置；`Iq=0` 命令下仍有交叉轴 RMS 响应。
4. **结论边界**：所有结论限定 `Reference / Platform`，不升级为设备、安全、析锂、寿命或产品级。

## 5. 技术原理研究结论（三问题）

详见 `M4_技术原理研究报告_v01.md`，此处只列要点。

1. **凭什么能发热**：低温下电池内阻显著增大，焦耳热 `P=I²R` 随内阻上升；脉冲电流使锂离子往返正负极，避免单极充电析锂。
2. **发的热是否合理**：能量路径 电池→DC-link→逆变器→电机绕组 可追溯；能量账本审计显示热源主要分配在电机绕组与逆变器（交流侧），**电池侧发热占比小**。
3. **怎么控制发热**：FOC 在 d 轴注入脉冲电流，通过幅值/频率/占空比调节；M4-03B 比较了不同频率/占空比/幅值的影响。

## 6. 卡点结论：电池侧电流过低的机理（转路线核心依据）

### 6.1 现象
- 电机侧 d 轴脉冲电流 `Id=40 A`（交流），电池侧 `I_batt` RMS 仅约 `5.5 A`（DC 分量约 1.6 A + AC 纹波约 5.25 A）。
- 预期：两并电池包下，母线 40 A 应分到约 20 A/支路；实际仅 5.5 A。

### 6.2 机理（M4-02B 扫描证据 + 理论分析）
- 零速锁止下电机近似纯电感负载，`Id=40 A` 主要是无功电流，在电机电感与 DC-link 电容间交换，不流回电池。
- DC-link 电容（470 μF）滤波，电池只承担有功功率（系统损耗约 92 W）+ 少量穿透纹波。
- DC 分量约 1.6 A 由能量守恒决定（损耗/电池电压），物理决定、不可调。
- AC 纹波受电容控制：M4-02B 证明电容减半（235 μF）→ `I_batt` 升至约 11.2 A，`P_batt_heat` 升约 4.2×；但母线电压纹波从约 10.8 V 升至约 19.2 V。

### 6.3 文献验证（PDF 调研，2026-07-22）
经查阅 `01_低温脉冲加热技术参考资料/` 文献库，确认卡点为架构固有局限：

- **当前模型架构 = 2021 iScience（清华）的 CMI 模式 = 2026 NPBH（北理工）对比表中的"常规驱动电路脉冲电流[Du 2022]"**。该架构在文献横向对比中 RTR 最低（约 2.88 °C/min），电池侧最大等效电流仅约 1.41C；而 DMSI 电池分组可达 4C/8.6 °C/min，NPBH 中性点重构达 3.6C/4.76 °C/min。
- **根本原因**：当前 50 Hz 低频开环 d 轴脉冲控制，**未实现"非零矢量（电池放电→电机电感储能）+ 死区/续流（电感能量经体二极管反向向电池充电）"的高频 PWM 换流循环**。文献（2024 IEEE TTE、ATIC）表明，正是这种换流循环让电池-电机-电池形成能量往返，电池电流才是有效交变充放电；零电压矢量作用下电机电流无法传输到电池。
- **电容非主因**：2024 IEEE TTE 指出 Cp<1 mF 时电池电流无显著衰减，Cp>100 mF 才使交流消失；当前 470 μF=0.47 mF 在安全范围。且当前模型 LC 谐振频率约 1/(2π√(0.2 mH×470 μF))≈519 Hz，50 Hz 远离谐振点。**故主因是控制方式，不是电容大小或谐振。**

### 6.4 卡点性质
- 非拓扑错误，非参数错误，是 **CMI（Combined Module Invert）架构下的开环低频 d 轴脉冲电流自加热路线**的固有物理特性。
- 在此架构内做参数优化有天花板（电池侧有效电流约 15 A 量级），无法达到文献所述 1.41C 以上水平，需改控制或改拓扑。

## 7. 转路线决策

M4 作为 CMI 架构的技术原理研究已完成其使命（机理清楚、账本可审计、卡点定位明确）。决定转向文献验证型平台 v2，候选改进路线（基于文献，按落地难度排序）：

| 路线 | 文献依据 | 改动范围 | 预期 |
|---|---|---|---|
| A. dSVPWM 换流型控制重构 | 2024 IEEE TTE（非零矢量+死区+零矢量换流，零矢量占空比 d0 调节，电压反馈 PI） | 仅控制层，不改硬件 | 电池侧有效电流显著提升，加热速率 +45%（文献） |
| B. 闭环目标切换 | 专利 CN114789679A（母线电流有效值闭环 + 转子角度前馈，Iq=0） | 仅控制层 | 电流控制精度与稳定性提升 |
| C. 电池电热模型补全 | 2026 NPBH（二阶 ECM 频域阻抗 Z(Tb,SOC,f)）、2015 频域热生成 | 电池模型层 | 频率相关发热评估准确化 |
| D. 析锂安全约束 | Jian 2023（负极电位 0 V vs Li/Li⁺ 判据、边界频率 TVCS/NEPCS 组合） | 安全判据层 | 安全性指标可评估 |
| E. 拓扑演进（远期） | 2021 DMSI 电池分组 / 2026 NPBH 中性点重构 | 硬件拓扑 | 电池侧电流达 3.6-4C |

v2 初期聚焦 A+B+C+D（在 CMI 框架内改控制与模型，不改硬件拓扑），E 作为远期评估。
