## 引言

本报告在 M4 阶段完成的技术原理研究基础上，系统回答低温脉冲加热的三个核心问题：

1. **凭什么能发热**——能量路径完整可追溯
2. **发的热是否合理**——热源分配可审计
3. **怎么控制发热**——控制策略可比较

所有结论限定为 Reference / Platform 通用系统集成基线，不构成设备参数、产品额定值或安全边界。

## 第一章：凭什么能发热——能量路径完整可追溯

### 1.1 系统能量路径

低温脉冲加热的能量路径从电池包出发，经高压路径、DC-link、逆变器，到达 PMSM 定子绕组，在零速锁止条件下将电能转化为各部件内部损耗（热源）：

```
电池包 (BatteryPack)
  │ 物理端口 Battery_pos / Battery_neg
  │ 测量：V_batt, I_batt, P_batt_terminal
  ▼
高压路径 (HV Path)
  │ 正负路径各一个 HVPath_R = 0.01 Ω
  │ 损耗：P_hv_loss = 2 × HVPath_R × I_batt²
  ▼
DC-link 电容 (DCLink_C = 470 μF)
  │ 储能：E_dc_link = 0.5 × C × V_dc²
  │ 测量：V_dc (逆变器侧)
  ▼
三相逆变器 (Three-phase Inverter)
  │ 6 个 IGBT，PWM 2kHz SVM 调制
  │ 损耗：P_inv = 导通损耗 + 开关损耗 + 死区损耗
  ▼
PMSM 定子绕组 (Stator Windings)
  │ 零速锁止，Id 脉冲激励
  │ 铜耗：P_cu = 1.5 × Rs × (Id² + Iq²)
  │ 铁耗：P_iron = 0（锁止边界，无转速）
  ▼
机械边界 (Park/Lock)
  │ 转速 = 0，反作用转矩记录
  │ 机械功率 P_mech = 0
```

### 1.2 各环节发热机理

**电池发热**：电池等效内阻 R_batt = 0.4198 Ω（Reference 代理），在脉冲电流 I_batt 下产生焦耳热：
```
P_batt_heat = I_batt² × R_batt
```
在基线工况（40A, 50Hz, 50%）的 40ms 窗口内，P_batt_heat ≈ 12.7 W。这是电池低温自加热的主要热源候选。

**高压路径发热**：正负路径各 0.01 Ω，总损耗系数 0.02 Ω：
```
P_hv_loss = 2 × HVPath_R × I_batt²
```
基线工况下 P_hv_loss ≈ 0.6 W，仅为电池热源的 5%，占比极小。

**逆变器发热**：6 个 IGBT 在 2 kHz PWM 开关下产生导通损耗和开关损耗：
```
P_inv = f(Ron, Vf, I_phase, fsw, Eon, Eoff)
```
基线工况下 P_inv ≈ 46.5 W，是最大的单一热源。

**PMSM 铜耗发热**：定子绕组在 Id 脉冲激励下产生铜耗：
```
P_cu = 1.5 × Rs × (Id² + Iq²)
```
基线工况下 P_cu ≈ 31.9 W，是第二大热源。

**PMSM 铁耗**：零速锁止条件下 P_iron = 0（无转速磁场变化）。

### 1.3 能量路径验证证据

| 验证项 | 基线工况结果 | 验证方法 |
|--------|-------------|----------|
| 电池端功率 | P_batt_terminal = 100.9 W | V_batt × I_batt 窗口积分 |
| DC 输入功率 | P_dc_input = 97.3 W | V_dc × I_dc 窗口积分 |
| 电气路径残差 | residual_dc_link = 0.80 W | P_batt - P_hv_loss - P_dc_storage - P_dc_input |
| 独立复算误差 | < 1e-12 | 模型输出与 logsout 独立复算比较 |
| 完整账本有效 | Energy_Ledger_Valid = 1 | 所有账本字段可复算 |

## 第二章：发的热是否合理——热源分配可审计

### 2.1 热源分配总览

基线工况（C00: 40A, 50Hz, 50%, [0.04, 0.08]s 窗口）的热源分配：

```
热源输入总计 ≈ 100.0 W
  ├── P_batt_heat (电池热源)     = 12.7 W  (12.7%)
  ├── P_cu (PMSM 铜耗)           = 31.9 W  (31.9%)
  ├── P_inv (逆变器损耗)         = 46.5 W  (46.5%)
  ├── P_hv_loss (高压路径损耗)   = 0.6 W   (0.6%)
  └── P_iron (铁耗)              = 0.0 W   (0.0%)

热去向：
  ├── P_thermal_storage (热储能) = -111.7 W (负 = 释放初始储能)
  ├── P_heat_rejection (热排散)  = 211.8 W  (向环境排热)
  └── P_unmodeled (未建模残差)   = 46.8 W  (明示的系统残差)
```

**关键判读**：
- 热源输入（100 W）与热去向（211.8 - 111.7 + 46.8 = 146.9 W）的差异反映系统处于瞬态过程
- 负热储能表明初始温度（263.15 K）高于解析热池的平衡温度，系统正在释放储能
- `P_unmodeled = 46.8 W` 是明示的系统残差，包含冷却液带走的热量、未建模热路径等

### 2.2 热源分配随策略变化

**频率影响**（Fig1, Fig5）：25/50/100 Hz
- 25 Hz 时 P_batt = 102.3 W（最高），P_cu = 32.4 W
- 50 Hz 时 P_batt = 100.9 W（居中），P_cu = 31.9 W
- 100 Hz 时 P_batt = 98.7 W（最低），P_cu = 30.5 W
- 趋势：频率越低，跟踪误差越小→实际电流更接近参考→铜耗略高

**幅值影响**（Fig2, Fig5）：20/40/60 A
- 20 A 时 P_cu = 9.5 W, P_inv = 26.5 W, 总热源 = 36.0 W
- 40 A 时 P_cu = 31.9 W, P_inv = 46.5 W, 总热源 = 78.4 W
- 60 A 时 P_cu = 68.5 W, P_inv = 67.2 W, 总热源 = 135.7 W
- P_cu 随幅值平方增长（I²R），P_inv 近似线性增长
- Fig6 参数敏感性龙卷风图显示：Rs 对 P_batt_terminal 影响最大（-15.3% / +30.2%）

**占空比影响**（Fig3, Fig5）：25/50/75%
- 25% 产生负 Id 偏置（均值 -20 A），50% 对称（均值 0 A），75% 正偏置（均值 +20 A）
- 总热源几乎不变（三个占空比下 P_cu、P_inv 基本一致）
- 结论：占空比改变电流方向偏置，但不改变总发热量

### 2.3 参数敏感性结论

基于 M4-02B 的 26 工况参数敏感性扫描（Fig6, Fig7, Fig8）：

| 排名 | 参数 | 对 P_batt 影响 | 物理路径/审计代理 |
|------|------|----------------|-------------------|
| 1 | Rs | -15.3% / +30.2% | 物理路径（PMSM 铜耗） |
| 2 | DCLink_C | +0.55% / -1.06% | 物理路径（母线纹波→电池 RMS） |
| 3 | IGBT_Loss | 无影响 | 仅审计代理 |
| 4 | R_batt_heat | 无影响 | 仅审计代理 |
| 5 | HVPath_R | -0.11% / +0.49% | 物理路径（损耗小） |
| 6-9 | 热池参数 | 无影响（40ms 窗口） | 仅审计代理 |

**关键发现**：
- `DCLink_C` 对母线纹波影响最大：235 μF → 19.2 V 纹波，940 μF → 5.87 V 纹波
- 热时间常数简并：200ms 窗口内 C_inv 与 Rth_inv 不可独立分辨（Fig7）
- 冷却边界对热储能/排热有独立影响（Fig7 T05/T06）

## 第三章：怎么控制发热——控制策略可比较

### 3.1 频率策略比较

**可用频率**：25 Hz / 50 Hz / 100 Hz（40ms 窗口内整数个完整周期）

**关键趋势**（Fig1）：

| 指标 | 25 Hz | 50 Hz | 100 Hz |
|------|-------|-------|--------|
| Id 跟踪 RMS 误差 | 11.5 A | 13.8 A | 17.9 A |
| P_batt_terminal | 102.3 W | 100.9 W | 98.7 W |
| P_cu | 32.4 W | 31.9 W | 30.5 W |
| P_inv | 46.5 W | 46.5 W | 46.5 W |
| 纹波 | 10.1 V | 10.8 V | 10.8 V |

**可解释趋势**：
- 低频（25 Hz）→ 跟踪误差小 → 实际电流接近参考 → 铜耗略高
- 高频（100 Hz）→ 跟踪误差大 → 电流难以跟随 → 有效值略低
- 频率对逆变器损耗无影响（PWM 频率固定 2 kHz）

**结论**：在当前模型和无电池频域阻抗数据下，不能选出"最佳频率"。低频更有利于电流跟踪，高频参考更平滑但实际响应更差。

### 3.2 幅值策略比较

**可用幅值**：20 A / 40 A / 60 A

**关键趋势**（Fig2）：

| 指标 | 20 A | 40 A | 60 A |
|------|------|------|------|
| P_cu | 9.5 W | 31.9 W | 68.5 W |
| P_inv | 26.5 W | 46.5 W | 67.2 W |
| 纹波 | 3.5 V | 10.8 V | 18.0 V |
| 跟踪误差 | 5.2 A | 13.8 A | 23.8 A |
| 调制指数 | 0.895 | 0.949 | 0.950 |

**可解释趋势**：
- P_cu ∝ I²：20→40 A 时 4 倍（9.5→31.9），40→60 A 时 2.25 倍（31.9→68.5）
- 纹波随幅值线性增长
- 60 A 时调制指数接近 0.95，电压裕度降至 3.4 V，接近调制极限

**结论**：幅值增加可有效提高发热功率，但代价是纹波增大、跟踪误差增大、调制裕度减小。60 A 在当前模型中是保守的上限。

### 3.3 占空比策略比较

**可用占空比**：25% / 50% / 75%

**关键趋势**（Fig3）：

| 指标 | 25%（负偏置） | 50%（对称） | 75%（正偏置） |
|------|--------------|------------|--------------|
| Id 均值 | -20.0 A | 0.0 A | +20.0 A |
| P_batt | 101.2 W | 100.9 W | 101.0 W |
| P_cu | 31.9 W | 31.9 W | 31.9 W |
| P_inv | 46.5 W | 46.5 W | 46.5 W |

**结论**：占空比改变电流方向偏置但不改变总发热量。25% 与 75% 互为镜像，50% 对称。这是非对称脉冲加热策略的基础——可以通过偏置方向控制电池是净充电还是净放电。

### 3.4 限幅与停机策略

M4-03B 验证了 C07-C10 四组限幅/停机配置（LimitId=20, LimitDuty=25, LimitMode=3, LimitStatus=0）：

**结果**：四组工况与 C00_Base 完全一致，`ControlSaturation_Any=0` 在所有工况保持。

**结论**：当前模型中的 `LimitConfig` 块为**被动监控层**，记录限幅配置但不主动约束脉冲参考。这不是设备保护已验证的结论，而是模型功能边界——`DeviceProtection_Evaluable=0` 在所有工况保持。

### 3.5 控制策略综合建议

| 控制维度 | 当前建议 | 依据 | 待确认 |
|----------|---------|------|--------|
| 频率 | 50 Hz 为合理折中 | 跟踪误差与功率的平衡 | 需要电池频域阻抗 Re(Z_bat(f)) 数据 |
| 幅值 | 40 A 为研究基线 | 60 A 接近调制极限 | 需要实际设备电流能力 |
| 占空比 | 50% 对称作为基线 | 方向偏置不影响总发热 | 需要电池充放电不对称限制 |
| 限幅 | 当前为被动监控层 | 模型功能边界 | 需要实际保护逻辑和阈值 |

## 第四章：研究结论与数据缺口

### 4.1 已确认的技术原理事实

1. **能量路径可追溯**：电池→HV→DC-link→逆变器→PMSM 的完整物理路径已建立，所有端口功率和储能变化可独立复算
2. **热源分配可审计**：P_batt_heat、P_cu、P_inv、P_hv_loss、P_iron 均有独立计算，与模型输出误差 < 1e-12
3. **控制策略可比较**：频率、幅值、占空比、限幅策略均可在同一账本口径下比较
4. **主导参数已识别**：Rs 对电池端功率影响最大（-15.3%/+30.2%），DCLink_C 对母线纹波影响最大

### 4.2 数据缺口与待确认项

| 缺口 | 影响 | 优先级 |
|------|------|--------|
| 电池低温频域阻抗 Re(Z_bat(T,SOC,f)) | 无法评估频率对电池内部发热的影响 | 高 |
| 实际设备电流/电压/温度能力 | 无法确定 60 A 是否在安全范围内 | 高 |
| IGBT 开关损耗标定数据 | P_inv 为 Reference 代理，非验证数据 | 中 |
| 完整热网络（冷却回路、热容/热阻） | P_thermal_storage 和 P_heat_rejection 为代理 | 中 |
| 可审计的保护状态机 | DeviceProtection_Evaluable = 0 | 中 |
| 电池充放电不对称限制 | 占空比偏置策略需要此数据 | 低 |

### 4.3 结论边界

```
最高结论等级：Reference / Platform 技术原理研究
不构成：     设备参数、产品额定值、安全边界、析锂结论、寿命评估
适用条件：   零速锁止、短时仿真（40-200ms窗口）、-10°C 环境
```
