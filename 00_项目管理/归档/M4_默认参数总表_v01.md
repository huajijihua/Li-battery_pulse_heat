# M4 默认参数总表 v01

日期：2026-07-22
适用基线：`pulse_heating_official_spine_v04.slx`（M4 收口完成）
配置身份：`ReferenceConfig_ID='M4-01-REF-PLATFORM-v02'`（通用系统集成 Reference / Platform 基线）

## 参数来源与读取规则

本表参数来自三个层次（按优先级从高到低）：

| 层次 | 来源 | 说明 |
|------|------|------|
| L1 | InitFcn 回调 | 模型初始化时执行，定义电气/控制/时序核心参数；运行 `clearvars('all')` 后从官方示例数据文件加载 |
| L2 | Model Workspace | 34 个变量，含 M4-01C 热参数、脉冲命令参数、KPI 状态码 |
| L3 | 块参数 | Simscape 物理块参数（电阻、电容、增益等），引用 L1/L2 变量名 |

**重要约束**：InitFcn 中 `clearvars('all')` 会清空基础工作区，因此所有变量依赖 InitFcn 或 Model Workspace 定义。M4-02B/M4-03B 的受控扫描通过 `SimulationInput.setBlockParameter`/`setVariable` 临时覆盖，不改写默认值。

| pulse_id_amplitude_A | 40 | A | Id 脉冲幅值 |

| diagnostic_frequency_Hz | 50 | Hz | 脉冲频率 |

## 10. 与 M4-02B/M4-03B 扫描的关系

- 本表是模型的**默认参数**（未覆盖时的值）
- M4-02B 使用 `SimulationInput.setBlockParameter` 临时覆盖 HVPath_R、DCLink_C、Rs、热参数等
- M4-03B 使用 `SimulationInput.setBlockParameter` 临时覆盖脉冲频率、幅值、占空比、限幅配置
- 所有覆盖不改写 `.slx` 默认参数，扫描后 `Dirty=off`
- 本表与 M4-02B/M4-03B CSV 中的 `Source_Tag` 列一致
