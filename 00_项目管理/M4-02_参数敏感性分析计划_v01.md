## 1. 目标与结论边界

本轮在固定的零速 FOC 脉冲命令下，识别高压路径、DC-link、电机定子电阻和损耗账本代理对电气路径 KPI 的局部敏感性。它服务于通用系统集成与后续参数获取优先级，不做参数拟合、全局优化、设备选型、安全、析锂、寿命、热性能或产品级结论。

固定工况为：闭环 `Id=40 A`、`Iq=0 A`、`diagnostic_frequency_Hz=50 Hz`、`pulse_duty_percent=50`、理想零速机械边界、`ambient=263.15 K`、`coolantTemp=263.15 K`。频率、占空比、波形和开环策略保留给 M4-03。

完整账本仍保持阻断：`P_dc_link_storage`、`DeltaE_batt`、`P_thermal_storage` 和 `P_heat_rejection` 为 `UNKNOWN/NaN`，故 `Energy_Ledger_Valid=0` 是本轮预期状态，不得被补零或解释为完整能量闭合。

## 3. 时间、窗口与运行方式

模型初始化回调会装载官方示例变量并清理基础工作区。因此 M4-02 不以基础工作区变量覆盖参数，而是以 `Simulink.SimulationInput.setBlockParameter` 对已读回的活动 V4 块参数做单次仿真覆盖，避免污染活动模型。

每次仿真使用 `StopTime=0.08 s`，评价窗口固定为 `[0.04, 0.08] s`，对应两个完整的 50 Hz 脉冲周期。运行器将 `RMS_Window_s_param` 临时设为 `0.04 s`，并独立复算同一窗口的 RMS 和平均功率，以避免瞬时 `RMS_Window_Complete` 事件未被日志末样本捕获而影响审计。

## 5. KPI、判据与结果分层

物理路径 KPI：`V_dc` 最小/最大/纹波、`I_batt` RMS、`I_dc` RMS、平均 `P_batt_terminal`、平均 `P_dc_input`、平均 `P_hv_loss`、HV 损耗系数、调制指标、实际闭环状态、机械转速和反作用转矩。

审计代理 KPI：平均 `P_cu`、`P_batt_heat`、`P_inv`、`P_loss_known`、`E_dc_link` 与 `DeltaE_dc_link`。`R_pack` 和 IGBT 参数只在审计代理表中排序，不得与母线、电流或热性能结论混合。

每个工况必须满足：root 结构检查通过；仿真到达 `0.08 s`；三桥臂门极严格互补；闭环使能为 `1`；机械速度保持零速量级；核心测量量可复算。`Energy_Ledger_Valid=0`、`P_dc_link_storage_UNKNOWN=NaN` 和完整热账本字段为 `NaN` 是预期记录项。

**关键发现--热时间常数简并**：T01（C_inv 减半）与 T03（Rth_inv 减半）结果完全相同，T02 与 T04 完全相同。原因是短时 Reference 区段内逆变器热响由热时间常数 `τ = C_inv * Rth_inv` 支配：无排热近似下 `T_inv - T_amb ≈ P_inv * t / C_inv`，储能项 `C_inv * (T_inv - T_amb) ≈ P_inv * t` 与 C_inv 无关，排热项 `∝ 1/(C_inv * Rth_inv)`。T01 与 T03 都使 τ 减半（0.5 C₀ R₀），T02 与 T04 都使 τ 加倍，故简并。结论：**200 ms 短窗口无法独立分辨 C_inv 与 Rth_inv 的敏感性，须更长仿真（趋近排热稳态）才能解耦**。`P_unmodeled` 在 T 系列均为 19.94 W，是明示的系统残差。

冷却边界（ambient/coolant 258.15 / 268.15 K）对 P_thermal_storage 与 P_heat_rejection 有独立影响（T05/T06 与 T00 不同），因环境温度直接进入排热温差。

### 8.4 M4-02B 结论边界

M4-02B 在 Reference / Platform 边界内形成可追溯的参数敏感性证据：物理路径主导参数为 `DCLink_C`、`Rs`、`HVPath_R`；热账本在短窗口内对热时间常数 `τ = C_inv * Rth_inv` 简并，对冷却边界独立敏感。这些是平台局部趋势，不构成设备电池低温阻抗、DC-link 器件选型、逆变器损耗标定、温升验收、析锂安全、寿命或产品级包线结论。`T_batt` 在全部 26 工况保持 263.15 K（环境温度），短时仿真电池温升可忽略，进一步证实当前热账本为 Reference 储能/排热代理，非设备热性能结论。M4-02 通过条件已满足；下一步进入 M4-03B 控制策略比较。
