- **M4-M5 过渡态：关闭。**

## 3. 非阻塞待办与禁止外推

1. CN114789679A、CN112977173A 和 ATIC 原件视觉核验尚未完成；三者分别只能作为控制对照或辅助材料，不阻断 M5-01。
2. 目标电芯化学体系、EIS、负极阻抗/电位映射、OCV、热边界和设备参数仍缺失；这些缺失阻断 `DeviceProtection_Evaluable=1`，不阻断通用 v2 接口建设。
3. IEEE TTE 的 `Kp` 在正文 `0.01` 与 Table II `0.1` 冲突，登记为 `LITERATURE_CONFLICT`；不得写入 v2 默认控制器。
4. Zhang/Jian 的频域热源以正弦 AC 为前提；PWM/脉冲必须通过谐波等效或时域交叉检查后才可形成平台热结论。
5. v2 当前尚未创建 `.slx`；M5-01 已准入但未启动。仅在用户明确启动后，才允许建立唯一模型并迁移受控接口；无论何时均不授权宣布任何性能改善。

## 5. 后续准入顺序

1. M5-01：在 v2 创建唯一模型，仅迁移 M4 的物理接口、测量/RMS 和固定窗口账本语义。
2. M5-02：证明 `bn=0` 退化为标准 SVPWM，再审计 `bn>0` 的实际六路门极、正负电池支路电流和转矩/速度边界。
3. M5-03：在明确波形假设下，将频域热源与时域账本并列复算。
4. M5-04：实现 `UNKNOWN/CANDIDATE/EVALUABLE` 安全状态；未满足数据资格时保持 `DeviceProtection_Evaluable=0`。
5. M5-05：仅在前述接口稳定后，独立比较 CN679、CN173 以及 NPBH/BYD/DMSI 拓扑。
