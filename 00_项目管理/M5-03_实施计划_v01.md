日期：2026-07-24

依据：`M5_顶层建模规划_v01.md` 第 2.4 节、`M5_文献评估报告_v01.md` 第 5.3/7.2 节、`M5_v2_准入与接口契约_v01.md` 第 3/4 节、AGENTS.md 第 11 节

## 0. 阶段目标

在 v2 唯一模型中接入候选二阶 ECM 频域阻抗和频域热生成模型，对 dSVPWM 产生的 PWM 电流执行谐波等效（路径A）和时域交叉校核（路径B），将频域热源与时域电气账本在固定窗口内交叉复算，形成可审计的电热耦合证据。

## 1. 准入条件

- M5-02 完成判据全部通过（bn=0 退化、bn>0 换流可审计）
- dSVPWM 电流波形已定义（bn>0 工况的 I_batt 正负交变波形可读回）

## 2. 技术方向（文献依据）

### 2.1 频域阻抗（NPBH 式5）

```
Z(T,SOC,f) = [R0 + R1/(1+(2πf)²R1²C1²) + R2/(1+(2πf)²R2²C2²)]
             - j[2πf·R1²C1/(1+(2πf)²R1²C1²) + 2πf·R2²C2/(1+(2πf)²R2²C2²)]
```

Re(Z) = R0 + R1/(1+(2πf)²R1²C1²) + R2/(1+(2πf)²R2²C2²)

### 2.2 频域热生成（Zhang 式1）

```
Q̇ = (A/√2)² · Re(Z) = I_RMS² · Re(Z)
```

### 2.3 波形等效校核（文献评估报告第 4.3 节）

- 路径A（谐波分解）：对 PWM 电流做 FFT，按各频率 Re(Z) 累加产热
- 路径B（时域交叉校核）：用时域 I_batt²×Re(Z_dc) 校核频域近似
- 两条路径均需完成

### 2.4 ECM 参数（工程默认值占位）

文献中没有可直接使用的 R0/R1/C1/R2/C2 具体数值（NPBH 只给公式结构，Zhang 的 EEC 数值表未提供）。采用工程默认值占位，全部标 `ENGINEERING_DEFAULT`，明确记录不代表任何真实电芯。

| 参数 | 暂定值 | 来源标签 | 说明 |
|---|---|---|---|
| ECM_R0 | 0.025 Ω | `ENGINEERING_DEFAULT` | 参考 templock 文献量级（单体 -30~-10°C 约 20-35mΩ） |
| ECM_R1 | 0.015 Ω | `ENGINEERING_DEFAULT` | 极化电阻1 |
| ECM_C1 | 0.1 F | `ENGINEERING_DEFAULT` | 极化电容1，f_ct1=1/(2πR1C1)≈106Hz |
| ECM_R2 | 0.010 Ω | `ENGINEERING_DEFAULT` | 极化电阻2 |
| ECM_C2 | 1.0 F | `ENGINEERING_DEFAULT` | 极化电容2，f_ct2=1/(2πR2C2)≈16Hz |
| f_eval | [1 10 50 100 500 1000 2000] Hz | `ENGINEERING_DEFAULT` | 频率评价点 |

温度/SOC 依赖暂用固定值（不查表），记录为平台假设。

## 4. 详细步骤

### 步骤 1：建立二阶 ECM 频域阻抗接口

**新增块**（blk_843 内）：
- `blk_905 Compute_Re_Z_FreqDomain`（MATLAB Function）
  - 输入：T_pack（From）、SOC（From）
  - 输出：Re_Z_vec（各频率点 Re(Z) 向量）、Re_Z_dc（f=0 时 Re(Z)=R0+R1+R2）
  - 算法：按 NPBH 式5 计算 Re(Z)，R0/R1/C1/R2/C2 从 base workspace 读取
  - 参数：ECM_R0/R1/C1/R2/C2（Parameter）、f_eval（Parameter）
- `blk_906 Compute_P_batt_heat_freq`（MATLAB Function）
  - 输入：I_batt（From）、Re_Z_vec（blk_905 输出）、t_sim（Clock）
  - 输出：P_batt_heat_freq（频域产热估算）
  - 算法：滑动窗口 RMS × Re_Z_eff

**base workspace 新增**：
- ECM_R0=0.025, ECM_R1=0.015, ECM_C1=0.1, ECM_R2=0.010, ECM_C2=1.0
- f_eval=[1 10 50 100 500 1000 2000]

### 步骤 2：谐波分解路径A

**新增块**：
- `blk_907 Compute_P_batt_heat_harmonic`（MATLAB Function）
  - 输入：I_batt（From）、Re_Z_vec（blk_905）、t_sim（Clock）、window_s（From）
  - 输出：P_batt_heat_harmonic（谐波分解产热）
  - 算法：固定窗口内对 I_batt 做滑动窗口 DFT，得到各次谐波幅值 A_k，按 Q̇=Σ_k (A_k/√2)²×Re(Z,f_k) 累加
  - DFT 频率点：window_s=0.01s -> 基频 100Hz，谐波 100/200/300...Hz
  - 采样：2e-6s，每窗口 5000 个采样点

### 步骤 3：时域交叉校核路径B

**新增块**：
- `blk_908 Compute_P_batt_heat_timedomain`（MATLAB Function）
  - 输入：I_batt（From）、Re_Z_dc（blk_905 输出，f=0 时 R0+R1+R2）
  - 输出：P_batt_heat_td（时域逐点产热）
  - 算法：P_batt_heat_td = I_batt² × Re_Z_dc

**并列对比**：P_batt_heat_td vs P_batt_heat_freq vs P_batt_heat_harmonic vs 现有 P_batt_heat（M4 固定 R_batt）

### 步骤 4：扩展固定窗口账本

**新增块**：
- `blk_909 Compute_FreqThermal_Subledger`（MATLAB Function）
  - 输入：P_batt_heat_freq、P_batt_heat_harmonic、P_batt_heat_td、P_batt_heat（M4）、t_sim、window_s
  - 输出：P_batt_heat_freq_window、P_batt_heat_harmonic_window、P_batt_heat_td_window、P_batt_heat_M4_window
  - 输出：freq_time_residual（频域-时域残差）、waveform_equiv_valid（波形等效校核有效标志）

**SystemKPI 扩展**：
- SystemKPI(46) = P_batt_heat_freq_window
- SystemKPI(47) = P_batt_heat_harmonic_window
- SystemKPI(48) = freq_time_residual
- SystemKPI(49) = waveform_equiv_valid

**logsout 新增信号**：P_batt_heat_freq、P_batt_heat_harmonic、P_batt_heat_td、Re_Z_eff、freq_time_residual
