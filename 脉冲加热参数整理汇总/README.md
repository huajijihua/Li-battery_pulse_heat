# 脉冲加热参数整理汇总

日期：2026-07-06

本目录是低温脉冲加热建模的参数来源证据链，存放整车、电池、电驱桥、多合一等外部提供的原始参数表与规格书。后续 Simscape/Simscape Battery 建模的参数追溯应回到本目录。本目录不随模型清理删除。

## 目录结构

| 子目录/文件 | 内容 | 状态 |
|---|---|---|
| `部件参数原始表/` | 总布置、电驱桥、电池、多合一参数 xlsx；255 电机 LdLq 标定表；电池电芯规格书与窗口表；电池包检验报告 | 4500-27 主线参数来源，活跃参考 |
| `3806整车平台参数汇总/` | 3806 平台零维脉冲加热重要参数汇总（html + xlsx） | 3806 已从主线移除，参数保留作历史参考 |
| `4500-27脉冲加热.txt` | 4500-27 脉冲加热参数问答与说明 | 4500-27 参数来源说明 |
| `补充和回复.pptx` | 参数补充与电驱回复材料 | 待后续资产整理核实去留 |

## 使用原则

1. 引用本目录参数时，应同时记录文件名、适用车型、参数来源和可信等级。
2. 3806 参数不得直接用于 4500-27 主线，只作历史对照。
3. 电芯规格书（BC5 324Ah、宁德时代 L324D06）参数不得外推到其他电芯。
4. 整理后的参数脚本放入模型目录的 `02_参数库/`，不改写本目录原件。
